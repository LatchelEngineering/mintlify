Recurring Work Orders allow you to automate repeating maintenance tasks — like landscaping, pest control, pool cleaning, or HVAC filter changes — so you don't have to manually create the same work order every week, month, or quarter. Use this guide to submit a new recurring work order or manage an existing one.

### **Submitting a New Recurring Work Order**

Use this process to set up repeating maintenance (e.g., Landscaping, Pest Control, Cleaning, HVAC, etc.).

**Step 1 — Open the Recurring Work Order Form**

Access the **Recurring Work Order Form** to begin the request process and complete all required fields according to your needs.

<Columns cols={2}>
  <Card title="Recurring Work Order Form here!" color="#ff9b8a" icon="file-waveform" iconType="duotone" horizontal href="https://recurring-work-orders.web.app/">
  </Card>
</Columns>

**Step 2 — Select the Property**

Within the form, select the property where the Recurring Work Order needs to be created. Choose the option that matches your situation:

- **For a specific unit or apartment:** Enter the Property URL/Name and skip the Common Area section.
- **For a whole building or common area property:** Select the appropriate Common Area options. If the Common Area Property has already been created, enter the Property URL.
- **Missing a location?** You can request that Latchel create a new Common Area Property directly within the form.

**Step 3 — Set the Frequency**

Choose how often the Recurring Work Order needs to be completed: Weekly, Bi-Weekly, Monthly, Quarterly, Semi-Annually, or Annually.

<Note>
  You only need to submit the form **once** per schedule. Our team will receive your request and schedule the Recurring Work Order to begin on the **Start Date** selected in the form.
</Note>

**Step 4 — Submit the Form**

Once all fields are completed, submit the form. Our team will receive your request and schedule the Recurring Work Order to begin on the **Start Date** you selected.

### **Editing or Canceling a Recurring Work Order**

To make changes to an existing Recurring Work Order, follow these steps:

**Step 1 — Access the Manage Recurring Work Orders Form**

Open the **Manage Recurring Work Orders Form** to view and update your existing recurring work orders.

<Columns cols={2}>
  <Card title="Manage Recurring Work Orders Form here!" color="#ff9b8a" icon="list-timeline" iconType="duotone" horizontal href="https://latchel.typeform.com/to/XfYEOMDe">
  </Card>
</Columns>

**Step 2 — Select Your Desired Action**

Choose what you'd like to do: **Edit** to update the frequency, property, or other details; **Pause** to temporarily stop the recurring work order without canceling it; or **Cancel** to permanently stop the recurring work order.

**Step 3 — Submit the Request**

Our team will process the changes accordingly. You will be notified once the update has been applied.

### **Important Disclaimer**

We do not recommend removing them, as this may impact the recurring work order process.
