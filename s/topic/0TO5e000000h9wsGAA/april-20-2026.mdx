3. **Latchel Overlay: Analytics MVP (AppFolio)**

We’ve added **Analytics to the Latchel Overlay for Property Management (AppFolio)**—bringing powerful insights directly into your existing workflow, especially for emergency-only and benefits-only customers who don’t run their full maintenance lifecycle in Latchel.

- **Maintenance Analytics:** Track key timelines like median days from work order open → close, open → scheduled, and scheduled → completed
- **Stall Rate:** Identify the % of work orders stuck in Open or Assigned status for 14\+ days
- **Property Insights:** Quickly see which properties have the highest volume and most urgent work orders
- **Vendor Insights:** Understand vendor concentration, workload, and time to complete

### **Resolved Bugs**

**Estimate File Viewing Issue**

We’ve resolved an issue where added estimates and files were displaying a “not found” error when viewed directly within the platform. Previously, these files could only be accessed by manually downloading them. With this fix, estimates and attachments now open and display as expected without requiring a download.
