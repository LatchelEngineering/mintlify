title: "Customizable Quick View"

## What is the Product Release?

Property Managers now are able to customize their default view for the work order detail page. They can choose which elements to show and the order that they want to see it in. Please note that this is on the specific PM User level.

In addition to the customizable quick view, we've added the following new pieces of information with this deployment:

- Owner
- Property Management Company with Assigned Property Manager
- Property Portfolio

## What was the Previous State?

Previously, we had given a pre-set default view based on product research. However, we made the decision to deploy this feature because every PM is different and wants a different default view based on how they handle their day to day.

## Who Is Impacted?

Only PMs have the ability to customize their default view for the work order detail page.

## When Is It Relevant?

This applies to any work order detail page for PMs.

## How Is It Used?

See overall demo in this [video](https://www.youtube.com/watch?v=Os89l4mVkYQ). Please note that the demo video is also embedded directly in the Latchel configure side panel and PMs can access at any time.

1. As a PM, open a work order page. PMs who have never set their default view will see a blank page as the default with instructions on how to configure their default page.

<Frame>
  ![Blank default view with configure instructions](/images/Eng-115.png)
</Frame>

2. Click on the "Configure" button. This will bring out a side panel that allows the PM to choose what to add to their default view and rearrange the order that they'd like to see it in.

<Frame>
  ![Configure side panel](/images/Eng-116.png)
</Frame>

3. In order to pick and choose which tiles to show, click on the eye icon within the rectangle (see orange arrow in the screenshot below).

<Frame>
  ![Eye icon to show or hide tiles](/images/Eng-117.png)
</Frame>

4. Clicking on an eye icon makes it visible on the WO quick view. In the screenshot below, you can see that we clicked the eye icon for Current State and it shows up on the quick view.

<Frame>
  ![Current State tile visible on quick view](/images/Eng-118.png)
</Frame>

5. In order to move tiles around, click on the handlebars and drag the tile to the place you'd like it to be. See orange arrow in the screenshot below.

<Frame>
  ![Drag tile using handlebars](/images/Eng-119.png)
</Frame>

6. Once you're done, click the x in the corner of the side panel. What you've chosen is now your default view for every WO.
7. If you'd like to change your default view at any time, click on the Configure button at the top right of the work order page. The side panel will then come out. See orange arrow in the screenshot below.

<Frame>
  ![Configure button on work order page](/images/Eng-120.png)
</Frame>

## What Are Some Risks or Ways Things Could Go Wrong?

There is a minor risk that the view that PMs see will be different from what Admins and Ops see. We are currently working on building a way for Admins and Ops to see any PM User's default view.
