### **Change Management Resources**

- How do I message Latchel Maintenance changes in AppFolio?
  - Add a banner in the tenant portal:
    - [<u>https://help.appfolio.com/s/article/Bulk-Portal-Communication-Settings</u>](https://help.appfolio.com/s/article/Bulk-Portal-Communication-Settings)
  - Add online maintenance instructions.
    - [<u>https://help.appfolio.com/s/article/Add-Online-Maintenance-Request-Instructions-to-the-Online-Portal</u>](https://help.appfolio.com/s/article/Add-Online-Maintenance-Request-Instructions-to-the-Online-Portal)
- How do I prevent residents from entering a maintenance request through AppFolio?
  - Advise the client to enter a support request in AppFolio requesting to disable the ability for residents to enter a maintenance request through that AppFolio tenant portal.
    - _Please note: this is only a viable option if Latchel is handling maintenance for ALL units in AppFolio, as this is a global change in AppFolio._
