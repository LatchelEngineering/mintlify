<Note>
  **Objective**

  The customer's Home Assistant folder consists of the assets the Property Manager will need to implement the Home Assistant program for existing and new leases.

  In this folder, you will find the following assets:

  **Addendums:**

  (1) Full HA Addendum Template

  (2) Fully Optional Full HA Addendum Template

  **Resident Offer Letters:**

  (1) Free Trial Full HA Resident Offer Letter Template

  (2) Full HA Resident Offer Letter Template

  _These assets will need to be modified on a per-customer basis. Please see the steps below on how to create a folder to house this information for the customer and modify the 3 custom assets._
</Note>

## **Step-by-Step Guide**

### **Step 1 - [<u>Create a folder for the Property Manager</u>](https://latchel.atlassian.net/wiki/spaces/CS/pages/865469170/How+to+Handle+Property+Manager+Specific+Files#Howto:HandlePropertyManagerSpecificFiles-step1Step1-CreateafolderforthePropertyManager "https://latchel.atlassian.net/wiki/spaces/CS/pages/865469170/How+to+Handle+Property+Manager+Specific+Files#Howto:HandlePropertyManagerSpecificFiles-step1Step1-CreateafolderforthePropertyManager")**

<u>Note</u>: Only create a folder for the Property Manager if one does not already exist. You can access the list of Property Manager folders [<u>here</u>](https://drive.google.com/drive/folders/1K-OxCEHlyfI6GNhukN9aNPsZho1yzz7T "https://drive.google.com/drive/folders/1K-OxCEHlyfI6GNhukN9aNPsZho1yzz7T"). 

### **Step 2 - Copy Standard Assets to the Property Manager Folder**

1. Open the shared Home Assistant Assets folder in our Customer Success Shared Drive: [<u>https://drive.google.com/drive/folders/1VFI8Lc13aQxVPqSA0sZWzNmhXWssCq2D</u>](https://drive.google.com/drive/folders/1VFI8Lc13aQxVPqSA0sZWzNmhXWssCq2D)
   - From here, you will see the following assets. See the descriptions next to each asset so you know which assets you need to copy into the Property Manager's folder that you are onboarding. 
     1. **Free Trial Full HA Resident Offer Letter Template -** _This is the main resident offer letter for customer's on the <u>full maintenance coordination</u> Home Assistant Plan with a free month to start. This is what the customer will send to their <u>EXISTING</u> residents._
     2. **Full HA Resident Offer Letter Template**_- This is the resident offer letter for customer's on the <u>full maintenance coordination</u> Home Assistant Plan (no free month). This is what the customer will send to their <u>EXISTING</u> residents._
     3. **Full HA Addendum Template**_- This is the lease addendum for customer's only on the <u>full maintenance coordination</u> Home Assistant Plan where new leases are mandatory. This is what the customer will send to their <u>NEW</u> residents or <u>renewals.</u>_
     4. **Fully Optional Full HA Addendum Template\_-\_**\_ This is the lease addendum for customer's only on the <u>full maintenance coordination</u> Home Assistant Plan where new leases and renewals are fully optional. This is what the customer will send to their <u>NEW</u> residents or <u>renewals.</u>\_
2. Make a copy of the appropriate Google assets and move them to the Property Manager folder.

   - Resident Offer Letter

### **Step 3 - Customize the Addendum In Google Docs**

1. Open the Property Manager's addendum file. On page 3 of the addendum document, replace the blank space **(a.)** with the **Company Name** and the blank space **(b.)** with the **Property Manager's Latchel Assigned Phone Number**.
   <Frame>
     ![CSM 77](/images/CSM-77.png)
   </Frame>
   - To find the customer's Latchel-Assigned Phone Number, see [<u>Step 4. Update Maintenance Communication Settings</u>](https://latchel.atlassian.net/wiki/spaces/CS/pages/796296137/How+to+Configure+Customer+Profile+Information+Details+Page+in+Latchel+Admin#Howto:ConfigureCustomerProfileInformation\(DetailsPageinLatchelAdmin\)-step4Step4.UpdateMaintenanceCommunicationSettings "https://latchel.atlassian.net/wiki/spaces/CS/pages/796296137/How+to+Configure+Customer+Profile+Information+Details+Page+in+Latchel+Admin#Howto:ConfigureCustomerProfileInformation(DetailsPageinLatchelAdmin)-step4Step4.UpdateMaintenanceCommunicationSettings") (# 2). This SOP explains how to update these settings but use this SOP to locate the number (the steps are the same). 
   - Once this asset has been updated, it will save it to the Property Manager's folder by default. You do not need to click to save.

### **Step 4 - Customize the Resident Offer Letter In Google Docs**

1. Open the Property Manager's Resident Offer Letter. This asset is only 1 page. Replace **(a.)** with the Property Manager's **start/go-live date**. Replace **(b.)** with the **Company Name.**
   <Frame>
     ![CSM 78](/images/CSM-78.png)
   </Frame>
   - The go-live/start date is determined on the <u>Phase 1 onboard call</u> with the customer (at the end of the call).
   - Once this asset has been updated, it will save it to the Property Manager's folder by default. You do not need to click to save.
