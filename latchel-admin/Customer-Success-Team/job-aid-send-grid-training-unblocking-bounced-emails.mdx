# **Purpose**

<Note>
  **Purpose of this document**

  This is a job aid on how to research whether sendgrid is successfully sending our emails and how to remove a user from a "suppression" list if they are not receiving emails (only for write access). We want to handle this more gracefully in the long term but for now, this gives us a way to unblock customers. **This should be useful anytime you are researching an instance where a customer reports either not getting emails, getting too many/duplicate emails etc.**
</Note>
