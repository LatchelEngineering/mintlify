# **Purpose**

<Note>
  By default, the _Latchel-Assigned Phone Number_ and _Latchel Communication Phone Number_ will default to the area code of the first user added to the console, trying to replicate a local number. In certain instances, a client will request a different phone number in the event the area code we assign is not a local number.
</Note>
