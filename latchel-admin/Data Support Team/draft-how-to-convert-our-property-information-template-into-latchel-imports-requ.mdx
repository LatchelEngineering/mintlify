# Objective

**Purpose**

The purpose of this document is to provide guidance on how to convert the information received via our "Property Information" template into a file that can be uploaded into the Latchel Console with the Import Tool

# Content

**What is happening prior to this process?**

We have asked Property Managers to send us information on their Portfolios and Properties and they have decided to use our template to provide this information

## Step-by-Step Guide

### **Step 1. Receive the Information**

1. When we send an email to Property Managers requesting this information, we include a link to their folder in our shared drive, so they can edit the file there and all of the information will be auto-saved there. 

    1. To find their file, go to the Customer success folder in the "[Property Manager Files](https://drive.google.com/drive/u/0/folders/1K-OxCEHlyfI6GNhukN9aNPsZho1yzz7T)" folder and search for the Property Manager Company name   
    
2. If you don't see the information updated there, check if the Property Manager sent a copy of the template via email.

    1. Go to Zendesk and search for the property manager company name (or for the name of the person you are in contact with) in the Onboarding Tickets view or other Customer Success views ([(Draft) How to: Handle Customer Success emails in Zendesk](https://latchel.atlassian.net/wiki/spaces/CS/pages/849314225))
    2. If the file was sent via email, open the file with Google Sheets and make a copy of it. Save to the Property Manager File folder in the shared drive. Name the file as "Property Information - received via email"
    
        1. To make a copy of the file, click on "File" menu in Google Sheets and select "Make a copy". A new window will open to name the file and select the location it should be saved.
        
            1. By default, it will show your personal drive as the location. Click on that field to show other location options, select Shared Drivers → Customer Success Shared Drive → Property Manager File and find the folder for that Property Manager Company
            2. See the step 4 of this SOP for more detailed instructions - [How to: Handle Property Manager Specific Files](https://latchel.atlassian.net/wiki/spaces/TST/pages/865469170)
            3. If you are not able to make a copy, you can download it to your computer and upload to Google drive. Follow instructions on step of this SOP - [How to: Handle Property Manager Specific Files](https://latchel.atlassian.net/wiki/spaces/TST/pages/865469170)

### **Step 4. Convert Property Information into Latchel Importer Required Format**

1. Return to the tab with the property information received. Click on the row with the table headers to select the whole row and drag down to select all rows that have information

    1. 
    
2. Copy this information. To do that, right-click on any of the row number selected and select "Copy"
3. Go to the new spreadsheet and paste the information there. To do that, right-click on the first row of the spreadsheet and select "Paste"
4. Review the information on each column and check if it is matching the format required 

    1. Step 2 of this SOP provides instructions on the required format: [How to: Add Properties in the Latchel Console to a New Account](https://latchel.atlassian.net/wiki/spaces/TST/pages/795934730)
    
5. The Latchel Importer can only be used to import the following information:

    1. **Street Address**
    2. **Street Address 2 (unit, apt #, etc)**
    3. **City**
    4. **State**
    5. **Zip Code**
    6. Property Critical Note
    7. **Resident Name**
    8. **Resident Phone Number**
    9. **Resident Email**
    10. Owner (Latchel ID)
    11. **Owner Name**
    12. **Owner Phone Number**
    13. **Owner Email**
    14. **Pre-dispatch Instructions**
    15. Escalation Contact List (i.e format --> "1) John Carl: 443-444-444 2) Samantha K. 555-888-999")
    16. Portfolio ID
    
6. You will notice that we don't have all of this information in the spreadsheet and that are columns of the spreadsheet that do not match these fields.

    1. You will select and delete the column for the following columns of the spreadsheet. The information provided in these fields will need to be configured manually
    
        1. "Do service providers have permission to enter this unit? (Yes or No). If yes, please provide access instructions."
        2. "Does this unit need to be treated any differently from the rest of the portfolio? if yes, specify what should be different"
        3. "Does this unit have any type of Warranty? if yes, provide the Warranty Information"
        
    2. Select and delete any column that hasn't had any information filled by the Property Manager
    
        1. To delete a column, right-click on it and select "Delete Column"

### **Step 5. Convert File to .csv and Import into the Latchel Console**

1. Finally, you need to convert the spreadsheet to .csv format. Click on "File", "Download" and select "comma-separated values (.csv, current sheet)

    1. 
    2. This file will be downloaded to your computer and you will need to upload it back to Google Sheets, now in the .csv format
    
        1. To upload the file, go to the "File" menu and click on "Import". A new window will open and you will go do the "upload" section. Click on "Select a file from your device"
        
            1. 
            2. A new window will open for you to show where the file is located. The file should have been downloaded into your "Downloads" file. Select the "Downloads" folder and the file you want to upload
            
                1. 
                
            3. Select the option to "create a new spreadsheet" to avoid potential mistakes of replacing spreadsheets or damaging other spreadsheets
            
                1. 
                
            4. Once the file was imported, you will see the message below. Click on "Open now" link
            
                1. 
                
            5. Another tab will open with the same information you had before downloading the spreadsheet, but now it will be in the .csv format, which is the type of file required by the Latchel Console importer
            6. This new spreadsheet was saved to your personal drive
            7. Use this SOP to guide how to import the file into the Latchel Console: [How to: Add Properties in the Latchel Console to a New Account](https://latchel.atlassian.net/wiki/spaces/TST/pages/795934730)
            8. Once the upload is complete, delete the file from your computer.
            
        
    
2. Repeat all of these steps for each "Portfolio" tab with property information
