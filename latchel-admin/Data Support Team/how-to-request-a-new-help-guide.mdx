title: "How to: Request a New Help Guide"
---

The purpose of this SOP is to define ownership of the creation of new help guides and ongoing improvements and maintenance. Check out the Jira Board here: <custom data-type="smartlink" data-id="id-0">https://latchel.atlassian.net/jira/software/projects/TS/boards/12/backlog</custom> 

Help Guides are worked on in “**Sprints**” which lasts one week (7 days) and begins on Monday. Help Guides prioritized for the current sprint are organized under the “**Help Guides**” sprint in Jira.

‌

## <custom data-type="emoji" data-id="id-1">:blue_book:</custom> Posting a Request in Slack

‌

To request a new help guide or an update to an existing one, follow these steps in sequence:

* Go to [#help-guide-requests](https://latchel.slack.com/archives/C04B3PNJ24V) Slack channel. 
* Click the plus sign <custom data-type="emoji" data-id="id-2">:plus:</custom> below the message box.

![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=fac578d6-1eeb-4eea-b2d1-a73691f7df0f&&collection=contentId-2635333691&height=155&occurrenceKey=null&width=822&__contextId=null&__displayType=null&__external=false&__fileMimeType=null&__fileName=null&__fileSize=null&__mediaTraceId=null&url=null)

* Choose “**New Help Guide Request**” if you need to request a new help guide or “**Update Help Guide**” if you need an existing published help guide to be updated.

**Note:** All updates regarding your request will be posted in the thread of your request. The requester will be informed once the help guide is published.

## <custom data-type="emoji" data-id="id-4">:blue_book:</custom> How to Create a Jira Ticket for the Requested Help Guide

<custom data-type="placeholder" data-id="id-5">Create a step-by-step guide:</custom>

1. Open the <custom data-type="smartlink" data-id="id-6">https://latchel.atlassian.net/jira/software/projects/TS/boards/12/backlog</custom> board and click on the “**create**” bottom at the top of the page. 

‌

![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=7e69e41d-2120-47c7-bac7-0d4446e41e67&&collection=contentId-2635333691&height=1359&occurrenceKey=null&width=1470&__contextId=null&__displayType=null&__external=false&__fileMimeType=null&__fileName=null&__fileSize=null&__mediaTraceId=null&url=null)
2. A pop-up will appear where you will be instructed to fill out a series of fields including the

    1. **Issue Type**
    
        1. Select **Help Guides** as the issue type
        
    2. **Summary** 
    
        1. Use this field as if you were **naming** the **help guide** article 
        
    3. **Description** 
    
        1. **Latchel Feature**: Describe the feature and what it’s intended for 
        2. **Loom Demo:** Attach a video showcasing this feature in action 
        3. **Additional Resources or Context:** Use this field in case there are any other relevant details we should be aware of when educating the selected user audience
        
    4. **Audience**
    
        1. Is this guide for a Property Manager (default), Service Provider, or Resident? Select the audience from the drop-down list! 
        
    5. **Task requested by**
    
        1. Select the **department** you are in from the drop-down list 
        
            ![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=b6808498-39cf-48ef-8d5e-30efeabfea1e&&collection=contentId-2635333691&height=1302&occurrenceKey=null&width=1470&__contextId=null&__displayType=null&__external=false&__fileMimeType=null&__fileName=null&__fileSize=null&__mediaTraceId=null&url=null)
        
    
3. Click **Create**

![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=9d9982de-1a81-4733-be1d-75c6f9b28f79&&collection=contentId-2635333691&height=265&occurrenceKey=null&width=375&__contextId=null&__displayType=null&__external=false&__fileMimeType=null&__fileName=null&__fileSize=null&__mediaTraceId=null&url=null)
All published articles will be posted at the end of the sprint in the **Help-Guide-Release** slack channel!
