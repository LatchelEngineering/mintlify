# Objective

**Purpose**

This document will guide you on how to merge duplicated properties in the Latchel Console. 

Note: There are some users with a permissions level that will allow them to bulk upload properties. This permission level is required to merge duplicate properties. 

# Content

**What is happening prior to this process?**

You have identified a duplicate property that is listed under a single Property Manager's Latchel account or you have been asked by the Property Manager to update a property list and merge duplicate properties on their account.

## Step-by-Step Guide

### **Step 1 - Search for the Property Manager account**

1. Access the Property Manager Account

    1. [How to: Access the Property Manager Profile Page](https://latchel.atlassian.net/wiki/spaces/TST/pages/813729656)
    

### **Step 2 - Search for the duplicate property**

1. Click the **Properties** tab on the Property Manager's account. 

    1. There you will see the list of all the properties added to the Property Manager's account. 
    

    1.   
    
2. Under the  **Properties** tab on the Property Manager's account, utilize the search tool to search the address.

    1. ![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=9ea684c1-978e-4a8a-af25-071552f481f0&&collection=contentId-795116246&height=988&occurrenceKey=null&width=1758&__contextId=null&__displayType=null&__external=false&__fileMimeType=image%2Fpng&__fileName=Latchel%20(60).png&__fileSize=182978&__mediaTraceId=null&url=null)
    
3. Double-check to see if the properties are indeed the same 

    1. Check if they have unit number, i.e. you can have the same address with multiple units at the property
    2. Check if they have different resident information
    3. **ONLY MERGE PROPERTIES IF YOU ARE SURE THEY ARE THE SAME**  
    

### **Step 3 - Merge the duplicate properties**

1. Click to check the boxes on the left of the property address to select the duplicate properties.

    1.   
    
2. Next, click on the drop-down menu on the far upper right of the Properties screen to access an option for merging duplicate properties. Click the **Merge** option**.**

    1.   
    
3. A pop-up will appear- Select the radio button next to the address you'd like to keep and then select **Merge**.

    1. ![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=a48393c4-b449-421b-ae43-0f550a3a0066&&collection=contentId-795116246&height=887&occurrenceKey=null&width=1825&__contextId=null&__displayType=null&__external=false&__fileMimeType=image%2Fpng&__fileName=Latchel%20(63).png&__fileSize=217866&__mediaTraceId=null&url=null)
    
4. That's it! The duplicate properties have been merged.
