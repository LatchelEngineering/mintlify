title: "How to Create Monthly Latchel Autopay Checker"
---

<Note>
**Purpose**

This SOP is created to ensure that autopay settings are reviewed and corrected on a monthly basis as part of the invoice creation process, prior to the autopay run on the customer's due date. It provides a step-by-step guide for checking autopay settings to ensure they are properly enabled and accurately reflected in the system.

The autopay toggle should be turned on before the autopay run on the customer's due date. However, due to a limitation in Rillet, the toggle may switch randomly, making regular checks necessary to ensure accuracy and prevent billing issues.
</Note>

## Creating Latchel Autopay Checker

### Step 1: Prepare the Spreadsheet

Make a copy of the Latchel Autopay Checker. Update the last part with the correct month

Delete the entire data from the following tabs:

- Invoices
- Rillet Autopay Tracker

<Warning>
Do not make any changes to the Customer List, Latchel Autopay Tracker, and the Autopay Checker tab
</Warning>

### Step 2: Filling out the Invoices Tab: Get the List of Customers with Invoices

- In Rillet, go to the Invoice tab.
- Click the funnel icon on the right side.
- Set the filter to:
  - Date: Current month
  - Status: Unbilled
- Click Apply. This will display the list of all unbilled invoices for the month (the invoices you uploaded).
- Export this list and import it into the Autopay Checker spreadsheet.
- Copy the data from the imported invoices tab to the existing Invoices Tab. Then delete the imported one.

### Step 3: Checking the Customer List Tab

With the pre-filled formula in the Customer Tab, the list of customers with open invoices for the month should be automatically listed.

### Step 4: Filling out the Copy of Autopay Tracker: Add the Autopay Tracker

The import formula does the updating in real time hence the only thing that should be done is to allow access.

### Step 5. Extract the Rillet Autopay Status

- Export the Customers list from Rillet and import it to a new spreadsheet.
- From the exported data, copy only columns B (Name) and U (Automatic Payments).
- Paste it on the Rillet Autopay Tracker Tab

### Step 6: Compare the Rillet Status with the Autopay Tracker

Highlight B2:E2 and drag down to copy the formula till the end of the list. This will check for mismatches between the Rillet status and the Autopay Tracker.

Make sure that N/A and other errors are accounted for

Filter Column E to show only #mismatch# results.

The file is now ready to be shared with the Rillet Team.
