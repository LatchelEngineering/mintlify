<Note>
The purpose of this job aid is to provide a standardized and secure method for logging into a  Appfolio, including the crucial **two-factor authentication (2FA) step.** Following this process ensures our team can access client data reliably, saving time, and preventing security issues.
</Note>

## **Logging into Appfolio**

<Warning>
Keep in mind each PM has their own personal link to appfolio so make sure you are always using the correct one. Once you select the link it will ask you for the email and password which you will find in the tracker.
</Warning>

### **Step 1 - Locate the correct PM and their specific login credentials**

a. Navigate to the **"Assisted Ledgering Customers Tracker"** and open the **"Overall Details"** tab.

b. Find the PM you need to track.

c. Locate the following three columns for that PM:

`- PMS URL` (This is the PM's unique Appfolio link)

`- Username`

`- Password`

d. Keep this information ready for the next step.

<Frame>
  <img src="/images/CPT-24.png" alt="image-20250808-220510.png" />
</Frame>

### **Step 2 - Initiate the login and trigger the security verification**

a. Click the unique `PMS URL` link from the tracker to open the correct Appfolio login page.

b. Enter the `Username` and `Password` you found in the tracker.

<Frame>
  <img src="/images/CPT-25.png" alt="" />
</Frame>

c. The system will prompt you for a secondary verification code if:

- It is the first time you are logging into this account.
- There have been 30 days of inactivity.

### **Step 3 - Complete the two-factor authentication (2FA) process**

a. On the Appfolio verification screen, ensure the **"Receive code vía SMS"** option is selected.

b. Have the Latchel console open in another tab or window.

c. Click the **"Send verification code"** button in Appfolio. The SMS will be sent to the Latchel console.

<Frame>
  <img src="/images/CPT-26.png" alt="" />
</Frame>

d. In the console, go to the SMS section and open the new message to find the code.

e. Copy the verification code.

f. Return to the Appfolio page, paste the code into the corresponding field, and click **"Sign In"** to complete the login.

<Frame>
  <img src="/images/CPT-27.png" alt="" />
</Frame>

<Danger>
**Codes expire after 10 minutes **so once you start the process make sure you do it and finish the process. If an error is produced or the code expires, please refer to your TL for further instructions.
</Danger>

<Warning>
**Before we begin**
With every new Appfolio Latchel user account we must do the following steps to avoid receiving unessessary notifications from Appfolio. We can also do this process with existing accounts to doublecheck them.

- Make sure you disable all email and text notifications by going to the right, top button. **“Team Latchel”** and choose **“My settings”**.
