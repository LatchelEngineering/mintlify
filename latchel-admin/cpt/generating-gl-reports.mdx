<Info>
  As the **Customer Profitability Department**, we need to ensure that Property Managers are fulfilling their contract and adding the **GL code** to all the **New, Renewal and Monthly Leases** to be able to get revenue from this contract and help the Property Manager to reach the **break-even point** where they will start to see profit from this program. The **GL Report** is one of the reports that helps us check this.
</Info>

## Property Manager Software (PMS)

<Note>
  Property Manager Softwares allow\*\* \*\*landlords and real estate professionals to track leases, residents, and maintenance tasks as well as collect rent and manage finances to reduce costs and streamline operations.
</Note>

Each Property Manager has their own Property Manager Software (PMS). We use their Property Manager Software (PMS) to be able to get the reports we need to track and verify GL codes in the properties. We currently work with the following Property Manager Software:

- [Appfolio](https://latchel.atlassian.net/wiki/spaces/AL/pages/2633793568/Appfolio)
- [Buildium](https://latchel.atlassian.net/wiki/spaces/AL/pages/2634612748/Buildium)
- [Rent Manager](https://latchel.atlassian.net/wiki/spaces/AL/pages/2660466693/Rent+Manager)
- [Propertyware](https://latchel.atlassian.net/wiki/spaces/AL/pages/2750676993/Propertyware)
- [DoorLoop](https://latchel.atlassian.net/wiki/spaces/AL/pages/2870247461/DoorLoop)
- [Rentvine](https://latchel.atlassian.net/wiki/spaces/AL/pages/3132489731/RentVine)
- Rent Direct
- Yardi

Each Property Manager Software (PMS) has its own navigation and process to get the GL report.
