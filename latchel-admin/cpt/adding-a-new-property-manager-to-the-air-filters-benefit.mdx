---
title: "Adding a new Property Manager to the Air Filters Benefit"
---

<Note>
The purpose of this article is to illustrate how the process of adding a new Property Manager into the Air Filters benefit program works. When a Property Manager accepts the Air Filters benefit, a specific process must be adhered to, the details of which will be provided herein.
</Note>

## Step by Step

### Step 1: How to add a New Filter Subscription

<Check>
When a Property Manager decides to offer the Air Filters benefit to their tenants, this will first be added to Hubspot by the Sales Team, which will trigger a message to the **#cpt-updates **channel providing all the subscription’s details, and this will serve as an alert for the CPT that a new Property Manager needs to be added to the FilterTime platform.
</Check>

<Warning>
We must always consider the type of subscription the Property Manager signed for, as it sets the criteria for both shipment frequency and filters amount.
</Warning>

<Check>
Ensure to follow these job aids ([https://latchel.atlassian.net/wiki/spaces/AL/pages/3584688155/How+to+add+newly+onboarded+Property+Managers+to+the+appropriate+tracker?atlOrigin=eyJpIjoiMzE2YWViZDkyMDhkNGFiZWE3MjViNWM3MzUxZTUzZDgiLCJwIjoiYyJ9](https://latchel.atlassian.net/wiki/spaces/AL/pages/3584688155/How+to+add+newly+onboarded+Property+Managers+to+the+appropriate+tracker) & [https://latchel.atlassian.net/wiki/spaces/AL/pages/3585015876/How+to+Create+a+Filter+Tab+Appfolio+Customers?atlOrigin=eyJpIjoiM2VlMzcwNGY5NDRmNGJhMjkyNmQ0MzFjYjNiMGY4NzYiLCJwIjoiYyJ9](https://latchel.atlassian.net/wiki/spaces/AL/pages/3585015876/How+to+Create+a+Filter+Tab+Appfolio+Customers)) to add the corresponding new PM in the correct tab before proceeding with the next steps.
</Check>

<Frame>
  <img src="/images/CPT-439.png" alt="0.png" />
</Frame>

a. Check the PM’s profile in **HubSpot** for the filter size list. If it is not found there, check the PM’s folder in the [**CSMs drive**](https://drive.google.com/drive/u/1/folders/1K-OxCEHlyfI6GNhukN9aNPsZho1yzz7T). If it is not found in either location, reach out to the CSM.

<Check>
The CPT will reach out to the CSM in charge of the account to collect the** Property Manager’s list of filters **(they can use the following template: [HVAC: PM Upload Template](https://docs.google.com/spreadsheets/d/1ARBUfqExwDFM3x00fR-mdJjS4NBswllw/edit#gid=782985211)), as well as the **Property Manager’s PMS access information. **
</Check>

<Frame>
  <img src="/images/CPT-440.png" alt="image-20240215-173728.png" />
</Frame>

The filters' list must contain the following details:

- Shipping address
- Shipping address 2 (if applicable)
- Shipping City
- Shipping State
- Shipping Zip Code
- Shipment frequency
- Start date (when the filter will be shipped for the first time)
- Filter size and Filter quality (for each of the filters needed for the property)

b. Next, the CPT will review the** filters list** to identify if there are any uncommon sizes that need to be double-checked with the Property Manager. Then the sizes will be added, along with the address details, in the corresponding AU tab for the PM.
