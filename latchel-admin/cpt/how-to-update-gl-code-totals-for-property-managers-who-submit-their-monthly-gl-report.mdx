---
title: "How to Update GL Code Totals for Property Managers Who Submit Their Monthly GL Report"
---

<Note>
The purpose of this SOP is to provide a step-by-step guide on how to report GL code totals on a monthly basis, ensuring accurate data and avoiding incorrect reporting to Finance when applicable.
</Note>

<Accordion title="Metropolitan Real Estate">
### Step 1: Verify whether the GL report has been received.

a. Go to the Assisted Ledgering email and **search for the property manager's name** in the search bar.

<Frame>
  <img src="/images/CPT-660.jpg" alt="picture.jpg" />
</Frame>

### Step 2: Locate the Rent Roll and Recurring Charges report.

a. Select **open with > Google sheets** to open the report.

<Frame>
  <img src="/images/CPT-661.jpg" alt="picture 2.jpg" />
</Frame>

b. A new window with the report will be opened.

### Step 3: Customize the report so you can count the properties with GL codes as of the end of the month.

a. **Click the top-left corner** between column A and row 1 to **apply a filter**.

<Frame>
  <img src="/images/CPT-662.jpg" alt="picture 3.jpg" />
</Frame>

b. Once the filter is applied,** click the three lines in column F**, select **only the GL code amount**, and **click OK**.

<Frame>
  <img src="/images/CPT-663.jpg" alt="picture 4.jpg" />
</Frame>

### Step 4: Select all residents shown in column B to see the total number who had a GL code.

a. Count the total number of residents with a GL code as of the end of the month.

<Frame>
  <img src="/images/CPT-664.png" alt="image-20250806-171551.png" />
</Frame>

b. You can confirm the full amount in the bottom-right corner.

<Frame>
  <img src="/images/CPT-665.png" alt="image-20250806-171719.png" />
</Frame>

### Step 5: Add the total GL codes in the [CPT Tracker](https://docs.google.com/spreadsheets/d/1bQ45AVpZG5-ONwnSLFlJP8M5GCl7J__VczN1rfeCFQI/edit?gid=0#gid=0).

a. Go to the **CPT Tracker > Overall Details tab**, and **locate the Property Manager** you’ll be reporting.
b. **Type the total GL codes in column E** (With GL code) of the corresponding Property Manager.

<Frame>
  <img src="/images/CPT-666.png" alt="image-20250806-172123.png" />
</Frame>

c. Update the date in column C.

Since this property manager is already profitable and part of Assisted Ledgering, it isn’t necessary to add the total number of GL codes to the Billing Inputs document.

### Step 6: Organize the email received with the reports in the corresponding month’s folder.

a. **Add the corresponding label**, so the received reports can be organized in the appropriate email folder and uploaded to the correct Drive folder automatically.

<Frame>
  <img src="/images/CPT-667.png" alt="image-20250806-174000.png" />
</Frame>

<Check>
**Example: **It’s July 1st and you’re looking for the June report. The label applied should be 'GL Reports 2025/June'.
</Check>

A '**Processed' label** will be applied automatically, indicating that the report was successfully uploaded to the Drive folder.
</Accordion>

<Warning>
If a resident appears in multiple reports, count them only once. However, if a resident is repeated within the same report, count each GL code separately.
</Warning>

<Frame>
  <img src="/images/CPT-678.png" alt="image-20250806-183906.png" />

c. Repeat the same process for all RHOME portfolios.
