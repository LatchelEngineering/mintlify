<Note>
The purpose of this document is to describe the types of reports in Propertyware, to be able to identify the particular features they offer, and recognize how they could be useful for our purposes.
</Note>

# Types of reports and how they work/information they display

<Check>
In Propertyware we have default folders of reports we can use and customize.
</Check>

The following default folders include:

<Accordion title="Administrative Reports">
1. **Audit Log:** This report displays all auditable actions performed in the application such as _Lease Created, Contact Edited, Unit Deleted, Email Sent, etc._
2. **Leasing Desk Screening Log Report: **This report displays all auditable actions performed when a screening result is Approved, Approved with Conditions, Declined, or Pending, or when a result changes and a new result comes down (i.e., Pending to Approved)
3. **User Login:** This report displays historical login logs
4. **Users:** A list of all users with profile info
</Accordion>

<Accordion title="Contact Reports">
1. **Active Owners:** A list of Owner Contacts attached to active Portfolios
2. **Active Tenants:** A list of Tenant Contacts attached to active Leases.
3. **All Contacts in Leases: **This report lists all contacts within all leases, not just the primary contact. Contact level fields as well as high-level lease fields are available.
4. **Contact Categories:** This report type displays the categorized contacts from the Contacts module.
5. **Contact Notes:** This report type displays all notes attached to contacts.
6. **Contacts: **This report type displays the contacts available from the Contacts module.
7. **Insurance Policies:** This report type displays the insurance policy information for your tenants.
</Accordion>

<Accordion title="Desktop Reports">
1. **Conversations and Comments: **This report displays all Conversations and Comments.
2. **Documents: **This report displays all documents uploaded to the Propertyware account.
3. **Events:** This type of report displays the team's appointment data.
4. **Task: **This report type displays your team's task data.
</Accordion>

<Accordion title="Lease Reports">
1. **Auto Charges: **This report displays all auto charges from the Summary page for Leases.
2. **Auto Payments**: This report displays all auto payments from the Summary page for Leases.
3. **Contact ePayment Info:** This report displays Tenant and Owner Contacts that have ePay/Banking information saved. You may add a custom filter by “Type” to limit the report to only tenants or only Owners, for example.
4. **Conversations and Comments: **This report displays all Conversations and Comments on Leases.
5. **Gross Potential Rent Report:** Gross Potential Rent Report by Units.
6. ** Inspection Schedules: **This report displays those Leases with an inspection schedule.
7. **Lease Notes: **This report displays all notes attached to leases.
</Accordion>

<Accordion title="Maintenance Reports">
1. **Assets and Work Orders:** This report displays all assets attached to work orders in the Maintenance module.
2. **Conversations and Comments:** This report displays all comments attached to Work Orders.
3. **Service Agreement Work Orders: **This report displays work orders created by service agreements created in the Maintenance module.
4. **Service Agreements: **This report displays service agreements created in the Maintenance module.
5. **Time card Entries:** This report displays time card entries on work orders for those Vendors that have time tracking enabled.
6. **Work Order Notes:** This report displays all notes attached to the work order in the Maintenance module.
7. **Work Order Vendors:** This report displays all vendors attached to work orders in the Maintenance module.
</Accordion>

<Accordion title="Marketing Reports">
1. **Campaign Statistics:** A report of statistical data surrounding campaigns by source and status. The report provides optional totals by week, month, quarter, and year-to-date based on the set Run Date.
2. **Most Recent Screening:** This report displays screening info from Prospects in the Marketing Module.
3. **On-Site Screening: **On-Site Screening results for all applicants that have been screened in the current month
4. **Propertyware Marketplace: **A list of all prospects that have been linked to a specific unit.
5. **Prospect Applications:** This report displays those Prospects with an attached Application in the Marketing Module. These types of Prospects are created manually or automatically from the online application form on your Propertyware Portals.
6. **Prospect Conversion by Agent:** This report displays those Leasing Agents that have generated showings and/or converted Prospects to leases for the period specified.
7. **Prospects:** This report displays those Prospects in the Marketing Module. These Prospects are created manually or automatically from the online guest card on your Propertyware Portals.
</Accordion>

<Accordion title="Property Reports">
1. **AssetProtect Policies:** This report type displays the AssetProtect Policies information for your tenants.
2. **Assets:** This report displays all assets attached to units or buildings in the Properties module.
3. **Building Insurance:** This report displays insurance information for those Buildings with available data in the Properties module.
4. **Building Management Fees: **This report displays the management fee information for those Buildings with available data in the Properties module
5. **Building Mortgages:** This report displays mortgage information for those Buildings with available data in the Properties module.
6. **Building Notes: **This report displays all notes attached to buildings in the Properties module.
7. **Buildings: **This report displays the Buildings available from the Properties module.
8. **Portfolio Notes: **This report displays all notes attached to Portfolios in the Properties module.
9. **Portfolios: **This report displays data from the Portfolios in the Properties module.
10. **Portfolios Balance < $0.00:** This report displays data from the Portfolios in the Properties module.
11. **Property Manager Activity:** Property Manager Activity Report.
12. **Target Rent Changes: **This report displays recorded target rent changes.
13. **Unit Availability Detail: **This report displays the available Units in the Properties module.
14. **Unit Notes:** This report displays all notes attached to units in the Properties module.
15. **Units:** This report only displays the rentable Units in the Properties module.
16. **Units Net Change Report: **This report will provide the change in the number of units and unit vacancy (with percentage) over a specified date range.
</Accordion>

<Accordion title="Finance Reports">
1. **Actual Management Fees: **A list of management and leasing fees for the selected date range.
2. **Budgeting:** A summary of budget vs. actual per account for the selected date range.
3. **Cash Suspect:** A list of portfolios that are suspected to have funds in the incorrect bank account in Propertyware.
4. **Credit Memos:** A list of all credit memos on all leases.
5. **ePayments Batch:** A list of ePayments, grouped by Settlement Date.
6. **ePayments Reconciliation:** A list of all ePayments, with sorting and filtering options.
7. **FTB Calculation Report:** Calculated withholdings due to the Franchise Tax Board based on rental income received by non-California residents as per the settings on Buildings.
8. **Income Statement Budget Comparison Summary:** A month and year-to-date comparison of actual income and expenses as of a specific date.
9. **Income/Expense:** A month-to-date and year-to-date list of income and expense account summaries, consolidated.
10. **Income/Expense By Portfolio: **A month-to-date and year-to-date list of income and expense account summaries per portfolio.
11. **Journal Entries:** A list of all journal entries throughout Propertyware for the selected date range.
12. **Leasing Agent Performance Report: **A list of statistics related to Leasing Agent performance.
13. **Maintenance Statistics Report:** A list of statistics related to Maintenance for the selected date range.
14. **Management Detail Report:** A list of transactions that effect a selected portfolio's balance for that time period, includes previous and ending balance.
15. **Managers Commission Extended Report: **A list of management commission fee amounts broken down per commission account.
16. **Managers Commission Fee Report:** A list of management commission fee amounts broken down per commission account, includes unique account such as leasing fees.
17. **Managers Commission Report:** A list of management commission fee amounts broken down per building, then commission account.
18. **Money In - Bank Deposits: **A list of all bank deposits for the selected date range.
19. **Money In - Charges:** A list of all tenant charges for the selected date range.
20. **Money In - Discounts, Adjustments, & Bad Debt:** A list of all discounts, adjustments, and bad debit for the selected date range.
21. **Money In - ePayments:** A list of all incoming ePayments for the selected date range.
22. **Money In - Payments**: A list of all tenant payments for the selected dated range.
23. **Money In - Payments Applied: **A list of all tenant payments with charge detail for the selected dated range.
24. **Money In - Tenant Charges: **A list of all tenant charges for the selected date range.
25. **Money Out - Auto Bills: **A list of all auto bills, does NOT show split information.
26. **Money Out - Bill Payments:** A list of all bill payments for the selected date range.
27. **Money Out - Bills: **A list of all bills, including split information, for the selected date range.
28. **Money Out - Bills With Work Orders:** A list of all bills attached to work orders, including work order information, for the selected date range.
29. **Money Out - Checks:** A list of all banking, management, leases, and money out transactions classified as checks for the selected date range.
30. **Money Out - Owner Draws:** A list of all owner draws for the selected date range.
31. **Money Out - Refunds:** A list of all lease refunds for the selected date range.
32. **Owner 1099s: **A list of 1099-related information displayed per owner contact for the selected date range.
33. **Owner Contacts:** A list of all owner contacts.
34. **Portfolio Bank Reconciliation Balancing:** A list comparing the current portfolio balance and the amount in the selected bank account as of a specific date.
35. **Portfolio Historical Balance: **A list of portfolio and security deposit balances as of a specific date.
36. **Portfolio Security Deposit Balancing:** A list comparing the current deposit held with the balance of the selected bank account, for a given portfolio, as of a specific date.
37. **Property Manager Performance Report:** A list of statistics related to the property manager for the selected date range.
38. **Vendor 1099s: **A list of 1099-related information displayed per vendor for the selected date range.
39. **Vendor Notes:** A list of notes attached to vendors.
40. **Vendors:** A list of all vendors.
</Accordion>

<Accordion title="Financial Statements">
1. **Balance Sheet Detail:** A list of all Asset, Liability, and Equity accounts per transaction for the selected date range.
2. **Balance Sheet Standard:** A list of all Asset, Liability, and Equity account summaries for the selected date range.
3. **Balance Sheet Standard by Month:** A list of all Asset, Liability, and Equity account summaries per month for the selected date range.
4. **Balance Sheet Standard by Portfolio: **A list of all Asset, Liability, and Equity account summaries per portfolio for the selected date range.
5. **Bank Balance Breakdown: **A list of bank balances as of a specific date, broken down per portfolio and including unreconciled items.
6. **Cash Flow Detail: **A selected date range and year-to-date list of income, equity, and expense accounts per transaction.
7. **Cash Flow Summary: **A selected date range and year-to-date list of income, equity, and expense account summaries.
8. **Cash Flow Summary by Month:** A by-month list of income, equity, and expense account summaries for the selected date range.
9. **Cash Flow Summary with Percentages: **A selected date range and year-to-date list of income, equity, and expense account summaries with percentages.
10. **Check Register Report:** A list of all banking, management, leases, and money out transactions classified as checks for the selected date range.
11. **Escrow Held:** A list of all transactions affecting escrow accounts for the selected date range and designated portfolios.
12. **General Ledger: **A list of all transactions affecting the designated portfolios and accounts for the selected date range.
13. **Income Statement Detai**l: A list of all Income and Expense accounts per transaction for the selected date range.
14. **Income Statement Portfolio Comparison: **A grid-style comparison of all Income and Expense account summaries per portfolio for the selected date range.
15. **Income Statement Standard:** A list of all Income and Expense account summaries for the selected date range.
16. **Income Statement Standard by Month: **A list of all Income and Expense account summaries per month for the selected date range.
17. **Income Statement Standard by Portfolio: **A list of all Income and Expense account summaries per portfolio for the selected date range.
18. **Income Statement Unit Comparison: **A grid-style comparison of all Income and Expense account summaries per unit for the selected date range.
19. **Schedule E: **A grid-style comparison of all Income and Expense account summaries in IRS Schedule E format for the selected date range.
20. **Trial Balance:** A list of all accounts with a current Debit or Credit balance for the designated portfolio as of a specific date.
</Accordion>

# Features within reports

The **Reports** tab provides a multitude of reports which you can customize, export to several formats, publish to owner portals, and send by email message once or on a schedule. You can also modify existing reports to create custom reports. Below you will find a quick step guide on how to manage reports.

<Accordion title="Customize Folders">
### Step 1: Create A Custom Report Folder

a. From the Reports module, **click New Folder**.

<Frame>
  <img src="/images/CPT-237.jpg" alt="" />
</Frame>

b. **Enter a Name and Description** for your folder.

<Frame>
  <img src="/images/CPT-238.png" alt="" />
</Frame>

c.  From** Access Restrictions**, determine if you need to control access to your folder. Default is No Restriction.

<Frame>
  <img src="/images/CPT-239.png" alt="" />
</Frame>

<Warning>
By **checking the box for any role**, access is restricted to only those who have been selected. 
 Access Restrictions for all sub-folders will follow the settings for the parent folder.
</Warning>

d. **Click Apply**

<Frame>
  <img src="/images/CPT-240.jpg" alt="" />
</Frame>

<Accordion title="Customize Reports">
### Step 1: Report Builder

Propertyware allows you to create customized reports from copies of existing reports.
You must select the original report, and then customize that report as needed.

a. From the Reports module, **click New Report**.

<Frame>
  <img src="/images/CPT-248.jpg" alt="" />
</Frame>

b. **Select** from the Category list.

<Frame>
  <img src="/images/CPT-249.png" alt="" />
</Frame>

To find the desired **Base Report**, expand the folders and then click the report name. When you select a report, the upper **Description** field shows the default **Description** of the report.

c. Enter a **Name** and **Description** for the report.

<Frame>
  <img src="/images/CPT-250.png" alt="" />
</Frame>

Name section: This defaults to **"Copy of [REPORT NAME],"** but you can change it as needed.
Description section: Enter the description of the report. This can be used to indicate how this report is different from the default report.

d.** Choose** whether to put the new report in the **Default Folder** **or a Custom Folder**.

<Frame>
  <img src="/images/CPT-251.png" alt="" />
</Frame>

e. Click **Apply**.

<Frame>
  <img src="/images/CPT-252.jpg" alt="" />
</Frame>

<Check>
When you create a new report, the system saves it with the name, description, and folder location that you defined. It also records the base report that your custom report was created from.
As you work with the report, the system keeps track of filters, options, schedules, and restrictions applied to the report, as well as whether the report has been published and who made the last change to the report and when.
</Check>
</Accordion>

<Warning>
It is important to note **we shouldn’t make any changes to the reports without the Property Manager’s permission** as the changes can affect other users.
</Warning>
