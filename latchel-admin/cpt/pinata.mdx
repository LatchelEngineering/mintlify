<Note>
  The purpose of this article is to find all the resources and processes related to Piñata.
</Note>

Piñata is a resident rewards program that is available as an optional addition to the Property Manager's subscription with Latchel.

<Warning>
  In **Piñata**, once a contract reaches 12 months, **<u>it auto-renews unless the property manager cancels</u>**. Monitor the [#service-changes](https://latchel.slack.com/archives/CKA2W1KJA) Slack channel regularly for any “**Desire-to-Churn”** **notifications** or **cancellation notices**. Make sure to check regularly the service start date for your PMs assigned in Piñata.
</Warning>

Each Property Manager that is part of this benefit, will be listed in the [Piñata Overview tab](https://docs.google.com/spreadsheets/d/1qd4oXlu6dBYwvPCVnl8fMwoulezF2MTZQoGOhqPqdo0/edit#gid=830634907), in the Benefits CPT Tracker. Here all the necessary information for each Property Manager is included.

<Frame>
  ![6.png](/images/CPT-471.png)
</Frame>

In the Benefits CPT Tracker there will also be included a tab for each Property Manager’s Piñata report. See how to add a new Piñata Tab for Property Managers in this job aid: [How to: Add a New Piñata Tab for PMs in the Benefits CPT Tracker](https://latchel.atlassian.net/wiki/spaces/AL/pages/2953379907/How+to+Add+a+New+Pi+ata+Tab+for+PMs+in+the+Benefits+CPT+Tracker).
