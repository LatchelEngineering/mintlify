---
title: "How to Report the Monthly Piñata Totals for New Age Realty Group"
---

<Note>
  The purpose of this SOP is to provide a step-by-step guide on how to report the monthly Piñata totals for New Age Realty Group to Finance.
</Note>

## Step by step

### Step 1: Go to the Property Manager’s PMS

a. Log in to the PMS and **click on the Reports section \> Rent Roll (Itemized)**

<Frame>
  ![Picture 19.jpg](/images/CPT-722.jpg)
</Frame>

b. Customize the report by selecting the GL code for the Property Manager. You can find this code in the Piñata Overview tab of the Benefits CPT Tracker.

<Frame>
  ![image-20250807-175630.png](/images/CPT-723.png)
</Frame>

c. Select the GL code and set the date to the last day of the month. Example: _If today is September 1st and you’re reporting totals for August, the date should be 08/31/2025._

d. Click Update.

<Frame>
  ![picture 20.jpg](/images/CPT-724.jpg)
</Frame>

### Step 2: Customize the Rent Roll report to display the totals to be reported.

a. Select the following columns in the columns section:

- Unit
- Status
- Property Address
- Tenant
- Lease from
- Lease to
- Eligible for rent increase
- Move in
- Move out

<Frame>
  ![image-20250807-180340.png](/images/CPT-725.png)
</Frame>

b. Click on Actions \> Export as CSV.

<Frame>
  ![image-20250807-180608.png](/images/CPT-726.png)
</Frame>

c. Open a new spreadsheet and apply a filter by selecting the left top corner between the column A and row 1.

d. File \> Import \> Upload the CSV file.

e. Select Replace all data at selected cell.

<Frame>
  ![](/images/CPT-727.png)
</Frame>

f. Move column Unit **(Column A)** to **Column C.**

<Frame>
  ![](/images/CPT-728.png)
</Frame>

g. Apply Filter. Sort **Column B** (Property) A to Z.

<Check>
  Note: _If many properties, filter twice and select all properties on the 2nd filter._
</Check>

h. Go to the **GL Codes column (E) and click the three lines in the corner**. <u>Select only the GL codes </u>and **click OK**

<Frame>
  ![picture 21.jpg](/images/CPT-729.jpg)
</Frame>

If you’re unable to see all GL codes together in the column, make sure to **apply the filter by selecting column A and choosing 'Sort A to Z'.**

i. **Select all the residents displayed with a GL code to view the total amount**. You can confirm the total in the bottom-right section. This will be the total amount that should be reported to Finance

<Frame>
  ![Picture 22.jpg](/images/CPT-730.jpg)
</Frame>

### Step 3: Go to the Billing inputs doc and report the total amount of GL codes.

<Check>
  If you’re reporting the total for August, you should look for the September tab.
</Check>

a. Search for the property manager’s name and input the total number in column Q.

<Frame>
  ![image-20250807-182556.png](/images/CPT-731.png)
</Frame>
