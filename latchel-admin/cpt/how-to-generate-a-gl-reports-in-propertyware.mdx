---
title: "How to: Generate a GL Reports in Propertyware"
---

<Note>
The General Ledger report allows you to examine the transactions for a specific period of an account over a given date range.
</Note>

## Step by Step

### **Step 1 - Navigate to the General Ledger report and customize the columns**

a. Log in to the PM's Propertyware account.

b. Navigate to the **Reports** module.

<Frame>
  <img src="/images/CPT-224.jpg" alt="" />
</Frame>

c. Expand the **[+] Financial Statements** section and select **General Ledger**.

<Frame>
  <img src="/images/CPT-225.jpg" alt="" />
</Frame>

d. Click **View Report**.

<Frame>
  <img src="/images/CPT-226.jpg" alt="" />
</Frame>

e. In the report customization area, select the **Columns** tab.

f. Expand the `Charge Detail` folder and select:

- `Account Code`
- `Account Number`
- `Amount`
- `Description`
- `Frequency`

<Frame>
  <img src="/images/CPT-227.png" alt="" />
</Frame>

g. Expand the `Lease Summary` folder and select:

- `Building Address`
- `End Date`
- `Lease Name`
- `Start Date`

### **Step 2 - Apply the correct filters for the monthly financial period**

a. Select the **Filters** tab.

<Frame>
  <img src="/images/CPT-228.png" alt="image-20230914-193001.png" />
</Frame>

b. **Configure **`Report Time Frame`**:**

- Set the range from the **1st of the month** to the **last day of the current month**.

<Frame>
  <img src="/images/CPT-229.png" alt="image-20230914-193128.png" />
</Frame>

c.** Configure **`Property Filters`**:**

- Set to **None**.

<Frame>
  <img src="/images/CPT-230.jpg" alt="" />
</Frame>

d.** Configure **`Portfolios/Buildings`**:**

- Select **ALL** unless you need a specific portfolio.

e. **Configure **`Account Filter`**:**

- Search for and select the specific Latchel-related accounts, or select **ALL** if required.

f. Click **Apply** to generate the report preview.

<Frame>
  <img src="/images/CPT-231.jpg" alt="" />
</Frame>

### **Step 3 - Export the final report and manage it with extreme caution**

a. Once the report is generated correctly, go to the settings menu and click **Export/Print**.

b. Choose the format required by the Finance department (e.g., PDF, Excel) and download the file.

<Frame>
  <img src="/images/CPT-232.png" alt="image-20230914-193302.png" />
</Frame>

<Warning>
**REMINDERS:**

1. This SOP is a **backup procedure only.** It should only be used if the PM fails to send the report.
2. **Do not save any changes** you make to the report configuration without explicit authorization from the PM.
3. If you must make temporary changes, you are responsible for **undoing them** and returning all settings to their default state before you log out.
</Warning>
