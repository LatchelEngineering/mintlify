<Note>
  The purpose of this document is to provide the information required to understand the General Ledger report, what it displays and how to identify the data we need.
</Note>

### Step 1: Undestanding the GL Report

a. Once we follow the steps to generate the [General Ledger Report](https://latchel.atlassian.net/wiki/spaces/AL/pages/2791571512/How+to+Generate+a+General+Ledger+Report+in+Propertyware), we need to identify the information we need to track the GL codes reported by the Property Manager.

The General Ledger Report (GL Report) needs to be sent to our Finance department the <u>last day</u> of each month, therefore the date range selected needs to be from the 1st of the month till the last day of the current month.

<Frame>
  ![image-20230914-195057.png](/images/CPT-262.png)
</Frame>

b. To recognize the GL code charge, we need to select the correct options we need in the report from the column section:

<Frame>
  ![](/images/CPT-263.png)
</Frame>

You need to select the following options:

1. **Charge Detail:** a. Account Code b. Account Number c. Amount d. Description e. Frequency
2. **Lease Summary:** a. Building Address b. End Date c. Lease Name d. Start Date

<Danger>
  From **Lease: Primary Contact**, we don’t need to select any option.
</Danger>

c. In Columns #3 and #7 you will find the GL code charge and description of the charge:

<Frame>
  ![image-20230914-195449.png](/images/CPT-264.png)
</Frame>

<Check>
  You will also be able to see additional information such as the **full name of the resident**, the **start date of the charge**, **the account code (or GL code)**, **the account number**, **the frequency of the charge**, **end date (should be the same as in the lease)** and the **building address (which is the full address)**.
</Check>

### Step 2: How to calculate the Monthly total GL codes

To calculate the total GL codes added during that period, we can do it in two ways:

a.  If all the GL codes added have the same amount, we can divide the total amount between the charge amount. Example: _Total amount_ \$280.00 / GL code amount \$_40.00 = 7 GL codes_

b. If the GL codes added have different amounts, we can manually count them to get the total GL codes added during that period.
