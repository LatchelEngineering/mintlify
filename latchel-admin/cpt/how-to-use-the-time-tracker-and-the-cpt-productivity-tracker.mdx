<Note>
The purpose of this article is to be a guide on how to use the CPT internal tools for keeping track of the team members production.
</Note>

<Check>
Every time you work on a task you will need to keep record of the time employed for each of them.
</Check>

## Step-by-step

### Step 1: Go to the [CPT Productivity Tracker](https://docs.google.com/spreadsheets/d/19ab0AIF8HxgX_r57wlwUnzflkQ_BgAV8jEtdZW6skKM/edit?gid=639706783#gid=639706783)

a. Open the CPT productivity tracker for the current year and go to the Main tab.

b. Scroll down to the next blank space and continue typing your current task along with the time and date.

<Frame>
  <img src="/images/CPT-550.png" alt="image-20250812-215019.png" />
</Frame>

c. You can go to the Agents Assigned Tasks tab to find the task you’ll be entering in the Main tab.

<Frame>
  <img src="/images/CPT-551.png" alt="image-20250812-215151.png" />
</Frame>

<Check>
If you are not sure which assigned task fits better what you are currently working on, ask for your TL assistance to make sure you are using the correct one.
</Check>
