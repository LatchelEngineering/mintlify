<Note>
The Objective of this resource is to provide all the details on everything we need from Property Managers with Appfolio PMS. This resource is for the CSM to understand what access level is required for the Customer Profitability Team to be able to have the visibility needed to deliver insight and visibility to Stakeholders and to track what is needed for the Assisted Ledgering contract, Filters, Piñata and Optional Benefits.
</Note>

In **Appfolio**, the Property Manager User Role <u>ensures the CP team can do weekly, monthly reviews, and monthly scorecards, which gather accurate information essential for the benefit and success of the PM contract acquired with Latchel</u>. User roles can be customized, therefore if this user role has been previously edited by the Property Manager, the access level may vary. For this, it is important to consider all the features the CP team requires access to, to ensure the user role assigned is appropriate. **If the Property Manager is not willing** to customize the User Role assigned to Latchel **to enable access to all the features the CP team needs**, as it may impact other users, <u>the CS team should set the right expectations that they must provide access to their General Ledger Report to us or send the General Ledger report monthly, otherwise, they will be billed a full invoice amount</u>. The CS team can assist the Property Manager on scheduling the General Ledger report as well, so it can be sent automatically to the Accounting department every month.

The Property Manager should take into consideration that even though user roles can be customized, doing this will affect the level of access for all users that are assigned that specific user role.
