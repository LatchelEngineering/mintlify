  ### **Main Menu & PMS Access Info:**

  <Frame>
    ![image-20250819-193457.png](/images/CPT-1.png)
  </Frame>

  Main Menu: In this section, you will find useful information about the PM Company.

  - Company ID: The code number it’s used in Salesforce to identify a PM company.
  - **Company Name:** The name of the PM company.
  - Last Update (AL): Date of last review
  - Ledgering Remarks: Relevant notes or other important information about the property manager.
  - With GL Code: Number of properties with a GL code the property manager currently has.
  - **No GL Code:** Number of ledgerable properties missing a GL code.
  - Total Ledgerable Units: Total number of ledgerable properties
  - CMRR: Current Monthly Recurring Revenue
  - Price Per Unit: The price Latchel charges the property manager per unit.
  - CSM: The Customer Success Manager assigned to the account.

  <Frame>
    ![image-20250819-195804.png](/images/CPT-2.png)
  </Frame>

  General Information:

  - Ledgering Stage: Current status of the property manager: Onboarding (service hasn’t started yet) or Live (currently active).
  - Days Since Onboard Ended: Number of days the property manager has been live.
  - Service Start Date: Property Manager service start date
  - Contracted Units: Number of contracted units in the AL contract.
  - Resident Charge: GL code charge amount
  - PMS: Property Manager Software Type
  - PMS URL: Website URL

  <Frame>
    ![image-20250819-202742.png](/images/CPT-3.png)
  </Frame>

  - Username: Unique account name used to sign in
  - Password: Access credential for login
  - GL Code: Unique code used by the property manager; add it to the resident’s account to apply the monthly charge.

  <Frame>
    ![image-20250819-203509.png](/images/CPT-4.png)
  </Frame>

  **Units Information:**

  - **Vacant Units:** Number of units currently unoccupied.
  - Expected collected revenue: Total amount collected from properties with a GL code.
  - Active Units: Total number of active properties in Latchel Console.
  - Missed opportunities: Total number of properties that are ledgerable and should have a GL code added.
  - Missed opportunities as of last month: Total number of properties that are ledgerable and should have a GL code added as of the last day of the previous month.
  - Total Missed month to month opportunities: Total number of month to month leases that are ledgerable and should have a GL code added.
  - **Total New Leases with No GL code:** Total number of New leases that are ledgerable and should have a GL code added.
  - **Total Renewals with no GL code:** Total number of Renewal leases that are ledgerable and should have a GL code added.
  - New Leases with No GL code of the month: Total number of New leases that should have a GL code for the current month.

  <Frame>
    ![image-20250819-205800.png](/images/CPT-5.png)
  </Frame>

  **Stakeholder Relevant Reporting Data:**

  - Renewals with no GL code of the month: Total number of Renewal leases that should have a GL code for the current month.
  - **Month to Month Leases of the month with no GL code:** 	Total number of Month to month leases that should have a GL code for the current month.
  - **Total amount of Missed opportunities per month:** Total number of leases that should have a GL code for the current month.
  - Total amount of opportunities: Total number of properties that are ledgerable.
  - % of missed opportunities: **General percentage of missed opportunities**
  - % of missed opportunities per current month: Percentage of missed opportunities for the month.
  - Provisioned Invoice: The result of the price per unit with the current active units.
  - **Rest of invoice amount:** The result of the Expected collected revenue with the Provisioned Invoice.
  - Units Needed to Reach Profitability: **Number of properties needed for the Property manager to reach profitability.**	**CPT Agent:**
  - Assigned Associate: **CP team member assigned to the property manager’s account.**

  ### **General Information:**

  <Frame>
    ![image-20250829-164302.png](/images/CPT-10.png)
  </Frame>

  In this section, you will find useful information about the PM Company:

  - Company ID: The code number it’s used in Salesforce to identify a PM company.
  - **_Company Name:_** The name of the PM company. PMS Access Info: In this section, you will find useful information about the PM Company.
  - **_Rent Roll:_** Report we use to track the changes made during the month, these include change on the name of the resident, charges, dates, etc.
  - General Ledger: Report we use to track the charges made during a month.

  <Frame>
    ![image-20250813-175122.png](/images/CPT-11.png)
  </Frame>

  <Frame>
    ![image-20250813-175156.png](/images/CPT-12.png)
  </Frame>

  ---

  - **_Last update:_** Date when the last review was done.
  - **_GL Code:_** Charge code we use to identify the charges in the PMS.
  - **_Notes:_** Any important highlights. **Example:** _Was in AL contract using the same GL code number_.
  - Resident Charge: The amount the resident will be charged on a monthly basis for the Optional benefits Package.
  - **PMS URL:** The website we will be using to track the Resident’s information.
  - **Username:** As the word indicates, the user we will be using as CP Team to log in.
  - **Password:** As the word indicates, the password we will be using as CP Team to log in.
  - **Agent Assigned:** The name of the agent who will be working with this PM.
  - **CSM:** The name of the CSM who will be working with this PM.
  - PMS: The type of Property Manager System the PM Company will be using. Example: _Appfolio, Buildium, Rent Manager, etc…_
  - **Service Start Date:** The date when the PM started their contract of Optional Benefits.
  - **Contracted Units:** The total number of the units to be assigned with a GL Code charge.
  - With GL Code: The total number of units with a GL code.
  - **No GL Code:** The total number of units without a GL code.
  - Active Units: The total number of properties added in the Latchel console.
  - **Section 8:** The total number of units who are part of Section 8.
  - HOA Units: The total number of units who are part of a Home Owners Association.
  - **Commercial Units:** The total number of units who are under a Commercial lease.
  - **Vacant Units:** The total number of units without a resident living in.
  - **Undefined type:** The total number of units that we do not know for sure what type of property belongs to. **Example:** Commercial, Section 8, Residential, HOA.

  ### **General Totals:**

  <Frame>
    ![image-20250813-175254.png](/images/CPT-13.png)
  </Frame>

  **In this section, you will find the general totals added. This section will be helpful in adding the totals on the [Optional Benefits Monthly Scorecard](https://docs.google.com/spreadsheets/d/1e9psvlXljLhCK_j_Dzhz1LmHRCYp7S23ns9kzJh19x0/edit#gid=594328040):**

  - Amount that Opted in to Benefits: The total amount of residents who opted in to the Optional Benefits Package.
  - Amount that Opted out of Benefits: The total amount of residents who opted out to the Optional Benefits Package.
  - Amount that did not answer to the notification sent: The total amount of residents who did not answer to the notification sent regarding the Optional Benefits Package.
  - Amount that did not receive a notification: The total amount of residents who did not receive any notification regarding the Optional Benefits Package.
  - Opt-in's Breakdown New Leases: The total amount of residents under a New lease who opted in to the Optional Benefits Package.
  - Opt-in's Breakdown Renewal Leases: The total amount of residents under a Renewal lease who opted in to the Optional Benefits Package.
  - Opt-in's Breakdown Month-to-Month Leases: The total amount of residents under a Month-to-month lease who opted in to the Optional Benefits Package.
  - Total Opportunities Missed: The total amount of missed opportunities. Example: _Residents who are under a New, Renewal or Month to month lease but have no GL Code added._
  - Total Opportunities Missed for New Leases: The total amount of missed opportunities under a New lease.
  - Total Opportunities Missed for Renewal Leases: The total amount of missed opportunities under a Renewal lease.
  - Total Opportunities Missed for Month to Month Leases: The total amount of missed opportunities under a Month-to-month lease.

  ### **Monthly Totals as of Last Month:**

  <Frame>
    ![image-20250813-175333.png](/images/CPT-14.png)
  </Frame>

  **In this section, you will find all the totals added as of the last month:**

  - Amount that Opted in to Benefits: The total amount of residents who opted in to the Optional Benefits Package during the month you will be reporting.
  - Amount that Opted out of Benefits: The total amount of residents who opted out to the Optional Benefits Package during the month you will be reporting.
  - Opt-in's Breakdown New Leases: The total amount of residents under a New lease who opted in to the Optional Benefits Package during the month you will be reporting.
  - Opt-in's Breakdown Renewal Leases: The total amount of residents under a Renewal lease who opted in to the Optional Benefits Package during the month you will be reporting.
  - Opt-in's Breakdown Month to Month Leases: The total amount of residents under a Month to month lease who opted in to the Optional Benefits Package during the month you will be reporting.
  - Opt-out's Breakdown New Leases: The total amount of residents under a New lease who opted out  to the Optional Benefits Package during the month you will be reporting.
  - Opt-out's Breakdown Renewal Leases: The total amount of residents under a Renewal lease who opted out to the Optional Benefits Package during the month you will be reporting.
  - Opt-out's Breakdown Month to Month Leases: The total amount of residents under a Month to month lease who opted out to the Optional Benefits Package during the month you will be reporting.
  - Total Opportunities Missed: The total amount of missed opportunities during the month you will be reporting. Example: _Residents who are under a New, Renewal or Month to month lease but have no GL Code added._
  - Total Opportunities Missed for New Leases: The total amount of missed opportunities under a New lease during the month you will be reporting.
  - Total Opportunities Missed for Renewal Leases: The total amount of missed opportunities under a Renewal lease during the month you will be reporting.
  - Total Opportunities Missed for Month to Month Leases: The total amount of missed opportunities under a Month to month lease during the month you will be reporting.
  - Has GL code but no Opt in/Opt out document or opt in agreement: The total amount of residents who have a GL code added during the month you will be reporting but no Opt in/out document/agreement sent.
