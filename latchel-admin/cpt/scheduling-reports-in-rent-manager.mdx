## Step by step

Automating the Report

<Check>
Report automation preferences allow you to schedule reports to run automatically at a set frequency and deliver the report results via email to whomever you choose.
</Check>

### **Step 1 - Navigate to the system-wide report automation settings**

a. Log in to the PM's Rent Manager account.

b. Click the main menu icon (blue lines in the top left corner) and select **Admin**.

<Frame>
  <img src="/images/CPT-287.png" alt="" />
</Frame>

c. Navigate to **Preferences > System > System Preferences**.

<Frame>
  <img src="/images/CPT-288.jpg" alt="" />
</Frame>

d. In the System Preferences menu, go to **Report Options > Report Automation**.

### **Step 2 - Set the default administrator email for all scheduled reports**

a. In the "Report automation administrator email" field, enter the primary email address that should receive automated report results and notifications.

b. Review any other system-level settings as needed.

c. Click **Save** to apply your changes.

For more information, please reference back to the Rent Manager help desk at: [Report Automation](https://rmxhelp.rentmanager.com/Topics/Express/Administration/System-Preferences/Report-Options/Report-Automation.htm?Highlight=schedule+report)
