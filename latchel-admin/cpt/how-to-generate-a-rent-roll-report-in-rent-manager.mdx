---
title: "How to: Generate a Rent Roll Report in Rent Manager"
---

<Note>
The purpose of this document is to be a guide for generating a Rent Roll report in Rent Manager for the weekly reporting process.
</Note>

## **Step-by-step **

### **Step 1 - Navigate to the report and configure its critical settings**

a. Log in to the Property Manager's Rent Manager account.

b. Navigate to **Reports > Rent Roll > Rent Roll and Recurring Charges**.

<Frame>
  <img src="/images/CPT-273.png" alt="" />
</Frame>

<Frame>
  <img src="/images/CPT-274.png" alt="" />
</Frame>

<Frame>
  <img src="/images/CPT-275.png" alt="" />
</Frame>

c. In the report configuration screen, apply the following settings **exactly**:

- **As of Date:** Ensure this is set to the **current date**.
- **Charge Type Order:** This is the most critical setting. Ensure the charge codes are ordered with `RC - Rent charge` at the top, and `RBP - Resident Benefits Package` immediately below it in the second position.

<Frame>
  <img src="/images/CPT-276.png" alt="" />
</Frame>

### **Step 2 - Generate and export the report in the correct format**

a. Next to the "Generate Report" button, use the dropdown menu to select **CSV** as the export format.

b. Click the **"Generate Report"** button.

c. The file will be downloaded to your computer, ready to be used in the weekly review process.

<Frame>
  <img src="/images/CPT-277.png" alt="" />
</Frame>

d. Once the report has been downloaded, go to the Rent Manager tab in Excel, and import the document.

<Frame>
  <img src="/images/CPT-278.png" alt="" />
</Frame>

e. Select the **upload **option, and browse for the file that will be uploaded.

<Frame>
  <img src="/images/CPT-279.png" alt="" />
</Frame>

f. In the “Import location” section, select the **Replace current sheet **option, and click on **import data.**

<Frame>
  <img src="/images/CPT-280.png" alt="" />
</Frame>
