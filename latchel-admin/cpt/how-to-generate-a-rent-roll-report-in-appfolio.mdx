title: "How to: Generate a Rent Roll Report in Appfolio"
---

<Note>
The purpose of this document is to be a guide for generating a Rent Roll report in Appfolio for the weekly reporting process. The goal is to generate a standardized Rent Roll (Itemized) report from Appfolio, filtered specifically by the customer's Latchel GL code. This ensures we have the precise, reliable data needed for the Assisted Ledgering tracking process, allowing us to accurately identify compliance gaps and revenue opportunities for our customers.
</Note>

## **Step-by-step **

### **Step 1 - Prepare for the report by gathering the correct GL Code**

a. Log in to the specific Property Manager's Appfolio account. You can find the Property manager’s info in the **Overall Details** tab of the [**CPT Tracker**](https://docs.google.com/spreadsheets/d/1bQ45AVpZG5-ONwnSLFlJP8M5GCl7J__VczN1rfeCFQI/edit?gid=0#gid=0).

c. Locate the PM you are working on and find the exact GL code created for them. Keep this code handy.

<Frame>
  <img src="/images/CPT-62.png" alt="image-20250811-170035.png" />
</Frame>

### **Step 2 - Configure and filter the Rent Roll report**

a. Inside Appfolio, navigate to **Reporting > Reports**.

b. Select the **Rent Roll (Itemized)** report.

c. In the **Filters** section of the report options:

- Under the GL code filter, select the exact GL code you identified in Step 1.
- Ensure the "Units" filter is set to its default: **Show Active Units**.

<Warning>
If the GL code from the tracker does not appear as an option in Appfolio, you must stop and notify the CSM in charge via the [#ledgering-mkt-opt-benefits](https://latchel.slack.com/archives/C03D4N7B0LF) Slack channel. Do not proceed until this is resolved.
</Warning>

### **Step 3 - Customize the report layout and export the data**

a. In the report options, go to the **Columns** section and select the following fields:

- Unit
- Status
- Property Address
- Tenant
- Lease From
- Lease to
- Eligible for Rent Increase
- Move-in
- Move-out

<Frame>
  <img src="/images/CPT-65.png" alt="" />

b. Click the **Update** button to generate the report preview.
c. Click **Actions > Export as CSV**. The file will be downloaded to your computer, ready for analysis.
