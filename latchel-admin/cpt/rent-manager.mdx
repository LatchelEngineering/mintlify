### **Step 1: Logging into Rent Manager**

<Warning>
  Keep in mind each PM has their own personal link to Rent Manager so make sure you are always using the correct one. Once you select the link it will ask you for the email and password which you will find in the tracker.
</Warning>

a. Go to the Overall Details tab in the [CPT Tracker.](https://docs.google.com/spreadsheets/d/1bQ45AVpZG5-ONwnSLFlJP8M5GCl7J__VczN1rfeCFQI/edit?gid=0#gid=0) Choose the PM you will track and refer to the **PMS column** to identify what PMS the PM uses. Select one with Rent Manager and use the link to log in with the username and password for that PM under the **columns PMS URL, Username, and Password**.
