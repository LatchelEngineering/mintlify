<Note>
  The purpose of this article is to be a guide for generating the Lease Recurring Charges Report in RentVine for the weekly reporting process.
</Note>

## Step by step

### Step 1: Go to Rentvine

a. Access the rentvine account of the property manager you’ll be checking.

b. On the left side of the screen, click on **Reports.** All the reports' categories will display. The first category you will view is the **Favorite Reports,** where you will find the reports you have marked as favorite for easier access.

<Frame>
  ![26.png](/images/CPT-324.png)
</Frame>

### Step 2: Customize the report

a. Select the <u>Lease Recurring Charges</u> report, and the customization window will open. In the **Account section select the option Equals, and then look for Latchel GL code and select it. Lastly, in the Lease Charge is Active** section, select the **Yes** option.

<Frame>
  ![Screenshot 2025-06-18 124302.png](/images/CPT-325.png)
</Frame>

b. Click on **Build Report**. The report will generate showing all the properties with an active GL code added. You will be able to see the **Charge Amoun**t and the Next Charge Date.

<Frame>
  ![Screenshot 2025-06-18 124329.png](/images/CPT-326.png)
</Frame>

c. Manually add the residents with the GL code in the Property Manager’s tab.

<Warning>
  Downloading this report is not necessary.
</Warning>
