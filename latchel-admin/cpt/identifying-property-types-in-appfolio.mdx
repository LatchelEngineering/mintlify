---
title: "Identifying Property Types in Appfolio"
---

<Note>
The purpose of this document is to be able to identify the property and lease types for the weekly reporting directly in Appfolio, to be able to report this correctly in the tracker.
</Note>

<Accordion title="Residential">
1. Open the PM’s Appfolio page, and using the** Search Bar,** enter the tenant’s name you want to check. Some options will be displayed, make sure to click on the tenant’s name with the correct property address, and that the tenant’s name has **(Current)** next to it.

<Frame>
  <img src="/images/CPT-76.png" alt="" />
</Frame>

<Frame>
  <img src="/images/CPT-77.png" alt="" />
</Frame>

1. From the **Summary page**, select the address of the property. This will direct you to the **Property page**. 
<Frame>
  <img src="/images/CPT-78.jpg" alt="" />
</Frame>

3.  Once on the Property page, you will be able to see the property specifications. Look under the address, the option that says **Property Type: Single-Family**. Residential properties would be under this type. 
<Frame>
  <img src="/images/CPT-79.jpg" alt="" />
</Frame>

</Accordion>

<Accordion title="Month to Month Units">
_**Option 1**_

1. Open the PM’s Appfolio page, and using the** Search Bar,** enter the tenant’s name you want to check. Some options will be displayed, make sure to click on the tenant’s name with the correct property address, and that the tenant’s name has **(Current)** next to it.

<Frame>
  <img src="/images/CPT-83.png" alt="" />
</Frame>

1. From the **Summary page**, you will be able to see if it is a month-to-month lease.

<Frame>
  <img src="/images/CPT-84.png" alt="" />
</Frame>

_**Option 2**_

1. From the **Summary page**, scroll down to the **Lease information section**. 
<Frame>
  <img src="/images/CPT-85.png" alt="" />
</Frame>

2. Here you will be able to see the following relevant information. **Month to Month:** It will let you know when the lease is a month-to-month type. 
<Frame>
  <img src="/images/CPT-86.png" alt="" />
</Frame>

<Warning>
Keep in mind that you should always corroborate with the lease file if the contract is in a month-to-month basis, since sometimes the lease is updated but Appfolio does not reflect the changes. (This if there is a lease file available).
</Warning>
</Accordion>
