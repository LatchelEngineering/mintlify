title: "Generating Rent Roll Reports"
---

<Info>
  As the **Assisted Ledgering Department**, we need to ensure that Property Managers are fulfilling their contract and adding the **GL code** to all the **New and Renewal Leases** to be able to get revenue from this contract and help the Property Manager to reach the break-even point where they will start to see profit from this program. The **Rent Roll report** is one of the reports that helps us check this.
</Info>

## Property Manager Software (PMS)

<Note>
  Property Manager Softwares allow\*\* \*\*landlords and real estate professionals to track leases, residents, and maintenance tasks as well as collect rent and manage finances to reduce costs and streamline operations.
</Note>

Each Property Manager has their own Property Manager Software (PMS). We use their Property Manager Software (PMS) to be able to get the reports we need to track and verify GL codes in the properties. We currently work with the following Property Manager Software:

- [Appfolio](https://help.latchel.com/latchel-admin/cpt/appfolio)
- [Buildium](https://help.latchel.com/latchel-admin/cpt/buildium)
- [Rent Manager](https://help.latchel.com/latchel-admin/cpt/rent-manager)
- [Propertyware](https://help.latchel.com/latchel-admin/cpt/propertyware)
- [DoorLoop](https://help.latchel.com/latchel-admin/cpt/doorloop)
- [Rentvine](https://help.latchel.com/latchel-admin/cpt/rentvine)
- Rent Direct
- Yardi

Each Property Manager Software (PMS) has its own navigation and process to get the Rent Roll report. You can view the specific process for each Property Manager Software (PMS) in the following SOPs.

- [How to: Generate a Rent Roll Report in Appfolio](https://help.latchel.com/latchel-admin/cpt/how-to-generate-a-rent-roll-report-in-appfolio)
- [How to: Generate a Rent Roll Report in Buildium](https://help.latchel.com/latchel-admin/cpt/how-to-generate-a-rent-roll-report-in-buildium)
- [How to: Generate a Rent Roll Report in Rent Manager](https://help.latchel.com/latchel-admin/cpt/how-to-generate-a-rent-roll-report-in-rent-manager)
- [How to: Generate a Rent Roll Report in Propertyware](https://latchel.atlassian.net/wiki/spaces/AL/pages/2805858309/How+to+Generate+a+Rent+Roll+Report+in+Propertyware)

<Callout icon="notebook" color="#c398e8">
  Currently, we do not have full access to all the PMS we work with, therefore it is not possible to generate a Rent Roll report in all of them.
</Callout>
