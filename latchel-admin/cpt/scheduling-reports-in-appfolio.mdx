title: "Scheduling Reports in Appfolio"
---

<Note>
This article will provide the step by step on how to automate reports in Appfolio during the onboarding process. The goal of this SOP is to **fully automate the delivery of the customer's General Ledger Report from Appfolio.** By setting up this scheduled report during onboarding, we ensure that Latchel receives consistent, accurate financial data every month without any further manual intervention. This saves valuable time, guarantees data integrity, and provides the foundation for our entire Assisted Ledgering program.
</Note>

## Instructions

Creating the Report

<Warning>
To be able to automate a report we first need to create the report with the correct day range and filters we need.
</Warning>

### **Step 1 - Configure the report filters to isolate Latchel-specific data**

a. Log in to the customer's Appfolio account.

b. Navigate to **Reporting > Reports** and select the **General Ledger** report.

<Frame>
  <img src="/images/CPT-101.png" alt="" />
</Frame>

c. In the **Customize Report** section, apply the following filters precisely:

- **Filters > GL Accounts:** Click "de-select all," then check the box for _only_ the Latchel GL code.
- **Filters > Date Range:** Select the second radio button and choose **"Last 30 days"** from the dropdown menu.

d. Click the **Update** button to apply the filters.

<Frame>
  <img src="/images/CPT-102.png" alt="" />
</Frame>

### **Step 2 - Save the configured report as a reusable layout**

a. After the report preview loads, click **Actions > Save Layout**.

<Frame>
  <img src="/images/CPT-103.png" alt="" />
</Frame>

b. A "Save Report" dialog box will appear. Configure it as follows:

- **Report Name:** Name it something clear, like **"Latchel Monthly Report."**
- **Checkbox:** Select the box for **"Lock this saved report to the selected dates."** This is critical to ensure the "Last 30 days" filter is always applied.

c. Click **Save**.

<Frame>
  <img src="/images/CPT-104.png" alt="" />
</Frame>

### **Step 3 - Create a new scheduled report job**

a. Navigate back to the main menu and go to **Reporting > Scheduled Reports**.

b. From the right-side "Tasks" menu, click **New Scheduled Report**.

<Frame>
  <img src="/images/CPT-105.png" alt="" />
</Frame>

<Frame>
  <img src="/images/CPT-106.png" alt="" />
</Frame>

### **Step 4 - Configure the report delivery and scheduling options**

a. Fill out the "New Scheduled Report" form with the following details:

- **Schedule Name:** Use the same name as your saved report (e.g., "Latchel Monthly Report").
- **Select Report Section:**

`- Saved Reports`: Choose the "Latchel Monthly Report" you created in Step 2.

`- Format`: Select **CSV**.

- **Email Options Section:**

`- Recipients`: Enter `accounting@latchel.com` and `ledgering@latchel.com`.

`- BCC`: Add the PM's email address here if they have requested a copy.

`- Subject`: `Latchel Monthly Reporting - [PM Company Name]`.

`- Message`: A simple message like "Monthly Reporting."

- **Schedule Section:**

`- Start Date`: Select the 1st day of the current month.

`- Frequency`: Set to **Monthly**

b. Click **Save**.

c. A preview will appear. Use this final screen to double-check that all settings (recipients, subject, report name) are correct before closing the window.

</Frame>
