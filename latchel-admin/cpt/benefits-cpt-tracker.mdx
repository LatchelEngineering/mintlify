<Check>

</Check>

### **Main Menu & PMS Access Info:**

<Frame>
  <img src="/images/CPT-17.png" alt="image-20250821-214814.png" />
</Frame>

**Main Menu: ** In this section, you will find useful information about the PM Company.

- **Company ID: **The code number that is used in Salesforce to identify a PM company.
- **Full Access to PMS: **This shows if a check mark is present, meaning you have complete access to the property management PMS.
- **Company Name:** The name of the PM company.
- **Last Update: **The last air filter review date.
- **CSM: **The Customer Success Manager assigned to the account.
- **Type of Subscription: **These options specify the shipment frequency and the number of filters per shipment.
- **Service Start Date: **The start date of the service.
- **Stage: **Shows if the account is Live or Onboarding.
- **PMS: **Property Management Software Type.
- **PMS URL: **The link to access the PMS.
- **Username:** Unique account name used to sign in.
- **Password: **Access credential for login.
- **GL Code: **A unique code from the property manager added to the resident’s account for the monthly charge.
- **Notes: **Important notes or details about the property manager.
- **Assigned CPT Specialist:** CPT team member assigned to the property manager’s account.
