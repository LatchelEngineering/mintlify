title: "How to: Generate a Transactions List Report in DoorLoop"

<Note>
  The purpose of this document is to be a guide for generating a Transactions List report in DoorLoop.
</Note>

<Check>
  The Transactions List Report allows you to examine transactions that impacted specific general ledger (GL) accounts over a given date range. This gives you the ability to see how GL accounts reached their balances.
</Check>

## Step by step

### Step 1: Login to DoorLoop

a. Go to Reports, and then Transactions List.

<Frame>
  ![](/images/CPT-343.png)
</Frame>

### Step 2: Customize the report

a. The report dashboard will display for you to customize it. You will be able to select the date range for the report. If you want to change the date range of the report, there are a couple of ways you can do this. You can use the Period Field and use one of the predefined date ranges or you can use the From and To fields to manually define a date range.

<Frame>
  ![00.png](/images/CPT-344.png)
</Frame>

b. The **Group By** filter is used to sort your report and arrange them by a specific set of data. Using the Transactions List Report, you can group all the data by Date, Type, Name, Account, Property, Payment Method, and **Deposit**.  This however will be different based on the report you are using.

<Frame>
  ![000.png](/images/CPT-345.png)
</Frame>

c. The **Type Filter** is used to filter for specific results only. For instance, if you only want to see Deposits, Charges, or **Payments,** that's what you would select.

<Frame>
  ![0000.png](/images/CPT-346.png)
</Frame>

The Charges filter will be used when we want to confirm how many GL codes have been added in Latchel’s GL Account, and the Payments filter will be used when we want to track how many of those charges have been actually paid.

d. The **Filter Panel** is where you'll find even more ways to filter your data. This can be enabled by clicking on the filter button, which is normally located to the right of the quick filter menu.

<Frame>
  ![0.png](/images/CPT-347.png)
</Frame>

e. Lastly, when you have customized the report as needed, you will be able to download it in Excel format. When clicking the button shown below, the file will be saved in .xlsx format to your default download location.

<Frame>
  ![00000.png](/images/CPT-348.png)
</Frame>
