The purpose of this SOP is to provide a step-by-step process on how to create a new _Piñata _tab in the CPT Tracker for a new property manager.

## Overview:

This SOP outlines the process for duplicating and setting up a new Piñata tab in the Benefits CPT Tracker for a newly onboarded property manager. The goal is to ensure the users can ensure the tab maintains the correct structure, formulas, and headers, making it ready for accurate Piñata data entry and reporting.
