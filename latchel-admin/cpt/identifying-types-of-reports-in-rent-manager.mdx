---
title: "Identifying Types of Reports in Rent Manager"
---

<Note>
  The purpose of this document is to describe the types of reports in Rent Manager, to be able to identify the particular features they offer, and recognize how they could be useful for our purposes.
</Note>

To access the available types of reports, log in to Rent Manager, click on the menu icon in the top left corner, and go to the Reports Section. The complete reports list will be displayed.

<Frame>
  ![](/images/CPT-272.png)
</Frame>

<Check>
  As CP Department, we use mainly the **Rent Roll** and **General Ledger Report.**
</Check>

Let’s take a look at the existing reports in the available sections for Rent Manager:

<Accordion title="Prospects">
  In Rent Manager, prospects are potential tenants. These individuals express interest in properties and rental units. The goal is to get prospects to sign leases and become paying tenants of the management company. Prospecting in Rent Manager is designed to help leasing agents and property managers manage prospect information. It also helps track the effectiveness of the leasing agents and marketing campaigns. Some reports available are:

  - Box Score
  - Prospect Listing
  - Prospect Stage Listing
  - Prospect Stage Comparison
  - Prospect Labels
  - Prospect Contact Labels
  - Prospect Waiting List
  - Leasing 12-Month Recap
  - Prospect Profile
  - Lost Prospect Breakdown
</Accordion>

<Accordion title="Rental Info">
  We can find in this section reports related to the resident information, deposits, leases, and violations. Some reports available are:

  - Tenant History / Notes
  - History / Notes
  - Tenant Labels
  - Tenant Contact Labels
  - Security Deposit Listing
  - Security Deposit Activity
  - Retail Sales
  - Retail Sales History
  - Lease Abstract
  - Tenant AutoPay Configuration
  - Month-to-Month Leases
  - Violations
</Accordion>

<Accordion title="Listings">
  The Listings section displays prospect information, unit preference, and information on the company's marketing efforts for prospects interested in the selected properties over a time range. It is also a great reference to initiate follow-ups with prospects. Some available reports are:

  - Tenant Listing
  - Tenant Profile
  - Property Listing
  - Unit Type Listing
  - Occupancy Listing
  - Rental Unit Listing
  - Tenant Phone List
  - Birthday / Age Listing
  - Architectural Requests
</Accordion>

<Accordion title="Rent Roll">
  The Rent Roll section displays a complete list of the recurring charges inherited by tenants as of the selected report date. Only current tenants (tenants with a move-in date on or before the report date, with no move-out date before the report day) with active recurring charges displayed in the report. Some available reports are:

  - [**Rent Roll & Recurring Charges**](https://rmxhelp.rentmanager.com/Topics/Express/Other/Report-Rent-Roll-Recurring-Charges.htm?Highlight=rent+roll)
  - Summary Rent Roll
  - Rent Roll Analysis
  - Market Rent Analysis
  - Rent Roll Comparison
  - Net Change in Recurring Charges
  - Commercial Rent Roll
</Accordion>

<Accordion title="Occupancy">
  The Occupancy Listing section displays leasing and contact information for tenants who are currently leasing units as of a selected date and their associated contacts. Some available reports are:

  - Unit Turnover
  - Unit Type Vacancy
  - Vacancy
  - Move in / Move out
  - Unit Availability Listing
  - Lease Expiration
  - Unit Status Chart
  - Inactive Unit Listing
</Accordion>

<Accordion title="Receivables">
  This receivables section displays reports with features used to charge tenants and prospects for goods and services and then collect payments on behalf of the managed properties. Some available reports are:

  - Aged Receivables
  - Aged Receivables with Notes
  - Delinquency
  - Receipts Breakdown
  - Charge Detail
  - Balance Due
  - Credit Detail
  - Charge Comparison
  - Payment Comparison
  - Unit Transaction Listing
  - All Receipts Deposited
  - Tenant Prepays
  - Recap Report
  - Last Paid Balance
</Accordion>

<Accordion title="Payables">
  The Payables section displays different reports related to bill paying features, bills, credit card transactions, vendor transactions, and vendor reports. Some of the available reports are:

  - Vendor Listing
  - Aged Payables
  - Open Aged Payables
  - Payables Listing
  - Payables By Account
  - Bills Paid
  - Vendor Labels
  - Aged Payables Detail
  - 1099 Vendor Breakdown
  - Vendor Transaction Listing
  - Unit Expense Detail
  - Vendor 1096
  - Vendor 1099
  - Vendor 1099 Copy B
  - Vendor Expiration Listing
  - Bill Worksheet
  - Vendor Balance
  - Vendor Expense
</Accordion>

<Accordion title="Banking">
  The Bank section\_ \_displays a list of the payments, journal entries, and deposits cleared when a user performs a [Bank Reconciliation](https://rmxhelp.rentmanager.com/Topics/Express/Accounting/Reconciliation/Bank-Reconciliation.htm). It also displays the cleared and actual bank balances as of the reconciliation date and the totals of the cleared payments and deposits. Some available reports are:

  - Check / Deposit Listing
  - Bank Reconciliation
  - Deposit Detail
  - Credit Card Expiration Listing
  - E-Pay Report - Tenant
  - Deposit Breakdown
  - Property Trust Account Balances
  - Trust Account Detail
  - Owner Trust Account Requirements
  - Checks By Account
  - ePay Breakdown
  - Credit Card Transaction Listing
  - Check Worksheet
  - Reconciliation Listing
</Accordion>

<Accordion title="General Ledger">
  The [**General Ledger**](https://rmxhelp.rentmanager.com/Topics/Express/Other/Report-General-Ledger.htm?Highlight=general+ledger+reports) section allows you to examine transactions that impacted specific general ledger (GL) accounts over a given date range. This gives you the ability to see how GL accounts reached their balances. Some available reports are:

  - General Ledger
  - Chart of Accounts
  - Journal Transactions
  - Unit General Ledger
  - Charge Type Links
</Accordion>

<Accordion title="Financial Statements">
  The Financial Statements section displays assets, liabilities, and equities from the GL start date to the end of the selected report date range for selected properties or ownership. It includes ending balances for all five of the major GL account types: Assets, Liabilities, Equity, Income, and Expenses. Some available reports are:

  - Balance Sheet
  - Balance Sheet Comparison
  - Unit Balance Sheet
  - Balance Sheet Property Comparison
  - Trial Balance
  - Trial Balance Activity
  - Profit & Loss
  - Profit & Loss Comparison
  - Profit & Loss Previous Year Comparison
  - Profit & Loss 12-Month Recap
  - Profit & Loss 12 Period Recap
  - Profit & Loss 12-Year Recap
  - Profit & Loss Multiple Periods by Year Recap
  - Profit & Loss Property Comparison
  - Profit & Loss MTD Comparison
  - Operating Expense Analysis
  - Fiscal Year Budget
  - Unit & Profit Loss
  - Unit & Profit Loss Detail
  - Cash Flow
  - Cash Flow MTD Comparison
  - Cash Flow 12-Month Recap
  - Budget Comparison
  - Actual / Budget Fiscal Year Analysis
  - Chart Account Activity Unit Breakdown
  - Financial Statement
  - Reserve Summary
  - Profit & Loss Forecast
  - Budget Exception
</Accordion>

<Accordion title="Tenant Statements">
  This section displays transaction information for selected tenants across a date range and provides information about the totals for charges, credits, and the current balance due. Some available reports are:

  - Post Card Statements 4 x 6, 3 per page
  - Statement 8.5 x 11
  - Statement 8.5 × 14 Mailer
  - Statement (Option 2)
  - Statement (Option 2) Lockbox
  - Payment Coupon
</Accordion>

<Accordion title="Service Manager">
  The Service Manager section provides access to service-related features such as issues and lists, service projects and templates, and list filters and options. Some available reports are:

  - Statistical Report
  - Top Issues Statistical Report
  - Service Order List
  - Issue List
  - Service Hours
  - Filtered Issue List
  - Issue Detail
  - Recurring Issue Detail
  - Project Status
  - Service Tech Hours
  - Recurring Issue List
</Accordion>

<Accordion title="Assets">
  The Assets section displays all the assets in the Rent Manager database. You can view asset information and details about the asset, such as Asset Status and Asset Type. Some available reports are:

  - Asset Listing
  - Warranty Expiration
  - Asset Aging
  - Overdue Asset Maintenance
  - Scheduled Asset Maintenance
  - Asset Profile
  - Fixed Asset Schedule
  - Depreciation Schedule
  - Asset Availability
</Accordion>

<Accordion title="Make Ready">
  The Make Ready section page displays additional information about a Make Ready process, including all actions and items within that process. Some available reports are:

  - Make Ready Detail
  - Make Ready Summary
  - Make Ready Expense
  - Make Ready Action Expense
</Accordion>

<Accordion title="Owners">
  In Rent Manager, owners are the people or organizations that own the properties the property manager manages. In this section, you can find reports of owner contributions, collect management fees, and owners pay reports of their equity distributions. It lets you view the general account and contact information for the currently selected owner. Some available reports are:

  - Owner Listing
  - Owner Open Payables
  - Owner Aged Payables
  - Owner Balance Sheet
  - Owner Bank Activity
  - Owner Statements
  - Owner Bank Balance
  - Owner Activity
  - Owner Profit & Loss
  - Owner Labels
  - Management Fee Listing
  - Distribution Breakdown
  - Owner 1096
  - Owner 1099
  - Owner 1099 Copy B
  - Owner 1099 Breakdown
  - Owner Available Cash Balance
  - Owner Bill Attachments
  - Owner Expense Detail
  - Basic Owner Statement
</Accordion>

<Accordion title="Owners Prospects">
  The Owner Prospects section displays all the owner prospects in Rent Manager database. You can view owner prospect information and optionally see details about the owner prospect. Some available reports are:

  - Owner Prospect Listing
  - Owner Prospect Box Score
</Accordion>

<Accordion title="PO / Inventory">
  The PO / Inventory section allows you to see reports related to purchase orders and inventory transactions. Some available reports are:

  - Purchase Order Listing
  - Inventory History
  - Inventory Reorder
  - Physical Inventory Worksheet
  - Inventory Valuation Summary
</Accordion>

<Accordion title="Sales Invoicing">
  The Sales / Invoicing section contains reports of features related to working with invoices and tracking sales data. Some available reports are:

  - Product Sales
  - Property Product Sales
  - Invoice Listing
  - Inventory Items Listing
  - Property Invoice Listing
  - Property Parts / Labor Sales
</Accordion>

<Accordion title="Estimates">
  The Estimates section displays all the estimates within the Rent Manager database. You can see the estimate information, see Inventory Items, and manage history/note items. Some available reports are:

  - Estimate Sales Representative Review
  - Items Closed from Estimates
</Accordion>

<Accordion title="Loans">
  Rent Manager provides a loan management feature to help users create and manage owner-financed loans. Owner-financed loans are typically used for home sales but can also be used for other loan types such as for land, equipment, appliances, auto, and so on. Some available reports in this section are:

  - Loan Listing
  - Transaction History
  - 1098 Interest Forms
  - Loan 1096
</Accordion>

<Accordion title="Metered Utilities">
  The Metered Utilities section lets see reports of tenant’s bills for usage of utilities such as gas, electric, water, and sewer. Every month, or billing period, individual meter readings are entered, and utility consumption charges are calculated and billed to the appropriate tenant. Some available reports are:

  - Utility Listing
  - Meter Type Listing
  - Utility Rental Units Listing
  - Utility Consumption
  - Utility Tenant Consumption
  - Utility Historical Consumption
  - Utility Reconciliation
  - Utility Field Entry
</Accordion>

<Accordion title="Commercial">
  The Commercial section allows you to see reports for defaults for commercial tenants, which affects the information tracked on their leases and lease abstracts. Some available reports are:

  - CAM Expense Breakdown
  - CAM Monthly Expense Breakdown
  - CAM Budgeted Expense Breakdown
  - CAM Monthly Budgeted Expense Breakdown
  - CAM Reconciliation Statement
</Accordion>

<Accordion title="System">
  The System section allows you to see reports related to system-level features such as email, user management, administration settings, Rent Manager updates, database maintenance and upgrades, and audits. Some available reports are:

  - Conversion Audit
  - Audit Trail
  - User Privilege Listing
  - Privilege User Listing
  - User Activity Log
  - MobileRM License Usage
  - Tenant Web Access License Usage
  - Owner Web Access License Usage
  - Text Usage
  - Text Usage by Property
</Accordion>

<Accordion title="Custom Reports">
  In the Custom Reports Section, you can customize a report according to the information needed. Some information you can include is related to properties, charges to include, and owner information.
</Accordion>

<Accordion title="Report Batches">
  Report Batches are collections of reports that can be generated simultaneously to save time and increase efficiency. The report options of each report within the batch can be configured to generate the report automatically with little to no input from the user. Report batches can also be scheduled to generate at a certain time and date, and the reports are then emailed to the specified addresses. In this section, you can find the following report:

  - Manage Report Batches
</Accordion>

<Accordion title="Memorized Reports">
  Memorized reports are reports that have been saved for later use by Rent Manager users. These reports can be quickly generated in the future without re-selecting the report options. It is also possible for you to adjust the settings so that a user running a memorized report is prompted to provide certain options. In this section, you can find the following report:

  - Manage Memorized Reports
</Accordion>

<Accordion title="Report Writer">
  The Report Writer section allows you to create your own custom reports to meet a business needs. Report Writer uses both insertable fields specific to the type of report, and scripting for more customizable options. In this section, you can find the following report:

  - Report Writer Manager
</Accordion>
