<Note>
  The aim of this document is to explain how this Marketplace benefit works, and how to send the monthly report to Cover Pest and the customer.
</Note>

Cover Pest is an add-on enhancement to Latchel’s standard benefits offering, that a Property Manager can choose to offer to their residents for pest control treatments.

<Warning>
  In **Coverpest**, once a contract reaches 12 months, **<u>it auto-renews unless the property manager cancels</u>**. Monitor the [#service-changes](https://latchel.slack.com/archives/CKA2W1KJA) Slack channel regularly for any “**Desire-to-Churn”** **notifications** or **cancellation notices**. Make sure to check regularly the service start date for your PMs assigned in Coverpest.
</Warning>

<u>By the 5th of the month</u> (or next business day), for each customer enrolled in this benefit, the CPT will prepare and send the monthly report to Cover Pest and the customer. CPT will include in this report, all the properties with GL code added until the previous month. **Example:** July 5th, CPT will send the report including all the properties that had a GL code added up to June 30th.

The property’s start date will be the next month of the date the GL code was added. **Example:** If the GL code was added in 6/15/2024, the start date will be July 2024.

The Tier description would depend on the subscription the customer signed for. If you are dealing with a new customer, confirm with the CS team the Property Manager’s type of subscription. <u>(Standard, Premium, Advanced).</u>

<Note>
  NOTE: The Tier can also be customized.
</Note>
