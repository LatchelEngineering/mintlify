title: "How to Get Access to the Property Manager’s PMS"
---

<Note>
  The purpose of this job aid is to provide a step-by-step guide on how to activate the account link received via email for newly onboarded property managers.
</Note>

## Step by step

### Step 1: Check the email

a. Go to the Assisted Ledgering email and **check the inbox for any new account activation link** that has been received.

<Frame>
  ![image-20250813-194701.png](/images/CPT-403.png)
</Frame>

### Step 2: Customize the account

a. Open the email and **click in “Create Your Account”**

<Frame>
  ![image-20250813-194802.png](/images/CPT-404.png)
</Frame>

b. A new window will appear, asking you to **enter the password** and click **Activate**.

<Frame>
  ![image-20250822-154518.png](/images/CPT-405.png)
</Frame>

<Check>
  Ensure to add this default info:

  - **Password:** CustomerObsessed123!
</Check>

c. Another window will then **confirm the phone number you should enter to receive an SMS** with a code to log in. Click on **Send verification code**.

<Frame>
  ![image-20250822-154741.png](/images/CPT-406.png)
</Frame>

<Check>
  Ensure to add this default info:

  - **Phone number:** 206-717-8374
</Check>

### Step 3: Login the PMS

<Warning>
  After you enter the phone number, you will receive an SMS to login the PMS.<u> Since the SMS **expires in 10 minutes**, make sure to grab it as soon as possible to enter the code. </u>
</Warning>

<Warning>
  If an agent grabs the SMS, they will post it in the [#ops\_info\_center](https://latchel.slack.com/archives/CNPUUL3PX) channel.
</Warning>

a. **Go to Latchel’s console** and look for the SMS received.

<Frame>
  ![image-20250813-195806.png](/images/CPT-407.png)
</Frame>

b. **Copy and paste the code on the PMS page to log in**. Make sure to close the SMS notification by clicking the three dots at the top of the SMS window.

c. **You will be logged in the PMS**.

### Step 4: Save the email received in the correct folder

a. Go back to the email and move it to the correct folder. If the folder hasn’t been created under the PMS Access folder, create it first.

<Frame>
  ![image-20250813-200158.png](/images/CPT-408.png)
</Frame>

<Warning>
  **Don't forget** to save the URL, username, and password <u>in the corresponding tracker where you recorded the PM. </u>
</Warning>

**If anything is missing** (e.g., GL code or required reports), escalate it to the CSM. For reference, see the following job aid: [How to: Add Newly Onboarded Property Managers to the Appropriate Tracker](https://latchel.atlassian.net/wiki/spaces/AL/pages/3584688155/How+to+add+newly+onboarded+Property+Managers+to+the+appropriate+tracker)
