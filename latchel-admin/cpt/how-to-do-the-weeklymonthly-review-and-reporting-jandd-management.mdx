title: "How to Do the Weekly/Monthly Review and Reporting J&D Management"
---

<Note>
The purpose of this SOP is to provide a step-by-step guide on how to ensure accurate weekly and monthly reviews for Filters by updating and reporting GL codes, filter size updates, and to outline the process for completing and submitting the Monthly Reports for Piñata and CoverPest for J&D Management.
</Note>

## Step-by-step guide

<Accordion title="Weekly Review for GL codes and Air Filters">
<Note>
The purpose of this SOP is to provide a step-by-step guide on how to review and update property details using the Rent Roll report and ensure all GL codes and benefits are accurately recorded in the Benefits CPT Tracker.
</Note>

<Warning>
Check the PM's note in **any of the tabs for the Overview section in the Benefits Tracker** to review the details of each benefit and the corresponding amounts.
</Warning>

<Frame>
  <img src="/images/CPT-571.png" alt="image-20250806-154818.png" />
</Frame>

### Step 1: Log into the PMS dashboard.

a. Check the PM’s login credentials in the Overview section in the Benefits Tracker.

<Frame>
  <img src="/images/CPT-572.png" alt="image-20250806-155618.png" />
</Frame>

b. Log in to PMS.

<Frame>
  <img src="/images/CPT-573.png" alt="image-20250806-222605.png" />
</Frame>

b. Navigate to the top menu and click on **Reporting**.

c. Select **Reports** from the dropdown menu.

d. Click on **Rent Roll (Itemized)** to open the report.

<Frame>
  <img src="/images/CPT-574.png" alt="image-20250806-160000.png" />
</Frame>

### **Step 2: Filter and Update the Rent Roll Report.**

a. Search for the GL Code.

b. You can find the GL code for the Property Manager (PM) in the **Benefits Tracker **under the **Overview **tab.

c. Set the Filters.

d. **Units:** Select **"Show Active Units"**

e.** As of Date:** Enter the date of your review. Since this is a weekly review, you must update the date to the current date on which you are doing the review. For the **Monthly Review**, ensure to set the date as of the last day of the month. _**Example:** It’s August 03rd, you will do the last review for July, please select 07/31/2025._

f. Click** "Update"** to generate the filtered report.

<Frame>
  <img src="/images/CPT-575.png" alt="image-20250806-160728.png" />
</Frame>

g. Ensure the following 9 columns are selected:

- Unit
- Status
- Property Address 1
- Property City
- Property State
- Property Zip
- Tenant
- Lease From
- Lease To

h.** Click on actions >** Export as **CSV.**

<Frame>
  <img src="/images/CPT-576.png" alt="image-20250806-161458.png" />
</Frame>

### **Step 3: Prepare the Spreadsheet.**

a. Open a **new spreadsheet.**

b. Click the top-left corner to **select all** → **Enable Filter.**

<Frame>
  <img src="/images/CPT-577.png" alt="image-20250806-161900.png" />
</Frame>

c. In your new spreadsheet, go to **File > Import**.

d. Click the **Upload** tab.

e. Click **Browse** and select the **CSV file** you exported from PMS.

f. When prompted, choose **"Replace data at selected cell"**.

g. Click **Import data** to load the data into your spreadsheet.

<Frame>
  <img src="/images/CPT-578.png" alt="image-20250806-162539.png" />
</Frame>

h. Move **Column A (Unit) to Column C** and **Column E** (GL code) to **Column J **to match the required format.

- Status
- Property Address 1
- Unit
- Tenant
- City
- State
- Zip Code
- Lease To
- Lease From
- Resident Benefits Package

<Frame>
  <img src="/images/CPT-579.png" alt="image-20250806-171008.png" />
</Frame>

<Warning>
**Note:**_ _
_<u>First, verify if the report displays the total number of properties in order. If it is displayed, there is no need to apply the filter twice, as is the case for J&D.</u>_

_If the report contains more than 300 properties, click Column B, right-click, and select "Sort sheet A to Z"._
</Warning>

<Frame>
  <img src="/images/CPT-580.png" alt="image-20250806-163414.png" />
</Frame>

<Frame>
  <img src="/images/CPT-581.png" alt="image-20250806-163734.png" />
</Frame>

i. Select and copy all necessary data from the report.

j. Navigate to the _Benefits CPT Tracker_ and paste the copied data into the appropriate tab (PM company name + GL codes) and section (From **Column AC or AC**,** depending on the tab of the PM**)

** **_**Note: **Use **CTRL+SHIFT+V** to paste values only, ensuring formatting is not disrupted. Make sure not to click anywhere without first checking whether any properties need to be deleted at the end of the newly pasted report._

k. Double-check that all data has been correctly pasted and aligned with the correct columns.

### **Step 4: Compare Current and Previous Reports.**

a. Focus on** Columns** **K to N** (with "NO") to identify the properties that need to be changed.

b. Compare the data with the newly pasted PMS report.

<Frame>
  <img src="/images/CPT-582.png" alt="image-20250806-171525.png" />
</Frame>

### **Step 5:  Review the type of property.**

a. If a Property is marked as  Vacant in the new report but had a GL Code in the previous report, side:

- Leave a note in **Column AC. **_**Example: **You are doing the weekly review on August 15th, and the GL code was deleted. You will leave a note stating: Had GL code in August and highlight with a color. _

<Warning>
Check the PM's note in the Filters or Piñata Overview section of the Benefits Tracker to review the details of each benefit and corresponding amounts, and verify if the GL code includes COVERPEST. If so, add a NOTE in Column AC. _“Need to be included in Coverpest Report (August)”_
</Warning>

<Note>
RBP1: (RBP [Filters and Piñata]+INS [OBIE]+PEST [CoverPest])  $66.95 / $69.95 / $69.99

RBP2: (RBP [Filters and Piñata]+INSUR [OBIE]) - $59.95

RBP3: (RBP [Filters and Piñata] +PEST [Cover Pest]) - $54.95

RBP4: (RBP BASE [Filters and Piñata]) - $40 / $44.95
</Note>

<Frame>
  <img src="/images/CPT-583.png" alt="image-20250806-172301.png" />
</Frame>

### Step 6: Handling Properties Marked in Red Despite GL Code Presence

a. If the property is marked in red, even if it has a GL code, follow the PM's advice: no filter size is needed, and filters should not be shipped. Add a note in **Column AC **_(Note)_.

<Frame>
  <img src="/images/CPT-584.png" alt="image-20250806-173355.png" />
</Frame>

_**Tip: **If there is a new property with a GL code, add it to the Filters report and to the CoverPest report if the GL code covers CoverPest._
</Accordion>

<Accordion title="Monthly Review for Air Filters">
<Note>
The purpose of this SOP is to provide a step-by-step guide on how to complete and send an accurate Monthly Filter Report by reviewing all GL-coded properties, updating the Benefits CPT Tracker, verifying changes, and ensuring the Air Filters report is submitted to the property manager accordingly.
</Note>

<Warning>
_The weekly review for Filters must always be completed first before proceeding with the Monthly Filter Review._
</Warning>

### Step 1: Go to the **Benefits CPT Tracker**

a. Filter the properties with GL codes from the Benefits Tracker.

<Note>
**Example**: _123 Almond road is marked in blue, which means this is either a new property that needs to be added in the report, or the property had a GL code during the previous month, and it should be reported in this month._
</Note>

<Warning>
**Important:** _When filtering the GL codes, select both **Column I** (Resident Benefits Package) and **Column AC **(Notes), then uncheck any entries labeled “Disregard from filter shipment” to ensure properties marked in red are excluded._
</Warning>

<Frame>
  <img src="/images/CPT-585.png" alt="image-20250806-204010.png" />
</Frame>

b. For the monthly Filters report, include all GL codes except those marked in **RED.** Apply a filter in **Column I (Select all the GL codes) and simultaneously apply a filter in the Notes column to exclude any entries labeled ‘Disregard from Filters shipment’. (Unchecked it)**

<Frame>
  <img src="/images/CPT-586.png" alt="image-20250806-232020.png" />
</Frame>

c. Search for each property in the duplicated Monthly Report to verify whether it is already included. This will help identify any newly added properties that need to be reflected in the Monthly Report. To easily identify new properties to be added to the report, you can mark them with a color.

<Frame>
  <img src="/images/CPT-587.png" alt="image-20250806-230519.png" />
</Frame>

<Note>
Remove the Filter in **Column I** _(Resident Benefits Package) and **Column AC **(Notes)_
</Note>

d. Filter **Column AC **(Notes) and search for the 'Had a GL code in (Month)'. **For example**, _if you are conducting the review on **September 3rd**, search for entries from **August.**_

<Frame>
  <img src="/images/CPT-588.png" alt="image-20250806-205358.png" />
</Frame>

<Frame>
  <img src="/images/CPT-589.png" alt="image-20250806-205615.png" />
</Frame>

e. Count **ALL **GL codes.

- Click and hold from the top of the column, then scroll down to view the total number of GL codes.

<Frame>
  <img src="/images/CPT-590.png" alt="image-20250806-232857.png" />
</Frame>

<Note>
Remove the Filter in **Column** AC_ (Notes)_
</Note>

f. Add the totals from GL code counts in **Column I** _(Resident Benefits Package)_ and **Column AC** _(Notes)_ to obtain the overall Air Filters count for the **Monthly Report.**

<Warning>
**Important: **_Properties marked in red with a GL code should not be counted for Filters._
</Warning>

### Step 2: **Billing Count.**

a. Go to **Billing Input Sheet.**

b. Select the corresponding **month tab.**

c. Search the PM’s name.

d. Update **Column Q** with the Total Air Filters count.

<Frame>
  <img src="/images/CPT-591.png" alt="image-20250806-210150.png" />
</Frame>

### **Step 3: Search for the previous Monthly report sent. **

a. Open Assisted Ledgering Gmail.

b. Search for the PM's previous monthly filter report.

<Frame>
  <img src="/images/CPT-592.png" alt="image-20250806-183504.png" />
</Frame>

<Warning>
**Important:** _Send the Monthly Reports for J&D (Filters, Piñata, and CoverPest) to the PM in a single email._
</Warning>

<Frame>
  <img src="/images/CPT-593.png" alt="image-20250806-184025.png" />
</Frame>

### Step 4: **Open the Monthly Report For Filters Only.**

<Frame>
  <img src="/images/CPT-594.png" alt="image-20250806-203126.png" />
</Frame>

a. Right-click on the tab, select **'Duplicate,'** and rename the duplicated tab with the corresponding date and month of your review, following the required format.

<Frame>
  <img src="/images/CPT-595.png" alt="image-20250806-203440.png" />
</Frame>

b. Cross-check the Monthly report with the AU tab and the GL Codes tab for accuracy.

c. This ensures the monthly filter list sent to the PM is always up to date.

<Check>
**Note:**

- _Delete rows for properties with no GL Code (In the Monthly report) unless they had a GL code in the month you’re reporting. _
- _Update the filter size details as needed. If there are any changes, you can verify the updated filter size(s) in the Benefits Tracker under the AU tab._
</Check>

### Step 6: Send the Report.

_If this is the first time you are sending the report, you should request the CSM for the POC information to send the report._

a. Add the same Recipients that appear in the “To” and “CC” sections (refer to the photos below).

- Subject: **J&D Management Latchel Filters / Piñata / Cover Pest Monthly Update**

<Frame>
  <img src="/images/CPT-596.png" alt="image-20250806-194756.png" />
</Frame>

b. Copy the template from the previous email and ensure that all details correspond to your current monthly report.

<Frame>
  <img src="/images/CPT-597.png" alt="image-20250806-195348.png" />
</Frame>

c. Click the “drive” Icon

<Frame>
  <img src="/images/CPT-598.png" alt="image-20250806-195720.png" />
</Frame>

d. Search for the Air Filters Report by typing the name of the Property Manager (PM) and the report title.
**Example:** _J&D Management_

<Frame>
  <img src="/images/CPT-599.png" alt="image-20250806-211650.png" />
</Frame>

<Warning>
Attach the report and first verify that it is the correct file by clicking on it before sending. If the file is incorrect, make sure to unselect it and attach the correct one.
</Warning>

e. Send all reports to the PM in a single email.

**Note:** Send the email to the PM as soon as the three reports _**(Air Filters, CoverPest, and Piñata)**_ are completed, and attach them in a single email.

- **Click Send.**

<Frame>
  <img src="/images/CPT-600.png" alt="image-20250806-211939.png" />
</Frame>

_**Tip: **If there is a new property with a GL code, add it to the Filters report and to the CoverPest report if the GL code covers CoverPest._
</Accordion>

### Step 5: **Open the Monthly Report For CoverPest Only.**

<Frame>
  <img src="/images/CPT-610.png" alt="image-20250806-191714.png" />
</Frame>

a. Right-click on the tab, select **'Duplicate,'** and rename the duplicated tab with the corresponding date and month of your review, following the required format.

<Frame>
  <img src="/images/CPT-611.png" alt="image-20250806-184323.png" />
</Frame>

b. Copy only the properties that are **NEWLY** added. Do not copy all properties, as those from the previous month's report are assumed to be already included. Also, include properties noted as **"Need to be included in CoverPest Report (Month)"** if the month matches your reporting period.

_For example: If you are sending the report on September 3rd, the month in the notes should be 'Need to be included in CoverPest Report (August).' Then paste them into the Monthly Report._

c. For CoverPest, every time a GL code is added for the corresponding month, the start date in the report should be set to the following month.

For example, if a GL code is added on August 6th, set the start date in the report as September.
