<Note>
  **Purpose of this document**

  The purpose of this document is to share the steps on how to scrub a PMS user list to use for email campaigns.
</Note>

<Note>
  **Who should read this document?**

  Anyone in marketing or sales who is trying to pull a list of property management companies who use certain property management systems that potentially fit our ideal client profile.
</Note>

## Step-by-Step Guide

### Process Overview

### Step 1 - Log into SEMrush

Log into SEMrush

### Step 2- Backlink analytics

Select "backlink analytics" on the left sidebar

<Frame>
  ![SEMrush left sidebar with Backlink analytics option highlighted](/images/Sales-196.png)
</Frame>

### Step 3 - Put PMS tenant portal link in the top bar

Put the PMS portal link in the top bar. Below are the tenant links for each PMS:

**Appfolio:**

- [https://passport.appf.io/sign\_in?idp\_type=tportal&vhostless=true](https://passport.appf.io/sign_in?idp_type=tportal&vhostless=true)
- appfolio.com/connect/users/sign\_in

**Buildium:**

- [https://signin.managebuilding.com/Resident/portal/global-login](https://signin.managebuilding.com/Resident/portal/global-login)

**Rent Manager:**

- [twa.rentmanager.com](http://twa.rentmanager.com)

**Propertyware:**

- [https://app.propertyware.com/pw/index.html#/login/tenant/](https://app.propertyware.com/pw/index.html#/login/tenant/)

<Frame>
  ![SEMrush Backlink Analytics with a PMS tenant portal domain entered in the search bar](/images/Sales-197.png)
</Frame>

### Step 4 - Select "root domain"

Select "root domain" from the drop down next to the PMS tenant log in URL and hit "search."

<Frame>
  ![SEMrush root domain option selected in the search dropdown](/images/Sales-198.png)
</Frame>

### Step 5 - Select "referring domains"

Select "referring domains"

<Frame>
  ![SEMrush Referring Domains tab selected in Backlink Analytics](/images/Sales-199.png)
</Frame>

### Step 6 - Select "active"

Select the tab "active"

<Frame>
  ![SEMrush Active tab selected to filter referring domains](/images/Sales-200.png)
</Frame>

### Step 7 - Filter by country

Select "\+ Add Filter" and select "country" and "United States"

<Frame>
  ![SEMrush Add Filter menu with country filter set to United States](/images/Sales-201.png)
</Frame>

### Step 8 - Filter by authority score

Change the authority score range from "0-30"

<Frame>
  ![SEMrush authority score filter set to a range of 0-30](/images/Sales-202.png)
</Frame>

### Step 9 - Export to a .csv file

Click "export" in the right top corner and select ".csv"

<Frame>
  ![SEMrush export menu with .csv file option selected](/images/Sales-203.png)
</Frame>

### Step 10 - Send to BI Team for enrichment

Send the .csv file to Diego (or BI team) to enrich. The enrichment items should include emails, first name, company name, and phone number. The BI team should also de-dupe this against our database to remove any existing customers or leads that are already enrolled in campaigns.
