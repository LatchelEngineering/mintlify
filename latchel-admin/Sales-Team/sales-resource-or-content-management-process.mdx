## Requests to Create New Resources or Update Current Ones

There are 2 ways a Sales team member can request a resource to be created or updated:

- Via workflow in the Slack Channel #sales-change-request (Recommended)
- By accessing the [Sales Library Management Jira board](https://latchel.atlassian.net/jira/software/c/projects/SLM/boards/66/backlog) and creating a ticket to the backlog (Less Preferred)

## Slack Channel #sales-change-request (Recommended)

1. To create a request, go to the #sales-change-request channel in Slack.
2. At the top of the channel, there is a bookmark section. There you will find the "Sales Resource Request" workflow

<Frame>
  ![Sales Resource Request Workflow](/images/Sales-7.png)
</Frame>

3. The workflow will request to fill in some information about the request, for example:

- Which confluence page needs to be updated or if a new resource needs to be created
- What type of request it is, e.g. incorrect or obsolete information, missing information, unclear information, small improvements, or request for a NEW resource
- Specify what part of the Confluence page needs to be adjusted, including if it is a link to an external file. This is to help to find where the update needs to happen. If it is a request for a new resource, just write "NEW"
- What is the information you want to add or correct. Or suggestions on what the final product needs to look like or information it needs to be included

4. Once the workflow is created, a new "issue" or "ticket" will be created in the Jira board

## Sales Library Management Jira Board

1. If you prefer to create the request directly in Jira, you can access the [Jira board](https://latchel.atlassian.net/jira/software/c/projects/SLM/boards/66/backlog) backlog page, scroll to the bottom of the page and click on "Create Issue"

<Frame>
  ![Create Issue in Jira](/images/Sales-8.png)
</Frame>

2. Type the name of the request and press enter

<Frame>
  ![Type Request Name](/images/Sales-9.png)
</Frame>

3. Click on the issue created to add more information into the ticket description

<Frame>
  ![Add Ticket Description](/images/Sales-10.png)
</Frame>

4. In order to ensure all of the information is needed to execute on the request, please include the same information as requested in the Slack form

## Communication of Changes - Published New Resource or Changed an Existing one

After the [Sales Library](https://latchel.atlassian.net/wiki/spaces/SAL) is launched, we will start announcing in the Growth channel??? Slack channel any new resource created or changes made to current resources.

Later, we will consider expanding to other communication channels, like Sales WBR, Team Meeting, Publishing, and/or Newsletters
