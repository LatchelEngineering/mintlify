## Property Types and their Fit with Latchel

| Property Type | Supported | Maybe | Not Supported |
| --- | --- | --- | --- |
| Single Family | This is the bread and butter! |  |  |
| Condos | In condos where property managers are responsible for a single-family home, Latchel can help with these. There are also a number of condo owners who engage property managers to rent out condos, Latchel can support this scenario as well. |  |  |
| Independent Living Communities | Latchel can help with independent living communities for those age 55 or older. In this scenario, the older adults need little to no assistance with daily living and activities. Most of these communities do not offer any type of medical care. |  |  |
| HOA Properties |  | For condos or single family where property managers act as an HOA, these properties are not in Latchel's wheelhouse. But can be supported on a case by case basis, for property managers bringing on a number of rentals. Reach out to your manager with the opportunity to discuss |  |
| Assisted Living Communities |  |  | For assisted living homes, Latchel does not get involved in these communities because medical care is offered. |
| Student Housing |  | We have yet to successfully implement Latchel in student only housing. It would make sense and be highly profitable. But the problem is all of the leases start on the same day. In the past if we get someone excited about Latchel that is in student housing we have passed them through for a conversation. More commonly we will see PM companies that have student housing in the portfolio, combined with normal SFRs or Multi-family. They'll qualify. Jan 2024: We just sold 2 midsize student housing deals, they don't start until June 2024, so we don't know how that will go, but the sales were great. THIS PROPERTY TYPE REQUIRES SPECIAL SET-UP |  |
| Rentals by bed |  | This is similar to student housing, short term rentals typically are not a match with Latchel. Each room is a lease. it can be a fit in most cases as long as they are long term leases. |  |
| Mobile Homes |  | Only if they are managing rentals. Call and ask if they manage rental mobile homes, or if they only manage owned mobile homes. 9 times out of 10 the mobile homes are owned and maintained by the person living in them - not by the management company. The management company is responsible for the grounds outside the home (similar with HOA). If the PMs manage rented mobile homes and they qualify- Pass them through! |  |

## Properties Requiring Special Set-Up in Latchel

In situations where the property set-up is not clear as in a single-family unit, the Sales team needs to understand how the lead have it set-up on their end (Property Management System) and if that will require a different or unusual set-up in Latchel:

<Warning>
  **GENERAL RULE TO SET-UP PROPERTIES IN LATCHEL**

  General Rule is that Latchel should match how their PMS (Property Management System) is set up

  - If the PM has individual leases for each bedroom, set-up an individual unit in Latchel
  - If the PM has a lease for the property with multiple residents, set-up the property as an unit in Latchel with multiple residents

  Any special set-up needs to be documented by Sales in the "Contract Modification" field in Salesforce and in Company notes in Latchel.

  If the PM doesn't have any set-up yet, our suggestion is that each rentable unit, bed or room, should be its own property in Latchel, so they can get their own benefits.
</Warning>
