## Step-by-Step Guide

### Go to Google Admin Page and Log in

[https://admin.google.com/ac/home?hl=en](https://admin.google.com/ac/home?hl=en)

<Frame>
  ![Google Admin Page](/images/Sales-11.png)
</Frame>

### Find the User

1. In the User Section, click on the drop-down menu

<Frame>
  ![User Section Drop-Down](/images/Sales-12.png)
</Frame>

2. Click on "manage"

<Frame>
  ![Manage User](/images/Sales-13.png)
</Frame>

3. Filter the team by organization to make it easier to find the user

<Frame>
  ![Filter by Organization](/images/Sales-14.png)
</Frame>

### Reactivate the User

1. Hover over the name, or click on the three dots, and you will see a menu where you can select to "Reactivate"

<Frame>
  ![Reactivate User Menu](/images/Sales-15.png)
</Frame>

If this doesn't work out, you can also try to follow the steps on this help page: [Restore a suspended user | User management | Google Workspace Help](https://support.google.com/a/answer/1110339?hl=en)
