## Create the Templates

When building a cadence, you will select calls and emails that will be sent to the lead. Emails can be sent manually or automatically. For both, calls and emails, you will have to select a template to be used in the cadence, therefore, unless those templates are already created, the first step is to create them.

### Call Scripts Templates

1. Go to the 9 dots on the top left side of the navigation bar, type "scripts", and click on "Call Scripts"

<Frame>
  ![](/images/Sales-88.png)
</Frame>

2. On the top right side, click on "New"

<Frame>
  ![](/images/Sales-89.png)
</Frame>

3. Fill up the form:
   - Name of the Call Script – use the template: CT - SA - \[situation or cadence in which it will be used\]
   - Note - CT stands for Call Template - using Call Template vs. Call Script verbiage to avoid confusion with the CS from Customer Success Team
   - Description - include here when or which situation this script should be used
   - Body - enter the script or the link to another place where scripts are being stored

<Frame>
  ![](/images/Sales-90.png)
</Frame>

<Note>
  **Cloning current templates**

  Instead of creating a new template from scratch, you can also create a clone, edit the content and Save! Just find the email template you want to clone and click on it. On the drop right side of the window, click on "clone".
</Note>

To find the Object API name,:

- Go to Set-up - [https://latchel1.lightning.force.com/lightning/setup/SetupOneHome/home](https://latchel1.lightning.force.com/lightning/setup/SetupOneHome/home)
- Click on Object Manager tab
- Search for the Object

<Frame>
  ![](/images/Sales-101.png)
</Frame>

- Click on the object and then click on "Fields and Relationships"

<Frame>
  ![](/images/Sales-102.png)
</Frame>

- Search for the field you are looking for, like name
- Then find the name you are looking for in and get the value of the second column (i.e. under Field Name)

<Frame>
  ![](/images/Sales-103.png)
</Frame>

Once you have the "Field Name", you will combine it with either Sender, Recipient or Organization using this syntax:

`{{{Recipient.FIELD_NAME}}}`

Example `{{{Recipient.FirstName}}}`

Replace all customs fields with this syntax into your email template. With this, the automation will fill in the contact specific information in that field, or your own personal information, like your name and calendar link

Some key syntaxes to use in your templates:

- Pull lead name ---\> `{{{Recipient.FirstName}}}`
- Include your calendar link ----\> `{{{Sender.Calendar_Link__c}}}`
- Include your name ----\> `{{{Sender.FirstName}}} {{{Sender.LastName}}}`

You may also add highlights, change font type and size, change background color, etc.

Once you are done creating the template, click on Save

### Cadence Builder 2.0 - Rules Based Cadence

1. Name the cadence
   Use this template: CA - SA - \[situation it should be used\]
   Example: CA - SA - Inbound Tier 1
2. Add a description with more details on what the cadence includes, like calls and emails
3. The Cadence has 3 tracks:
   - **Main** - it will include all actions for the SDR to take until we either receive a positive or negative response from the lead, or the exit criteria is met. You will need to define the exit criteria
   - **Positive** - it will include all actions for the SDR to take when there was a positive response from the lead. You will need to choose what defines a positive response or situation.
   - **Negative** - it will include all actions for the SDR to take when there was a negative response from the lead. You will need to choose what defines a negative response or situation.
4. Set the **Exit** criteria for the Main Track
   Exit criteria defines the situations in which the contact will reach the end of the cadence, i.e. the contact will not longer be a target of any action in the cadence.
   In the example below, the contact will remain in the cadence unless it is classified as unqualified.

<Frame>
  ![](/images/Sales-108.png)
</Frame>

5. Set the Entry criteria for the Positive Track
   The Entry criteria defines the rules that will be used to change the cadence track from the main track to the positive track. This means, for example, if the contact has a positive conversation with the SDR or replies to an email, or calls us and leave a voicemail, the main track actions will be interrupted and the actions from the positive track will start

<Frame>
  ![](/images/Sales-109.png)
</Frame>

6. Add the actions that we will take until either we get a positive or negative situation, or meets the exit criteria
7. Set the Entry criteria for the Negative Track
   The Entry criteria defines the rules that will be used to change the cadence track from the main track to the negative track. This means, for example, if the contact has a negative conversation (not interested) with the SDR, the main track actions will be interrupted and the actions from the negative track will start

<Frame>
  ![](/images/Sales-110.png)
</Frame>

8. Add the actions that we will take until either we get a positive or negative situation, or meets the exit criteria.
   To add an action, first select the track you want to add the actions to
   Go to the actions list, click on the one you want to add, drag and drop in the center of the screen.
   Then you will need to configure the action. For example, if it is an email, select the email template to be used (if it is a call, select the call template), add a description, when the action will be triggered to start based on the prior step, and the due date when it needs to be completed by, or it will be marked as missed

<Frame>
  ![](/images/Sales-111.png)
</Frame>

## Edit or Modify a Cadence

To Edit or Modify a Cadence, go to the cadence you want to adjust and click on it

1. Click on the nine dots icon in the navigation bar
2. Type cadences
3. Select All Cadences

<Frame>
  ![](/images/Sales-112.png)
</Frame>

You can edit the name and description by clicking on "Edit", or you can modify the content of the cadence, like steps, rules and templates being used, by clicking on "Modify Steps"

<Frame>
  ![](/images/Sales-113.png)
</Frame>
