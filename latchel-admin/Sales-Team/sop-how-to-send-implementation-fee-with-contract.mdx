<Note>
  Purpose of the SOP: to understand and use the process of sending out implementation fees to new customers with contract.
</Note>

## Send the Implementation invoice link with the Panda Doc Contract

1. Here are the links for the invoice to generate:
   1. If they are under 500 doors: [\$500 Implementation link](https://buy.stripe.com/aEU17X1nH0gi9ZS000)
   2. If they are over 500 doors: [\$1,000 Implementation link](https://buy.stripe.com/9AQbMBc2l8MO3BuaEF)
2. When sending the contract in Panda doc, there is a message field. Fill that out with the following copy:
   1. We are so excited to partner with you! Here is the link to pay your implementation fee \[insert correct link\], this must be paid before we can schedule a kick-off meeting. Let me know if you have any issues/questions.

<Frame>
  ![Screenshot of Panda Doc send document screen with an arrow pointing to the implementation fee message in the email message field](/images/Sales-162.png)
</Frame>

3. Also send a follow up email via gmail with the link to ensure they got it.
4. Login into stripe and check if they paid
5. If they have not paid by kick-off meeting, let them know it needs to be rescheduled or paid immediately
