This document explains the help guide process to follow when a new product is released in the [#product-release](https://latchel.slack.com/archives/C038Z6M0T6H) Slack channel.

‌

- Starting on April 20, 2023, all newly released products and features will be monitored and recorded in the <custom data-type="smartlink" data-id="id-0">[https://docs.google.com/spreadsheets/d/1Yvmgl-3IcOCAqzWwIRpAy2NrTdexlIOWEWRohatbtK8/edit#gid=1711766912](https://docs.google.com/spreadsheets/d/1Yvmgl-3IcOCAqzWwIRpAy2NrTdexlIOWEWRohatbtK8/edit#gid=1711766912)</custom> sheet.
- Additionally, all products and features released before the date can be found in the <custom data-type="smartlink" data-id="id-1">[https://docs.google.com/spreadsheets/d/1-bMOFp2GtsWuSCDMno_x1akEBLwsnKW2zq6uimfTFDQ/edit#rangeid=1271925802](https://docs.google.com/spreadsheets/d/1-bMOFp2GtsWuSCDMno_x1akEBLwsnKW2zq6uimfTFDQ/edit#rangeid=1271925802)</custom> sheet.
- The Data Support Supervisor (<custom data-type="mention" data-id="id-2">@Malen Macasaquit</custom>) is responsible for monitoring both sheets.
- For every release, the team must sort through the sheet and identify if there are existing help guides related to the product or if new ones need to be created, providing clear steps for the audience of the new product release.
- To ensure all necessary help guide requests are monitored for new product releases, a request must be posted in the [#help-guide-requests](https://latchel.slack.com/archives/C04B3PNJ24V) channel, following the steps in <custom data-type="smartlink" data-id="id-3">[https://latchel.atlassian.net/wiki/spaces/TST/pages/2635333691](https://latchel.atlassian.net/wiki/spaces/TST/pages/2635333691)</custom> job aid.
