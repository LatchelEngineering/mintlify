# Objective

**What is the Customer Success Shared Drive?**

Property Managers will send Latchel documents for their account data. For example, we can receive property and resident data, owner data, preferred Service Provider data, onboard recordings, etc. We house all of these documents in the [Customer Success shared drive](https://drive.google.com/drive/folders/1n9ZSyi53tgTt0OK1covgFQEodFn-xh-F) which is accessible to the Customer Success Team and other team members that support with data entry. In this SOP, we will define the steps needed to create a folder for a Property Manager within the shared drive and how to add their specific files to their folder. 

# Content

### **Step 5 - Rename the File**

**To write an effective file name,** 

1. It should be named consistently.
2. It should be short but descriptive (<25 characters)
3. Avoid special characters or spaces in a file name.
4. Use capitals, dashes, and underscores instead of periods or spaces or slashes.
5. Always add a date to the file name so we know when it was received/uploaded 

![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=72cc4190-2499-4116-99d4-94388c0d84de&&collection=contentId-865469170&height=488&occurrenceKey=null&width=1076&__contextId=null&__displayType=null&__external=false&__fileMimeType=null&__fileName=null&__fileSize=null&__mediaTraceId=null&url=null)
1. Right-click the file you wish to rename
2. Select **Rename** from the drop-down options
3. Rename the file
4. Click **OK** to save

### **Step 6 - (Optional) Share the Files with the Property Manager**

If applicable, you can share files with the Property Manager. You will not be able to share the folder, however, you can select all files within the folder at once to share the files with the Property Manager.

1. Double-click to open the folder for the Property Manager from this list of folders [here](https://drive.google.com/drive/folders/1K-OxCEHlyfI6GNhukN9aNPsZho1yzz7T). 
2. Click the first file and drag your courser down the list of files to select all. The file will be highlighted blue if selected.

    1. 
    
3. Click the person icon after you have selected the files you want to share with the Property Manager.

    1. 
    
4. Type the email address for the Property Manager and then press the return key on your keyboard to save. 

    1. 
    
        1. You will want to make sure the Property Manager has editing permissions
        
    2. Click **Send**. This will send the Property Manager an email with the shared file link.
