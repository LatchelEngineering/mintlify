# Objective

**Purpose**

Access instructions will automatically populate on work order creation if a property has default access instructions

If the Property Manager has lockbox codes or master key instructions for service providers, you should configure each property with default access instructions. These instructions will automatically appear on work orders sent to service providers. This is especially useful when we get permission to enter from residents because it simplifies scheduling and doesn’t require the resident to be on site for the service provider to perform work.

Access instructions can be configured at the company level, property level, or portfolio level. If configured at the portfolio level, this will override the company level access instructions. If configured at the property level, this will override portfolio level and company level access instructions. 

# Content

**What is happening prior to this process?**

You have received instructions from the Property Manager to configure properties, portfolios, or their account to include default access instructions for Service Providers. Do not add instructions to enter a property without approval from the Property Manager.

## Step-by-Step Guide

### **Step 1 - How to add Default Access Instructions at the Property Level**

1. Go to the Property Manager Profile page:[How to: Access the Property Manager Profile Page](https://latchel.atlassian.net/wiki/spaces/TST/pages/813729656)
2. On the Property Manager's account, click the **Properties** tab.
3. Search for the property and click on the **address**.
4. Under the **Details** tab, scroll down to the "Maint. Communication Settings" section. From here, you can update or add Default Access Instructions at the Property Level.

### **Step 2 - How to add Default Access Instructions at the Portfolio Level**

1. On the Property Manager's account, click the **Portfolios** tab.
2. Click on the name of the **Portfolio**.
3. On the **Details** page, scroll down to the Maint. Communication Settings section. From here, you can update or add Default Access Instructions  at the Portfolio Level. 

### **Step 3 - How to add Default Access Instructions at the Company Level**

1. On the Property Manager's account, click the **Details** tab.
2. Scroll down to the "Maint. Communication Settings" section. From here, you can update or add Default Access Instructions at the Company Level.
