# Purpose

**Purpose of this document**

It is a critical step in the Home Assistant onboarding process to audit the client’s general ledger to ensure the Latchel Home Assistant fee was added to each unit.

**Who should read this document?**

Launch Managers, Customer Success Managers

# Content

## Step-by-Step Guide

**What happened prior to this step?**

Prior to this step, the customer needs to generate and send you a general ledger report after the Home Assistant fees have been [configured ](https://latchel.com/gl-code-setup-home-assistant/)for each unit.

**Example GL Report:**

## Audit Checklist

- [ ] Open GL report
- [ ] Verify this report was generated from the Property Manager's property management system where rent is also being collected _(i.e. Appfolio, Buildium, Rent Manager, PropertyWare, Propertymeld, Yardi)_
- [ ] Review the report to ensure all property addresses that are going live with Home Assistant appear on the report
- [ ] Verify each unit has a $$6.99 // $$14.99 (depending on the contract) concierge fee added
- [ ] **_OR_** verify the fee is included in a bulk resident benefit package fee that already exists as a tenant ledger line item
