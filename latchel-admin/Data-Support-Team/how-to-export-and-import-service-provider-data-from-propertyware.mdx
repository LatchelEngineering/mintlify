# Objective

**Purpose**

The purpose of this document is to provide an overview process on how to pull required Service Provider data, convert to the standard data entry template, and import this data to a new Property Manager account.

# Content

**What is happening prior to this process?**

The Property Manager has provided login credentials to access their PropertyWare account. 

## Step-by-Step Guide

### **Process Overview**

Once the Property Management account has been set-up into the Latchel Console, you will be able to start adding and configuring Service Providers. The process starts with logging into the customer's PropertyWare account and exporting a Service Provider report. That information will be exported from PropertyWare and converted into a format that can be imported into the Latchel Console. After all Service Providers have been imported, you will be able to finalize configurations and prioritization. 

### **Step 1 - Export Property Information**

The initial request for login information is done by the Sales team before the onboarding meeting. If this information is not received, please follow-up with the Property Management company.
