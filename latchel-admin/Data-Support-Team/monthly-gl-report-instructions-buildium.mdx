#### **Monthly Reports**

- Please send over a CSV version of the “Transaction detail by account” report on the 20th of each month to [accounting@latchel.com](mailto:accounting@latchel.com) using the below filters:
  - Properties: All
  - Accounts: GL Code Name\*
  - Date Range: Last 30 Days
  - Accounting Basis: Cash

\*To locate the GL Code name, please reference the “Resident Charge Additional Information” field on the Onboarding tab of the Opportunity in Salesforce.
