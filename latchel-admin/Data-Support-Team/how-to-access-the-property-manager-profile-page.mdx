# Objective

**Purpose**

The purpose of this document is to show how you find a property manager in the Latchel Console and access their profile page. 

# Content

## Step-by-Step Guide

### **Find and enter the Property Manager Profile Page**

1. Go to "People" menu and click on "Property Managers:
2. Search the name of the new Property Manager account
3. Click on the Property Manager account name to open the account
4. The Property Manager profile page will open
