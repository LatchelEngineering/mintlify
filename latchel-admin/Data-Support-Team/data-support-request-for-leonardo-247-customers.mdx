### **Objective**

For Leonardo 24/7 customers, the goal of the onboarding and account update process is to make the process of setting up and updating the property management company's account as simple and easy as possible. It also aims to prioritize each ticket that is received in order to guarantee that each request is fulfilled on time.

‌

### **Jira Request Process**

In order to easily identify Account update and/or onboarding Jira requests for Leonardo 24/7 customers, the requester must use the workflow [Leo24/7 Request](https://slack.com/shortcuts/Ft0782KTEV39/399dcba3eadf5b11e9dd4328f160e9e9) within the <custom data-type="smartlink" data-id="id-1">https://latchel.slack.com/archives/C05CGVC9FPV</custom> channel. 

The requester must provide the following details in the workflow:

1. **Company Name**
2. **Customer’s company URL in Latchel**
3. **The property management integration** (if active)
4. **Type of request**

    1. **Onboarding** - choose this option if the request involves onboarding any data in Latchel or if it pertains to a subsequent configuration related to a previous onboarding.
    2. **Account Update** - choose this if an existing data in Latchel need to be updated.
    
5. **What needs to be done** - provide a detailed information/instruction relevant to the type of request.
6. **Deadline**
7. **Link to the property manager’s report if in Google spreadsheet format**, or
8. **Attach report if report is in csv format**

### **Leonardo 24/7 Priority Level**

As indicated in <custom data-type="smartlink" data-id="id-2">https://latchel.atlassian.net/wiki/spaces/TST/pages/3072688257</custom>, all **Leonardo 24/7 requests** _**take precedence over**_ any **non-Leonardo 24/7 Jira requests (normal Account Update requests)** unless the request is also an Onboarding type or falls under Service-Impacting or Critical tasks. 

_In the event that both Leonardo 24/7 Onboarding, and non-Leonardo 24/7 Onboarding is available, the Data Support Team must prioritize completing the Leonardo 24/7 Onboarding request._

‌

### **Leonardo 24/7 Requests via Assembly**

All requests submitted for Leonardo 24/7 customers will automatically create an _**Assembly ticket**_. For more information visit <custom data-type="smartlink" data-id="id-3">https://latchel.atlassian.net/wiki/spaces/TST/pages/3222437894</custom>.
