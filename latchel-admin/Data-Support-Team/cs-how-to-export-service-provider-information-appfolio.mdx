# Objective

This guide walks the Customer Success team through how to instruct the Property Manager on how to gather Service Provider information from a customer's AF account. **The Latchel team is to never log into an AF account on behalf of the customer.** Steps 1-5 are to be done by the Property Manager. Steps 6-7 are to be done by Latchel. 

We do not need to export property or resident data, as this information is transferred via the Latchel data sync. See this SOP on [How to: Install the Chrome Plug-in (Appfolio)](https://latchel.atlassian.net/wiki/spaces/CS/pages/813729953).

# Content

## Step-by-Step Guide

### **Step 1. Access the Vendor Directory**

1. Intruct the Property Manager to log into their software.
2. On the left side, intruct the Property to go to **Reporting**, then click **Reports**.
3. Under the **Owner & Vendor Reports** section, instruct the Property Manager to click on **Vendor Directory** and a new page will open.

### **Step 4. Customize the Vendor Report**  

1. On this new page, at the top right-hand side of the screen, the Property Manager can click **customize**.
2. Once they click customize, a pop-up will appear that will allow the Property Manager to select the columns for the report. Instruct the Property Manager to check the following boxes and uncheck everything else:
   1. **Company Name**
   2. **Name**
   3. **Phone Numbers**
   4. **Emails**
   5. **Tags**
   6. **Vendor Trades**
   7. Have the Property Manager click **Update** once the boxes specified above are checked. 
3. The pop-up will disappear and on that same page, the report will update with the selected columns.

### **Step 5.** **Export the Report**

1. On the top right-hand side of the report page, instruct the Property Manager to click **Actions** and then **Export as CSV** to download this report to their computer. 
2. Have the Property Manager email this report to [onboarding@latchel.com.](mailto:onboarding@latchel.com.)
3. See SOP: [(Draft) How to: Handle Customer Success emails in Zendesk](https://latchel.atlassian.net/wiki/spaces/CS/pages/849314225) to find this email. 

### **Step 6. Add this document to the Property Manager's Information folder in Google Drive.**

1. See SOP: [How to: Handle Property Manager Specific Files](https://latchel.atlassian.net/wiki/spaces/TST/pages/865469170)

### **Step 7. Import Service Provider Data** 

1. You will notice this report will need to be modified before we import this data into our system.
   1. The name of the Service Provider point person will need to be rearranged to display as follows: first name last name. For example, "Fence, All State" should be modified to "All State Fence". 
      1. To do this in Google sheets or excel, you will first need to split the text into 2 separate columns. Select the name cells and click **Data**. Under the Data drop-down, click **Split Text to Columns**. 
      2. The data will split will detect automatically and the first name and last name will appear in separate columns. Drag the column contacting the last name to the right of the column containing the first name (see video below).
      3. In cell C1, type "**=A1&" "&B1**". This formula will merge columns A and B into column C. 
      4. Click the bottom right corner to drag the formula in C1 down the column until all cells have been merged. 
   2. The words "Phone:" and "Office:" will need to be deleted. Our importer only allows for 1 phone number to import. If you are given an office number and phone number, choose the phone number over the office number. Make note of the numbers you did not import and add then to the service provider profile afterwards (See How to: [Configure Preferred Service Provider Accounts](https://latchel.atlassian.net/wiki/spaces/CS/pages/794198095/How+to+Configure+Preferred+Service+Providers+to+a+New+Account#Howto:ConfigurePreferredServiceProviderstoaNewAccount-step5Step5-ConfigurePreferredServiceProviderAccounts)). 
      1. To mass delete "Phone:" and "Office:" from your spreadsheet, highlight/select the Phone Numbers column. Next, click **Edit** and then **Find and Replace**. 
      2. A pop-up with appear. If you see the word "Phone:", type "Phone:" in the **Find** field. Add a blank space in the **Replace with** field. If you see the word "Office:", type "Office:" in the **Find** field. Add a blank space in the **Replace with** field. 
         1. Click **Find**.
         2. Click **Replace all**.
         3. Click **Done**.
   3. "Tags" and "Vendor Trades" will need to be consolidated into one column and reworded to match the Skill Categories in Latchel. If the list is under 10 vendors, consult [↔ Job Aid: Service Provider Skills and Definitions](https://latchel.atlassian.net/wiki/spaces/LTG/pages/791119550) to see all current Skill Categories listed in the system. 
      1. If the list of vendors exceeds 10, copy all tags listed in the spreadsheet and add a new sheet to your current spreadsheet. 
      2. On the new blank sheet, paste the tags. Refer back to the original sheet to copy the vendor trades and then paste the vendor trades on the new sheet in the same column the tags were pasted.
      3. Highlight column A and then split the tags and vendor trades into separate columns by clicking **Data** and then **Split text to columns**. 
      4. The text should have split into separate columns. 
      5. Copy/highlight column B and then right-click to delete the column. Paste the text in column A. Repeat this step for column C. All tags and vendor trades should be separated and listed in column A. 
      6. Select/highlight column A and click **Data** and then **Remove Duplicates** so you are aware of all tags/vendor trades in the spreadsheet. 
      7. In column B, for each row, add the matching Skill Category that Latchel has in the system. Consult [↔ Job Aid: Service Provider Skills and Definitions](https://latchel.atlassian.net/wiki/spaces/LTG/pages/791119550) to see all current Skill Categories listed in the system. If you do not see a match, highlight the row red and see this SOP on [↔ How to: Trigger a Deep-Dive Investigation by Customer Success](https://latchel.atlassian.net/wiki/spaces/LTG/pages/792822006) for help after the task has been completed. 
      8. Go back to the original sheet to start replacing the tags and vendor trades (Column A shown above) with Skill Categories in Latchel's system (Column B shown above). On the original sheet, do "**Ctrl F**" on your keyboard to populate a spreadsheet search tool to easily find the tag or vendor trade to replace with Latchel's Skill Category.
