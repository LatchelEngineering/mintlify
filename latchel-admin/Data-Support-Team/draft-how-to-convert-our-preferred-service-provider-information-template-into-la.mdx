# Objective

**Purpose**

The purpose of this document is to provide guidance on how to convert the information received via our "Preferred Service Provider" template into a file that can be uploaded into the Latchel Console with the Import Tool

# Content

**What is happening prior to this process?**

We have asked Property Managers to send us information on their Preferred Service Providers and they have decided to use our template to provide this information

## Step-by-Step Guide

### **Step 1. Receive the Information**

Refer to Step 1 of this SOP: [(Draft) How to: Convert our Property Information Template into "Latchel Imports" Required Format](https://latchel.atlassian.net/wiki/spaces/TST/pages/889782300)

### **Step 2. Understand the Information in the Template**

See template in the shared folder: [https://drive.google.com/drive/u/0/folders/1fPMYgh3niGJUwrAZFBP5dFtOeIkeWzlP](https://drive.google.com/drive/u/0/folders/1fPMYgh3niGJUwrAZFBP5dFtOeIkeWzlP)

1. The original template has four tabs: (1) Instructions; (2) Service Providers (REQUIRED); (3) Other Job Categories (Optional); (4) Potential Customizations (Optional)

    1. "Instructions" tab is to guide Property Managers on how to fill out the template. There, they can also list questions that they would like to discuss during the onboarding meeting
    2. "Service Providers (REQUIRED)" tab is requesting information on service providers for key skill categories that we utilize with more frequency
    3. "Other Job Categories (Optional)"  tab is requesting information on service providers that may be required 
    4. " Potential Customizations (Optional)" tab provides a list of certain customization we can provide. Those will mostly be discussed during the onboarding meeting
    
2. Both tabs that requests information from Service Providers will ask for company information and to mark with an "X" the skill categories that they provide. It will also have a question about prioritization of that vendor and other information the Property Manager may want us to know about that vendor, for example, situations in which that vendor should not be used

    1. If there is a comment about prioritization, please make sure to address that once you are working through step 7 of this SOP: [How to: Configure Preferred Service Providers to a New Account](https://latchel.atlassian.net/wiki/spaces/TST/pages/794198095)
    2. If there is a comment about skills or types of job a vendor should not do, or other skills not listed that the could be used in a job, please remove / add that skill during step 7 of this SOP: [How to: Configure Preferred Service Providers to a New Account](https://latchel.atlassian.net/wiki/spaces/TST/pages/794198095) 
    3. If you don't know what to do with some information provided by the Property Manager, escalate to a Customer Success Manager - [↔ How to: Trigger a Deep-Dive Investigation by Customer Success](https://latchel.atlassian.net/wiki/spaces/LTG/pages/792822006)
    


###  **Step 3. Make a copy of the file with the Service Provider information Received**

Refer to step 4 of this SOP: [How to: Handle Property Manager Specific Files](https://latchel.atlassian.net/wiki/spaces/TST/pages/865469170)

Note: we will make a copy of the file as we will move information around in the next step. To avoid the risk of damaging the file with the information the Property Manager sent us, we will make changes to a copy of that file 

###  **Step 4. Convert Property Information into Latchel Importer Required Format**

1. Open the file that you just created (i.e. the copy of the filled template sent by the Property Manager)
2. Delete the "Instructions" tab and the "Customizations" tab

    1. To delete a tab, right click on it and select "delete"
    
3. Go to the REQUIRED TAB:

    1. Select and delete the first rows/lines including the title and "section header". Keep the header of each column (i.e. company name, company phone number, etc)
    
        1. ![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=7a5ba0f3-9362-4378-b4ea-2b20d160d558&&collection=contentId-889947002&height=268&occurrenceKey=null&width=949&__contextId=null&__displayType=null&__external=false&__fileMimeType=null&__fileName=null&__fileSize=null&__mediaTraceId=null&url=null)
        
    2. Select and delete columns M and N that have information on prioritization and other information on each service provider
    
        1. Note that we are doing this on a COPY of the original file. Never delete information on the original file
        2. You will need to go back to these columns in the original file to finalize configurations
        
    
4. Go to the OPTIONAL TAB

    1. Select all of the other types of service provided (i.e appliances, awnings & sunshades, etc). Copy these cells by right clicking on one of the cells selected and selecting "copy"
    
        1. ![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=ad0f86a7-86c6-4671-98d9-162456cc771f&&collection=contentId-889947002&height=107&occurrenceKey=null&width=1429&__contextId=null&__displayType=null&__external=false&__fileMimeType=null&__fileName=null&__fileSize=null&__mediaTraceId=null&url=null)
        
    
5. Go to the REQUIRED TAB, right click on Column M, line 1, and select "Paste". Now you have combined all skill categories into the same tab

    1. 
    
6. Go back to the optional tab, select all cells with service provider contact information (column A to E). Copy these cells by right clicking on one of the cells selected and selecting "copy"
7. Go to the REQUIRED TAB, right click on the first empty cell on column A and paste the information. Select that row with the information you just copied and mark that in a different color.

    1. To select a row, click on the row number on the left side of the spreadsheet
    
        1. 
        
    2. To color a row, click on the "paint bucket" icon and select any color
    
        1. 
        
    
8. Go back to the optional tab, select all of the cells under the skill categories (i.e appliances, awnings & sunshades, etc) to include all of the cells with information provided by the Property Manager. Copy these cells by right clicking on one of the cells selected and selecting "copy"
9. Go to the  REQUIRED TAB, go to column "M" (with the appliances header) and scroll down until you see the line you colored. Click on that cell and paste the information that you copied from the optional tab in the prior step
10. Now you all all of the service providers and their information in one tab
11. Delete the optional tab
12. In order to import this information into the Latchel Console, you will need to consolidate in one cell all of the Skill Categories applicable to each Service Provider.

    1. To do that, select column "E", right click on it and select "1 left". This will add a column to the spreadsheet right after the service provider company information
    2. Label that column as "Skill Categories"
    3. For each Service Provider, look at the Skill Categories that they had an "X" and write that information in the new column created
    
        1. In the "Other" column, if the Property Manager added any skill, look for the corresponding Skill Category name that we have in the Latchel Console
        
            1. Go to this SOP ([↔ How to: Add or Remove Service Provider Skills](https://latchel.atlassian.net/wiki/spaces/OM/pages/791281864)) and search for the match (Press "Ctrl" key and the letter "F" together to open the search field, and type keywords to find the corresponding skill category
            
        
    4. Make sure to check step 2 of the following SOP on how the Skill Categories should be separated into that cell to avoid having an error during the import
    
        1. [How to: Configure Preferred Service Providers to a New Account](https://latchel.atlassian.net/wiki/spaces/TST/pages/794198095)
        
    5. For handyman or general contractors, use this SOP to define which Skill Categories should be used: [(CS/Admin) How to: Set-up Skills for Handyman and General Contractors](https://latchel.atlassian.net/wiki/spaces/CS/pages/796623267)
    6. Once you have consolidated all of the Skill Categories for each Service Provider into that new column, you can delete all of the columns with the individual skill categories
    7. You can also delete the column that has this header "Is this Service Provider available 24/7?"
    8. At the end of this process, you should only have these columns:
    
        1. Company Name
        2. Company Phone Number
        3. Company E-mail Address
        4. Name of the Main Point of Contact
        5. Skill Categories
        
    
13. Next step is to convert this spreadsheet into .csv format, see next section


###  **Step 5. Convert File to .csv and Import into the Latchel Console**

1. Refer to step 5 of this SOP on how to convert the file to .csv format:[(Draft) How to: Convert our Property Information Template into "Latchel Imports" Required Format](https://latchel.atlassian.net/wiki/spaces/TST/pages/889782300)
2. Once you have the .csv file, use this SOP to guide you on how to upload it to the Latchel Console: [How to: Configure Preferred Service Providers to a New Account](https://latchel.atlassian.net/wiki/spaces/TST/pages/794198095)
3. After all Service Providers have been uploaded into the Latchel Console, finalize configurations by following  step 7 of this SOP: [How to: Configure Preferred Service Providers to a New Account](https://latchel.atlassian.net/wiki/spaces/TST/pages/794198095)

    1. Note: Go back to the original filled template provided by the Property Manager and check if there is any additional information in the Prioritization or Other information columns that needs to be configured
