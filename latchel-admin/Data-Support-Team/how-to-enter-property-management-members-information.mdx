# Objective

**Purpose**

Latchel supports multiple users to a Property Manager's account. All members added can log in to the Latchel Portal with the same permission levels. This means all account members will be able to see work order statuses, add and disable properties, add and disable residents and configure settings on their account. Once additional members are added to a Property Manager's account, you can then assign account members as the primary Property Manager for the company, each portfolio, or each property. 

This SOP will walk you through how to add a new member to a Property Manager's account. 

# Content

**What is happening prior to this process?**

On the onboarding call, you collected a list of members the Property Manager would like added to their account. _See_ [_Step 1 (on the second tab)_](https://docs.google.com/spreadsheets/d/1iMJZ5uZjlAGkqoKCxO9zLzrii6s4YDYozTwe8csrpLI/edit#gid=0) _for this data template._

Additional members can also be requested after the onboarding call by the Property Manager. \*\*Please do **not** add additional members to a Property Manager's account without approval from an existing member on the account.

## Step-by-Step Guide

### **Step 1 - Add a Member to Property Manager Account**

1. Go to the Property Manager Profile Page: [How to: Access the Property Manager Profile Page](https://latchel.atlassian.net/wiki/spaces/TST/pages/813729656)
2. Click on the **Members** menu on the Property Manager's profile page.
3. On the righthand side of your screen, click the **\+New User** button.
4. The primary email associated with the user account is where the Property Manager receives all email notifications related to work orders. Fill out the member's information including their phone number (optional), email address (optional), and name (required) and then click **Save**. 

### **Step 2- Assign an Account Member as the Primary Property Manager for a Portfolio**

When a an account member is assigned to a portfolio, the user will only receive notifications for work orders associated with properties listed under a portfolio.

1. On the Property Manager's profile page, click the **Portfolios** menu.
2. Click on the name of the portfolio to open the portfolio.
3. From here, click the **three horizontal dots** under "Assigned PM User" and then click Reassign PM User.
4. Click the drop-down to choose from the added users on the account. 
5. Click **Submit** to save.

### Step 3- **Assign an Account Member as the Primary Property Manager for a Property**

When an account member is assigned to a property, the user will only receive notifications for work orders associated with that property. For more information on what is a Portfolio, how to create and configure one, see SOP: [How to: Create & Configure a Portfolio](https://latchel.atlassian.net/wiki/spaces/TST/pages/796623197)

1. On the Property Manager's profile page, click the **Properties** menu.
2. User the search tool to search for the property. 
3. Click on the address to open the property page.
4. Under the Property Manager section, click the **three horizontal dots** and then click **Change PM**.
5. Search for the Property Manager's Company name and select the name from the search results. 
6. Next, click the drop-down to choose from the added users on the account. 
7. Click **Submit** to save.
