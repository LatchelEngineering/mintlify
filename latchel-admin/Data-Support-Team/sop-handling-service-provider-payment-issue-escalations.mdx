## Step-by-Step Guide

**What happened prior to this step?**

A Property Manager's default credit card for vendor payments has a processing issues, such as insufficient funds, expired cards, etc.

### Process Overview

1. Sammie creates a Case in Salesforce by emailing [success@latchel.com](mailto:success@latchel.com) using [this template](https://docs.google.com/document/d/1LiDyvLtKG7lfq8az6yoOyAaLjkZqFwxg9mJ5DVxWS6k/edit#bookmark=id.v8ag1zhikqj5) when this issue arises.
2. Product Support will be the first to reach out via email using this [template](https://docs.google.com/document/d/1LiDyvLtKG7lfq8az6yoOyAaLjkZqFwxg9mJ5DVxWS6k/edit#bookmark=id.7aa0n58aypie)
3. Product Support then assigns the case to a CSM if the PM is unresponsive for **more than 24 hours**.
