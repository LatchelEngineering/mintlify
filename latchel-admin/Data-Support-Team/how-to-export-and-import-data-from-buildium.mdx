# Objective

**Purpose**

The purpose of this document is to provide an overview process on how to pull required property data and service provider data, convert to the standard data entry templates, and import this data to a new Property Manager account.

# Content

**What is happening prior to this process?**

The Property Manager has provided login credentials to access their Buildium account.  The initial request for login information is done by the Sales team before the onboarding meeting and should be added to this [tracker](https://docs.google.com/spreadsheets/d/16B1d5acwDRvx6w8BkSuL4_QAOZmcXKYwf5HFpMcwRdU/edit?ts=5d3f560b#gid=0). If this information is not received, please follow the process and templates outlined in this SOP to follow-up with the Property Manager: [(Draft) Job-Aid: Onboarding Flow & Communications](https://latchel.atlassian.net/wiki/spaces/CS/pages/851247114)

## Step-by-Step Guide

### **Process Overview**

Once the Property Management account has been set-up into the Latchel Console, you will be able to start adding and configuring properties. The process starts with logging into the customer's Buidlium account and exporting a full property report. That information will be exported from Buildium and converted into a format that can be imported into the system. After all properties have been imported, you will be able to finalize configuration and customizations. 

### **Step 1 - How to Export Property & Service Provider Information**

1. Log in to Buildium [here](https://signin.managebuilding.com/Manager/PublicPages/Login.aspx?ReturnUrl=%2FManager%2Fapp%2Fhomepage%2Fdashboard).
2. To export data from Buildium, navigate to **Settings >** **Export Data**.

3. Click **Tenants** for a property export (including tenant data which is preferred). 

    1. ![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=7d659400-9686-4a09-9b62-1ade05f7d2f2&&collection=contentId-795148345&height=860&occurrenceKey=null&width=1572&__contextId=null&__displayType=null&__external=false&__fileMimeType=null&__fileName=null&__fileSize=null&__mediaTraceId=null&url=null)
    
4. Click **Vendors** for a Service Provider export. 

    1. ![](blob:https://media.staging.atl-paas.net/?type=file&localId=null&id=4311337e-65e9-4e6a-a237-c7dbacf5d191&&collection=contentId-795148345&height=860&occurrenceKey=null&width=1572&__contextId=null&__displayType=null&__external=false&__fileMimeType=null&__fileName=null&__fileSize=null&__mediaTraceId=null&url=null)
    
5. Depending on your web browser, a pop-up window prompts you with options to open or save the file. Click the appropriate button to save the exported data file to your desktop. 

    1. This file will contain all the information under a certain heading (i.e, information for all properties), and is a .xlsx file, which can be viewed using Microsoft Excel, Numbers for Mac or OpenOffice.
