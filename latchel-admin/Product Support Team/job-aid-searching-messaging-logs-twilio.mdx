Searching for Messaging Logs
You can filter and search messaging logs based on specific criteria to locate relevant message
details quickly.
1. Search by Date Range
Use the date range picker to specify the time frame for which you want to view logs.
Select the start date and end date in the “Date Sent” field.
This filter displays all messages sent within the selected date range.
2. Search by Phone Number
In the “To” or “From” fields, enter the phone number (in E.164 format, e.g., +1xxxxxxxxxx) to
locate messages sent to or from that number.
3. Search by Message SID
If you have the unique Message SID (a unique identifier assigned to each message), enter it
into the Search bar to locate the exact message.
4. Search by Status
Filter messages based on their delivery status:
Status Meaning
Sent Message was successfully sent from Twilio but not yet delivered.
Delivered Message was successfully delivered to the recipientʼs device.
Failed Message could not be sent due to an error (e.g., invalid number).
Undelivered Message was sent but was rejected or could not reach the recipient.

Message Status Definitions:
Sent: The message has been successfully sent to the carrier.
Delivered: The message has been confirmed as delivered to the recipientʼs phone.
Failed: The message was not sent due to an error (e.g., invalid phone number, carrier issues).
Undelivered: The message was sent but was not delivered, possibly due to carrier problems
or filtering issues on the recipientʼs end.
2. Delivery and Status Breakdown
Delivery Status: This shows whether the message was delivered successfully or
encountered issues.
A Delivered status means the message reached the recipient.
A Failed or Undelivered status requires further investigation (see Error Code analysis
below).
3. Error Codes and Troubleshooting
If the message failed or was undelivered, an error code will appear in the log. This code
indicates why the message failed.
Click on the Message SID to view the detailed error information.
Locate the Error Code and Error Message fields.
Use Twilioʼs Error Code Documentation to identify the cause and recommended actions.
