# SOP: Running AppFolio Sync on Behalf of Customers

Purpose
Purpose of this document
To provide a backup solution for customers experiencing issues with the AppFolio
integration. This allows Latchel to run the sync on the customerʼs behalf while maintaining
full support for integration troubleshooting.
Scope
Applies to all customers with Appfolio.
Only used as an offer when customers encounter issues with the AppFolio sync.
Does not replace existing troubleshooting or standard support processes.
Responsibilities
Product Support Team:
Offer this backup sync solution to customers.
Guide customers in creating the necessary user account in AppFolio.
Notify Ethan so he can run the integration from an internal computer.
Monitor the process and confirm completion with the customer.
Customer:
Create a new user account in AppFolio with the specified permissions.
Respond to Product Support when setup is complete.
