# Job Aid: Tech High Sev Framework

Channel Purpose
The #tech-high-sev-escalations channel is only to be used for issues that block the
critical function of the platform where there is NO workaround available, including:
Production-blocking issues
Business-critical errors or bugs
Incidents impacting multiple customers or Latchelʼs core operational workflows
Non-critical issues, single-account errors, or UI bugs do not belong here and should
follow the standard #bug-report process.
To ensure a successful #tech-high-sev-escalations submission, reference Tech
High Severity Escalations Process.
Required Pre-Escalation Troubleshooting
Before posting, PS must confirm and document:
Reproduce the issue independently to verify itʼs not user error.
Check Twilio / API / Integration logs for errors or abnormal states.
Impersonate the affected user and replicate the issue step-by-step.
Test on control accounts to rule out isolated account failures.
Identify scope (# of PMs, residents, or workflows affected).
Document repro steps and key links to provide Engineering with immediate context.
Capture screenshots and/or screen recordings of every major step to support your
escalation.

guesswork, and speeds up resolution.
