# Job Aid: Handling User Login Issues

Objective: Assist users (Service Providers, Property Managers, and Residents)
experiencing login issues by confirming email validity and facilitating access to their
accounts.
### Step 1: Confirm User is Active
Access User Profile:
Navigate to the userʼs profile in the system.
Verify that the user account is active.
### Step 2: Email Validity
Access User Profile:
Verify that the email is correctly entered and valid for the user experiencing login problems.
### Step 3: Resolve Login Issues by User Type
A. Service Providers
Access User Profile:
1.
Locate the Service Providerʼs profile.
Initiate Login Email:
2.
Click the three dots located on the user profile.
Select "Send Account Email."
This will trigger a login email to the Service Provider.
B. Property Managers
Access User Profile:
1.
Locate the Property Managerʼs profile.
Initiate Login Email:
2.
Click the three dots located on the user profile.
Select "Send Account Email."
This will trigger a login email to the Property Manager.
C. Residents
Access Resident Profile:
1.
Locate the Residentʼs profile.
Obtain Public Dashboard URL:
2.
Navigate to the residentʼs miscellaneous settings.
Copy the Resident Public Dashboard URL.
Refer to Job Aid: How to Locate Public Dashboard URLs
Send URL:
3.
Email the Resident Public Dashboard URL to the resident.
This allows the resident to access their dashboard directly.
### Step 4: Follow-Up
Ensure confirmation of receipt of email or URL by the user.
Provide any additional assistance if further issues arise.
Note: Always maintain privacy and security protocols when handling user information and
sending emails. Confirm the user's identity by asking for their address, phone number, or
another unique piece of information on file.
