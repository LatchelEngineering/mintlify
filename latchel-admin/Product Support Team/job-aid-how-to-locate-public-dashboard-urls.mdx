# Job Aid: How to Locate Public Dashboard URLs

Resident Public Dashboard URL
The Resident Public Dashboard Link provides residents with direct access to their Latchel
dashboard quickly and easily.
This can be particularly helpful for property managers as it allows them to guide residents
more effectively by directing them to their customized dashboard. This enables residents
to see or update maintenance progress, file new requests, or access their benefits.

Service Provider Public Dashboard URL
Similar to the Resident Public Dashboard Link, we also offer a Service Provider Dashboard
Link. This process mirrors the one used for residents.
However, it's important to note that this is an admin-only URL, meaning that a property
manager does not have access to the public dashboard URL for any of their service
providers. Only a Latchel admin can provide this link to either residents or property
managers.
