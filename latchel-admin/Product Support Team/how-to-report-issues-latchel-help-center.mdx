# How to: Report Issues with the Latchel Help Center

Slack Workflows in the #help-guide-requests slack channel
New Help Guide Request
1.
This workflow is used when there is a need to create a completely new help guide. It
a.
involves capturing initial requests for documentation that addresses new topics or areas not
covered by existing guides.
Update Existing Help Guide
2.
This workflow is utilized when existing help documentation needs revisions or updates. This
a.
could be due to outdated information, changes in procedures, or feedback suggesting
improvements.
Help Guide Escalation
3.
This workflow is activated in situations involving complex loading issues or high-priority
a.
requests related to help guides that are inaccessible to our customers. Consider this type of
request as the highest priority issue, as it is obstructing a customer's access.
Filling Out a Slack Workflow
### Step 1: Access the Slack Workspace
Open Slack and log in to your workspace if you havenʼt already.
### Step 2: Navigate to the Correct Channel
Locate and enter the specific channel where the workflow is set up.
### Step 3: Initiate the Workflow
Look for pinned messages or channel descriptions that provide instructions.
Click on the lightning bolt icon (workflows) next to the message input field and search for
the workflow by name.
### Step 4: Fill Out the Workflow Form
Enter the required details in the form that appears. Include all necessary information
specific to the workflow.
### Step 5: Review Details
Double-check the information you've entered for accuracy and completeness.
### Step 6: Submit the Workflow
Click the "Submit" or "Done" button to send the workflow for processing.
