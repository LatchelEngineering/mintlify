# How to: Subscription Upgrade

The purpose of this document is to walk PS through the steps to take when a PM requests
additional coverage for their entire account and/or portfolio. This is how to updated the
subscription from after-hours emergency to 24/7 emergency.
Step-by-Step Guide:
1. Receive Request.
2. Create the Product Package in Latchel.
3. Make the Product Package the Active Subscription.
4. Notify Billing.
5. Confirm with the PM.
Step-by-Step Guide:
1. Receive Request.
Confirm if the change will be account wide or specific portfolios.
Let the PM know that the price will change to $1.99/unit for those units.
2. Create the Product Package in Latchel.
Go to the PM Account page in Latchel.
Go to the Product Packages tab and click New Package.
Enter the name for the new package (use the subscription type).
Once the Product Package is created, click on it from the list to add the products.
For Base Product, select 24/7 Emergency and choose the start date for today.
3. Make the Product Package the Active Subscription.
Go to the PM Account page or the Portfolio page.
Then go to the Subscriptions tab and click New Subscription.
You will select the Product Package you just created.
4. Notify Billing.
In the #accounts-receivable channel, use the Subscription Invoicing Request workflow.
Fill it out with all the details and the Category of: Change Request (CS).
Tag the CSM in the thread for awareness.
5. Confirm with the PM.
Notify the PM that the change is done and they are good to go!
