# How to: Build Reports via the Analytics Dashboard

Purpose
This SOP outlines how to use the Latchel Analytics Dashboard to view performance metrics,
apply filters, and generate reports for analysis, sharing, or recordkeeping.
Accessing Analytics
Navigate to Analytics within Latchel to view dashboards and reporting tools that provide insight
into maintenance performance, work order trends, and operational efficiency.
Building & Exporting Reports
The Analytics Dashboard allows viewing and downloading reports for data created
between 90 days and 2 days ago
### Step 1: Apply Filters
Use filters to ensure your report reflects the correct data set:
Date Range – Select the timeframe you want to review.
Portfolio / Owner – Narrow data to specific portfolios or owners.
Tip: Always apply filters before exporting reports
### Step 2: Review Key Metrics
The Analytics dashboard displays high-level performance metrics, including:
Work Orders Prevented
Emergencies De-escalated
Resident Review Score
Conversations Handled
Hours to Schedule
Days to Complete Work Orders
Each metric tile allows you to:
Filter data
View underlying data
Refresh data
Export results
### Step 3: Analyze Work Order Trends
Use the Work Orders Created vs. Completed chart to:
Compare monthly work order volume
Identify backlog or completion trends
### Step 4: Export Reports
Option A: Export from Key Metrics
Select a key metric tile.
1.
Click the ellipsis (…).
2.
Choose Export.
3.
Select the desired file type and save.
4.
Note: This export includes only data tied to the selected metric
Option B: Direct Download
Select Download / Export within Analytics.
1.
Choose specific reports or download the entire workbook.
2.
Confirm file format (PDF is the default).
3.
Export Other Reports
Available report types include:
Work Orders
Residents
Properties
Invoices
Steps:
Click Reports → Export.
1.
Select Reports from the dropdown.
2.
Choose the report type and apply filters.
3.
Click Export.
4.
Option D: Export via Email
From Download / Export.
1.
Select Export via Email.
2.
Enter recipient email address(es).
3.
Adjust settings as needed and click Export.
4.
Feedback & Improvements
Use the Feedback option in the top-right corner of the Analytics dashboard to submit feature
requests, report issues, or share suggestions.
