# Purpose

<Info>
**Purpose of this document**

To provide a backup solution for customers experiencing issues with the AppFolio integration. This allows Latchel to run the sync on the customer’s behalf while maintaining full support for integration troubleshooting.
</Info>

<Info>
**Scope**

- Applies to all customers with Appfolio.
- Only used as an offer when customers encounter issues with the AppFolio sync.
- Does not replace existing troubleshooting or standard support processes.
</Info>

<Info>
**Responsibilities**

- **Product Support Team:**

  - Offer this backup sync solution to customers.
  - Guide customers in creating the necessary user account in AppFolio.
  - Notify Ethan so he can run the integration from an internal computer.
  - Monitor the process and confirm completion with the customer.

- **Customer:**

  - Create a new user account in AppFolio with the specified permissions.
  - Respond to Product Support when setup is complete.
</Info>

1. Reach out via email or support ticket using the template below:

  1. *Dear [Client's Name], I understand you’ve been running into issues running the sync from your end. We are pleased to offer our support in **running the AppFolio extension on your behalf.**To do so, we will need you to set up a user account with specific permissions in your AppFolio system. Our process involves creating a user with access to view units, residents, and the ability to view, edit, and create work orders. We can then run the integration on one of our machines while monitoring the process from our side.*[*<u>Here is a step-by-step guide</u>*](https://docs.google.com/document/d/1z8aflT7uonxNBByl3UWhU21U-10fXkadK53mNoDu-Pc/edit?tab=t.0)*to help you set up the new user account. Once completed,**please respond to this email**and we will do the rest! If you have any questions or require assistance, we're here to help. We look forward to supporting you in this setup.*
  2. Provide the following instructions to the customer:[<u>Appfolio User Creation-Latchel Integration Management</u>](https://docs.google.com/document/d/1z8aflT7uonxNBByl3UWhU21U-10fXkadK53mNoDu-Pc/edit?tab=t.0)
