<Note>
  The purpose of this SOP is to provide a step-by-step guide for the Latchel team on how to correctly log in to the TeamViewer remote server and access the specific Property Manager profiles for which Latchel runs syncs. Following this process ensures a secure and organized connection to the virtual desktop environments.
</Note>

<Note>
  ### Prerequisites:

  - [TeamViewer](https://www.teamviewer.com/es/download/windows/?msockid=2fb7661c85e3609d1e6273e9840d6150) application installed and logged in with authorized credentials.
  - Access permissions to the specific remote device
</Note>

## Step by step

### Step 1: Connect to the TeamViewer Virtual Desktop

a. Open TeamViewer and log using the login credentials provided.

<Tip>
  Currently, only members of the **Latchel Integration Team and Ops Support** have access to the Latchel cloud account. If you have been approved for access, <u>reach out to **Trisha** to **request login credentials**.</u>
</Tip>

b. On the left-hand sidebar, click on the **Devices** icon. c. Locate the required device (e.g., **70323.local**) and click the **Connect** button next to it.

<Frame>
  ![image-20260603-214713.png](/images/Intg-56.png)
</Frame>

### Step 2: Access the Chrome Profile Manager

a. Once the virtual desktop has loaded, navigate to the dock at the bottom of the screen. b. Click on the **Chrome application icon** to open the browser. This will trigger the Profile Manager window to appear.

<Frame>
  ![image-20260603-214938.png](/images/Intg-57.png)
</Frame>

### Step 3: Select the Specific PM Profile

a. A list of available Property Management profiles will be displayed. b. Scroll through the list and click on the **specific PM tab** you need to work on (e.g., _Genuine Property Management_ or the assigned company). c. If the tab is already open, ensure you are in the correct profile before proceeding with any tasks.

<Frame>
  ![image-20260603-215039.png](/images/Intg-58.png)
</Frame>

### Step 4: Close the Connection Correctly

a. Once your work is finished, do not simply minimize the window. b. Navigate to the **TeamViewer toolbar** at the top of the remote session window. c. Click the **'X' (Close)** button on the far right of the top bar to disconnect from the remote server cloud safely.

<Frame>
  ![image-20260603-215225.png](/images/Intg-59.png)
</Frame>

<Warning>
  Make sure you don’t click **“End Session,”** as this will cause the remote server connection to be lost.
</Warning>
