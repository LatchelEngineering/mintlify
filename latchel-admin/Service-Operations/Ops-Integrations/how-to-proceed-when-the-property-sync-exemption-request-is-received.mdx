---
title: "How to: Proceed when the Property sync exemption request is received"
---

<Note>
The purpose of this SOP is to provide a step-by-step guide on how to process a "Property Sync Exemption Request". This workflow is used when a Property Manager requests to exclude certain properties (e.g., commercial units, non-revenue units, parking spots, storage units) from the automatic data sync with Latchel. This procedure applies only to PMs whose syncs are managed internally by our team.
</Note>

## Step by step

### **Step 1: Locate and Acknowledge the Exemption Request**

a. Look for the **Property Sync Exemption Request** workflow posted in the `#integration-sync-issues-internal` Slack channel. **<u>This request is typically filled out by a CSM or PST.</u>**
b. React to the workflow thread with a check mark (✅) to let the team know you are working on it.

<Frame>
  <img src="/images/Intg-68.png" alt="image-20251114-184505.png" />
</Frame>

### **Step 2: Access the Property Manager's (PM) Software**

a. Open the remote desktop server to access the client's work environment.
b. Log in to the specific PM's AppFolio account using the credentials stored.

<Note>
Typically, all login credentials are stored in the AppFolio account. If the account is logged out for any reason, please refer to the [Software Login document](https://docs.google.com/spreadsheets/d/16B1d5acwDRvx6w8BkSuL4_QAOZmcXKYwf5HFpMcwRdU/edit?gid=2017091315#gid=2017091315) for the necessary login details.
</Note>

### **Step 3: Find and Exclude the Property**

a. Once logged into the PM's account, navigate to their property list.
b. Use the search function to find the specific property address that was requested in the workflow.

<Frame>
  <img src="/images/Intg-69.png" alt="image-20251114-184613.png" />
</Frame>

c. Click on the property to open its details page.
d. Locate and click the blue button labeled **“Exclude from Property Sync”**.

<Frame>
  <img src="/images/Intg-70.png" alt="image-20251114-185010.png" />
</Frame>

e. Once the button is clicked, its text changes to **"Add to Property Sync"**, allowing the customer to click it later to add the property back.

<Frame>
  <img src="/images/Intg-71.png" alt="image (78).png" />
</Frame>

### **Step 4: Confirm Task Completion to the Stakeholder**

a. Once the property has been successfully excluded, return to the original Slack thread for the **Property Sync Exemption Request**.
b. In the thread, tag the stakeholder who submitted the request and inform them that the change has been made. You can use the following template:

> Hello [Stakeholder Name],The requested property has been successfully exempted from the sync. Let me know if we can help with anything else
