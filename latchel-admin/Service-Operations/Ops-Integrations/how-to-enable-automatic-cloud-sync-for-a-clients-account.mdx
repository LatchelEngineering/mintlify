---
title: "How to: Enable automatic cloud sync for a client's account"
---

<Note>
The purpose of this SOP is to provide a step-by-step guide on how to correctly set up a Chrome profile on the Latchel cloud device for a new property manager's AppFolio account to ensure the sync runs correctly
</Note>

## Step by step guide

### **Step 1: Log in to the Latchel cloud device**

a. Log in to the designated **Latchel cloud account** where syncs are managed.

<Tip>
Currently, only members of the **Latchel Integration Team** have access to the Latchel cloud account. If you have been approved for access, <u>reach out to </u>**<u>Trisha</u>**<u> to </u>**<u>request login credentials</u>**<u>.</u>
</Tip>

### **Step 2: Open the configured Chrome window**

a. From the desktop view, **open the main Chrome application**. This will open the profile manager window.

<Frame>
  <img src="/images/Intg-60.png" alt="image-20260603-211403.png" />
</Frame>

b. A new window will open for creating a **new Chrome profile**.

<Note>
**NOTE:** Each customer is managed in a unique Chrome guest profile to keep credentials and sessions isolated.
</Note>

### **Step 3: Add a new profile**

a. Click the **Add** button at the bottom of the displayed profiles.
b. <u>Make sure you are in </u>**<u>Guest mode</u>**<u> before proceeding</u>.

<Frame>
  <img src="/images/Intg-61.png" alt="image-20251029-235028.png" />
</Frame>

### **Step 4: Continue without an account**

a. In the new window that appears, click **Continue without an account**.

<Frame>
  <img src="/images/Intg-62.png" alt="image-20251029-235148.png" />
</Frame>

### **Step 5: Create and name the new profile**

a. **Create a new guest profile** and name it so it’s easy to identify.
b. Use the naming convention: **"PM Company Name + PM"**
c. Choose an icon color and select **Done**.

<Frame>
  <img src="/images/Intg-63.png" alt="image-20251029-235406.png" />
</Frame>

### **Step 6: Log in to the customer’s AppFolio account**

a. In the **newly created Chrome profile**, navigate to the **AppFolio login page**.
b. Use the credentials provided to log in.

<Note>
**NOTE:** See the **AppFolio** tab in the [**Software Login document**](https://docs.google.com/spreadsheets/d/16B1d5acwDRvx6w8BkSuL4_QAOZmcXKYwf5HFpMcwRdU/edit?gid=0#gid=0) for the username and password.
</Note>

### **Step 7: Complete Two-Factor Authentication (2FA)**

a. When logging in, you will likely need to complete 2FA by entering a code received via SMS.

<Note>
**NOTE: **The code can be retrieved from the Latchel console or the `#ops_info_center` channel if an ops agent received it first.
</Note>

<Tip>
<u>Ensure you save the login info in Chrome after you sign in to AppFolio to avoid sync disruptions.</u>
</Tip>

### **Step 8: Install the Latchel Chrome Sync extension**

a. Once logged into AppFolio, navigate to the following [URL to install the Latchel Chrome Sync](https://chromewebstore.google.com/detail/latchel-data-sync/cefobanegfgmcneibgdbcjbegnnoafob) extension:
b. Click **Add to Chrome** and confirm by clicking **Add extension** in the pop-up.

### **Step 9: Pin the extension for easy access**

a. **Click the puzzle icon** (Extensions) in the Chrome toolbar.
b. Find the "**Latchel Data Sync**" extension and **click the pin icon** next to it. The Latchel icon will now be visible in your toolbar.

<Frame>
  <img src="/images/Intg-64.png" alt="Screenshot 2025-10-29 at 10.38.41 AM" />
</Frame>

### **Step 10: Log in to the Latchel extension**

a. **Click** the newly pinned **Latchel extension icon** in the toolbar.
b. A login window will appear. **Enter the username and password** for the PM's Latchel account.

<Note>
**NOTE**: This login information can be found under the **AppFolio** tab in the [**Software Login document.**](https://docs.google.com/spreadsheets/d/16B1d5acwDRvx6w8BkSuL4_QAOZmcXKYwf5HFpMcwRdU/edit?gid=0#gid=0)
</Note>

### **Step 11: Verify the extension is active**

a. After logging into the extension, refresh the AppFolio page. A message will appear asking to look for and connect to any device on your local network - **Click Allow and refresh the page again**.

<Frame>
  <img src="/images/Intg-65.png" alt="image-20251120-195845.png" />
</Frame>

b. You should now see a floating Latchel button near the bottom-right corner of the AppFolio window.

<Frame>
  <img src="/images/Intg-66.png" alt="image-20251030-000309.png" />
</Frame>

### **Step 12: Initiate the initial sync**

a. **Click** the floating **Latchel button**.
b. From the options that appear, **initiate the Slow sync**. This will sync residents, properties, owners, and work orders.

<Frame>
  <img src="/images/Intg-67.png" alt="image-20251030-000621.png" />
</Frame>

### **Step 13: Confirm the process is complete**

a. You’re done. **<u>The sync has been initiated and will continue to run automatically on the Latchel cloud device</u>** within this dedicated Chrome profile.

<Tip>
Automated syncs run every four hours.
</Tip>
