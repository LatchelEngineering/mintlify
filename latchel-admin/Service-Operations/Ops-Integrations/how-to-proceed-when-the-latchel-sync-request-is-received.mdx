<Note>
  The purpose of this SOP is to provide a step-by-step guide on how to proceed when the CSMs fill out the Latchel sync request for new property managers who want Latchel to run their sync.
</Note>

## Step by step

### Step 1: Look for the Latchel Sync Request Workflow Received

a. React to the new Latchel sync request with a check mark so the team knows you're already working on it.

<Frame>
  ![image-20250822-193225.png](/images/Intg-39.png)
</Frame>

### Step 2: Add the PM in the PM Exclusions tab.

a. Go to the _Integration Health Metrics Dashboard_ and unhide the **PM Exclusions** tab. b. Copy the new PM name from the **PM Metrics** tab and paste values (Ctrl\+Shift\+V) at the end of the list in **PM Exclusions**.

<Frame>
  ![image-20251120-153448.png](/images/Intg-40.png)
</Frame>

c. Select **Yes** in column B (**Exclude from Outreach?**) and column C (**Should sync issues be flagged internally?**).

<Frame>
  ![image-20251120-153817.png](/images/Intg-41.png)
</Frame>

<Tip>
  By selecting **Yes** in both columns, you ensure the new PM you add to the internal syncs managed by Latchel won't be flagged at the same time in the **Integration Issue PMs** and **Integration Issue PMs (Internally Flagged)** tabs if an issue is detected.
</Tip>

d. Once you have finished this step, right-click the tab name and hide it again.

<Warning>
  <u>Follow **Step 3** only if the request was submitted by a CSM. If it was submitted by the **PS team**, skip Step 3 and continue with **Step 4**.</u>
</Warning>

### Step 3: Go to the success email to find the account activation link

a. Look for the account activation Appfolio email. b. Open the email and **click in “Create Your Account”**

<Frame>
  ![image-20250813-194802.png](/images/Intg-42.png)
</Frame>

c. A new window will appear, asking you to **enter the password** and click **Activate**.

<Frame>
  ![image-20250822-154518.png](/images/Intg-43.png)
</Frame>

<Tip>
  Make sure to add the password shown in the **Latchel sync request workflow** received, if no password is provided ensure to add this default info:

  - **Password:** L@tchel187
</Tip>

d. Another window will then **confirm the phone number you should enter to receive an SMS** with a code to log in. Click on **Send verification code**.

<Frame>
  ![image-20250822-154741.png](/images/Intg-44.png)
</Frame>

<Tip>
  Ensure to add this default info:

  - **Phone number:** 206-717-8374
</Tip>

### Step 4: Login the PMS

<Warning>
  After you enter the phone number, you will receive an SMS to login the PMS.<u> Since the SMS **expires in 10 minutes**, make sure to grab it as soon as possible to enter the code.</u>
</Warning>

<Warning>
  If an agent grabs the SMS, they will post it in the [#ops\_info\_center](https://latchel.slack.com/archives/CNPUUL3PX) channel.
</Warning>

a. Log in to the PMS using the credentials provided in the **Latchel Sync Request**.

b. **Go to Latchel’s console** and look for the SMS received.

<Frame>
  ![image-20250813-195806.png](/images/Intg-45.png)
</Frame>

c. **Copy and paste the code on the PMS page to log in**. Make sure to close the SMS notification by clicking the three dots at the top of the SMS window.

d. **You will be logged in the PMS**.

### Step 5: Create the Latchel Account

a. Go to [Latchel’s console](https://app.latchel.com/) and login as an admin.

b. Once logged in, go to People \> Property Managers

<Frame>
  ![image-20250916-195339.png](/images/Intg-46.png)
</Frame>

c. Enter the name of the Property Management Company in the search bar and select it.

d. Once in the PMs profile, click in Users \> New User

<Frame>
  ![image-20250916-195930.png](/images/Intg-47.png)
</Frame>

e. Ensure to add this default info:

<Tip>
  - **Name:** Latchel Integration
  - **Primary Phone:** 206-717-8374
  - **Primary Email**: [success\+pmcompanyname@latchel.com](mailto:success+assetlocale@latchel.com)

  **NOTE: You can use the same username provided in the \_Latchel Sync Request workflow**\_ as the primary email.
</Tip>

f. Once the info is added click on **Save**.

<Frame>
  ![image-20250916-200500.png](/images/Intg-48.png)
</Frame>

g. Go to the success email and look for the message you just received to confirm the password.

<Frame>
  ![image-20250916-200652.png](/images/Intg-49.png)
</Frame>

h. Click on **Create Password** and enter the default password.

<Tip>
  - \*\*Password: \*\*L@tchel123!
</Tip>

i. Confirm the password, then click on **Submit Password**.

<Frame>
  ![image-20250916-201028.png](/images/Intg-50.png)
</Frame>

j. You will now be logged into Latchel’s account. You can log out once you’ve accessed the account.

### Step 6: Confirm the Integration is Active

a. Go back to your Latchel account and open the Property Manager profile. b. Click **More \> Integrations** and select the appropriate third-party software integration.

<Warning>
  **NOTE:** If it’s already enabled, you don’t need to make any changes.
</Warning>

<Tip>
  \*\*NOTE: \*\*For AppFolio customers, select **Chrome Plugin**.
</Tip>

<Frame>
  ![image-20251105-222323.png](/images/Intg-51.png)
</Frame>

c. Once you select the integration, click **Activate**. d. A new window will appear. Update the plugin settings with the following default information:

<Tip>
  - **App Slug:** pmcompanyname
  - **Starting Sync Date:** Today’s date
</Tip>

<Frame>
  ![image-20251105-222635.png](/images/Intg-52.png)
</Frame>

e. Click **Save**.

### Step 7: Make sure to record the login information

a. Ensure that the company name, URL, username, password, and two-factor authentication number are added to the [**Software Login**](https://docs.google.com/spreadsheets/d/16B1d5acwDRvx6w8BkSuL4_QAOZmcXKYwf5HFpMcwRdU/edit?gid=0#gid=0)\*\* \*\*document under the AppFolio tab. The login information should be entered for both the PMS access and Latchel's account

<Frame>
  ![image-20250828-225422.png](/images/Intg-53.png)
</Frame>

### Step 8: Complete the configuration so the sync can run.

a. Follow all steps in the SOP [**How to: Enable automatic cloud sync for a client's account**](https://help.latchel.com/how-to-enable-automatic-cloud-sync-for-a-clients-account).

b. Once all steps are complete, proceed to Step 7.

### Step 9: Confirm to the stakeholder that you have accessed the account and that everything is ready for the sync.

a. In the thread of the received **Latchel Sync Request workflow**,<u> send the following message depending on the stakeholder who submitted the request:</u>

**Template for CSMs:** _Hello \[CSM Name\],_

\_We now have access to the PM account and have completed setup on our end. The sync is running successfully. If any issues related to the sync are encountered, we will tag you in an alert notification in this channel. As a final step, please ensure the PM uninstalls the Latchel Data Sync extension, as they no longer need to perform the sync on their end. \_

_Please encourage them to install the_[\_ Latchel Overlay\_](https://chromewebstore.google.com/detail/property-manager-visor/nbifpjdaeocmafihkpjohfcodimelibc)\_ instead, which includes “View in Latchel” and several other features (view the assigned vendor in Latchel directly from AppFolio, cancel and complete work orders without leaving AppFolio, click “Create work order in Latchel” to be taken directly to the Latchel workflow, analytics, and more).\_

_Let me know if we can help with anything else!_

**Template for PST:** _Hello \[PST Member\],_ \_We now have access to the PM account and have completed the setup on our end. The sync is running successfully. As a final step, please ensure the PM uninstalls the Latchel Data Sync extension, as they no longer need to perform the sync on their end. \_

_Please encourage them to install the_[\_ Latchel Overlay\_](https://chromewebstore.google.com/detail/property-manager-visor/nbifpjdaeocmafihkpjohfcodimelibc)\_ instead, which includes “View in Latchel” and several other features (view the assigned vendor in Latchel directly from AppFolio, cancel and complete work orders without leaving AppFolio, click “Create work order in Latchel” to be taken directly to the Latchel workflow, analytics, and more).\_

_Let me know if we can help with anything else!_

### **Step 10: Save the Latchel Sync Request in the channel**

a. In **#integration-sync-issues-internal**, go to **More** at the top of the channel and select the **Latchel Sync Requests** folder.

<Frame>
  ![image-20251120-200728.png](/images/Intg-54.png)
</Frame>

b. Click the **\+** button and select **Link** to add the new request.

<Frame>
  ![image-20251120-200851.png](/images/Intg-55.png)
</Frame>

c. Add the link and the PM company name, then click **Save**. d. The request will be successfully saved.
