title: "How to: Submit a New Project Request"
---

<Note>
The purpose of this SOP is to provide a clear, step-by-step guide for all stakeholders to submit new project requests using the dedicated Slack workflow.
</Note>

<Tip>
This workflow is designed to ensure that all necessary information is captured upfront, allowing for faster review and assignment of new projects. This process begins in the [#ops-integrations](https://latchel.slack.com/archives/C09QE22VBDF) Slack channel.
</Tip>

## Step by step

### **Step 1: Go to the #ops-integrations Slack channel**

a. Navigate to the [#ops-integrations](https://latchel.slack.com/archives/C09QE22VBDF) channel in Slack to begin the submission process.

### **Step 2: Look for the "New project request" workflow**

a. At the top of the channel, you will find a **"Workflows"** button. Click it to see available workflows.

<Frame>
  <img src="/images/Intg-15.png" alt="image-20251104-214507.png" />
</Frame>

### **Step 3: Select the Slack Workflow: "New Project Request"**

a. From the list of available options, select the **New Project Request** workflow to open the submission form.

### **Step 4: Fill out the project request form**

a. To submit a new project, stakeholders will now use this Slack workflow. The form will prompt you for the following information:

- **What is the priority of this task?:**
Indicate the urgency of the request (e.g. Low, Medium, High).
- **Please provide a description of the project:**
A summary of what you want built and why, including any important context.
- **What is the desired outcome?:**
The concrete result you expect (e.g., “automate X reports” or “reduce manual steps”).
- **What key problem will this project solve?:**
A description of the pain point or inefficiency you want to eliminate.
- **Please add any existing assets, research, or documents related to this project that you can share. (optional):**
This field allows for file uploads or links to provide context and speed up the review process.

<Frame>
  <img src="/images/Intg-16.png" alt="image-20251104-214846.png" />
</Frame>

### **Step 5: Await confirmation and updates**

a. Once your request is submitted, you will receive updates directly in the workflow thread in Slack. This ensures you are kept informed of the status of your request.
