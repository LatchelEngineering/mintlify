## Audit Question Calibration

| **Audit Question** | **What we’re looking for** |
| :-- | :-- |
| Did the associate take action on the non-emergency Autopilot WO for one of the 3 reasons listed in [<u>the SOP</u>](https://latchel.atlassian.net/wiki/spaces/OPS/pages/2563276894 "https://latchel.atlassian.net/wiki/spaces/OPS/pages/2563276894")? | The associate took action for one of the following reasons: <ol><li>The PM explicitly asked them to do something</li><li>They received an inbound communication from a resident, SP or owner about the WO</li><li>A task on the WO was assigned to Latchel Service Ops</li></ol> |
| Did the associate successfully update and share the relevant script template or specialized script along with the corresponding topic or help article, enabling the Property Manager to autonomously undertake necessary actions? | The associate used the correct script template from the “PM Communication” section of [<u>the SOP</u>](https://latchel.atlassian.net/wiki/spaces/OPS/pages/2563276894 "https://latchel.atlassian.net/wiki/spaces/OPS/pages/2563276894") to assist the customer. They made sure they understood the customer’s question and responded with the relevant help article. |
| If no Help Article existed for the PM’s question, did the associate submit a Public Latchel Help Article Request Slack workflow? | The review team [<u>checks</u>](https://docs.google.com/spreadsheets/d/1WTbsLA1joyIHnoO9I9fXEjdhK39QUxH8GjuZADUs_Rg/edit#gid=1769173476 "https://docs.google.com/spreadsheets/d/1WTbsLA1joyIHnoO9I9fXEjdhK39QUxH8GjuZADUs_Rg/edit#gid=1769173476") the Public Latchel Help Article Request submissions to verify if the associate submitted a request for a new blurb and/or Help Article after the customer interaction. |
| If the Property Manager resisted taking independent actions, did the associate assume responsibility and carry out the necessary tasks on their behalf? | If the Property Manager resisted taking the action on their end, the associate responded and fulfilled the customer’s requests and assigned any follow up tasks to the PM, if applicable. |
| If a task (other than “SP left a note” and “Job is incomplete”) was assigned to Latchel Service Ops, did the associate submit a #bug-report? | The review team checks the #bug-report Slack channel to verify if the associate escalated the issue with a task being incorrectly assigned to Latchel Service Ops. |
