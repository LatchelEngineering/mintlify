## Motivation and assumptions

<Note>
  This format was designed to facilitate functionalities like phone search or click-to-dial.

  Our assumption is that all phone numbers stored in Salesforce are associated with the United States.
</Note>

## Format Standard

At the moment of updating any phone-related field in Salesforce, the following rules will be automatically applied to the input.

<Check>
  1. Removing Non-Digit Characters: We will only keep the numbers. Any symbols or spaces will be removed.
  2. Required Length: Phone numbers must have either 10 or 11 digits. If there are more or fewer digits, the system will save a NULL value.
  3. Removing Country Code: If the phone number has 11 digits, and the first digit is a '1', the system will automatically remove it; if the phone number has 11 digits and the first digit is not a '1', the system will save a NULL value.
</Check>

## Examples

Here are a few examples of how the phone numbers will be stored and displayed after saving a record:

## Background tools

We used a mix of two flows \+ one APEX Class to validate the format every time a record is saved.

`The flows are:`

- FL - SA - CO - Phone to Numeric
- FL - SA - AC - Phone to Numeric

`The APEX Class code is:`

```apex
public class AX_SA_Phone_Number_to_Numeric {

    @InvocableMethod(label='AX - SA - CO - Phone Number To Numeric' description='Takes a list of phone numbers as input and returns a list of numeric representations according to specific rules. It returns an empty string if the input is null or empty. It transforms the input to numeric format, adds 1 at the beginning if it has 10 digits, retains the number if it has 11 digits and starts with 1, and returns an empty string for any other length or format.')
    public static List<String> extractNumeric(List<String> phoneNumbers) {
        List<String> result = new List<String>();

        for (String phoneNumber : phoneNumbers) {
            if (phoneNumber == null || String.isBlank(phoneNumber)) {
                result.add('');
                continue;
            }

            String numericChars = '';
            List<String> phoneChar = phoneNumber.split('');
            for (Integer i = 0; i < phoneNumber.length(); i++) {
                if (phoneChar.get(i).isNumeric()) {
                    numericChars += phoneChar.get(i);
                }
            }

            // Check the length of numericChars
            if (numericChars.length() == 10) {
                result.add(numericChars); // If 10 digits, return the same number
            } else if (numericChars.length() == 11) {
                if (numericChars.startsWith('1')) {
                    result.add(numericChars.substring(1)); // If 11 digits starting with '1', return substring starting from the second character
                } else {
                    result.add(''); // If 11 digits but not starting with '1', return ''
                }
            } else {
                result.add(''); // return '' for any other length
            }
        }

        return result;
    }
}
```
