---
title: "How to add a new user in Salesforce"
---

<Note>
In order to add a new user in Salesforce you'll need a Salesforce license.

If the user needs access to the phone dialer, they will need an Amazon Connect Account as well.

SDRs will also need to be given permissions and do other set-up activities to be able to use cadences.

See this SOP for these other user set-up steps:
[SOP - How to Add or Remove Users to Sales Team - Salesforce and Amazon Connect](https://help.latchel.com/latchel-admin/Sales-Team/sop-how-to-add-or-remove-users-to-sales-team-salesforce-and-amazon-connect)
</Note>

## 1. Get to the "Users" setup page

Click on Setup (gear icon) > type "users" on the Quick Find search bar > Click on Users

<Frame>
  <img src="/images/DA-24.png" alt="Users setup page" />
</Frame>

Once inside Users, you'll see the users grouped by different views. Views are just different types to group users so we can find them faster. Feel free to change the view if you need to copy permissions or licenses from another user.

<Frame>
  <img src="/images/DA-25.png" alt="Users grouped by views" />
</Frame>

## 2. Create new user

Click on "New User".

<Frame>
  <img src="/images/DA-26.png" alt="New User button" />
</Frame>

Fill out the General Information required fields:

- Always fill out in this order
- First Name
- Last Name
- Email
- Company: "Latchel"
- Department: For this field copy the same name of one of the view names e.g. "Sales", "Customer Success", etc

Continue filling out the rest of the fields selected on the screenshot with no specific order. You can add more depending on the specific need.

<Frame>
  <img src="/images/DA-27.png" alt="General Information required fields" />
</Frame>

Fill out the Additional Information required fields:

- Slack Member ID: To find a user's Member ID, open their profile in Slack, click the three-dot button, and copy the member ID.

<Frame>
  <img src="/images/DA-28.png" alt="Additional Information required fields" />
</Frame>

Keep the Locale Settings this way:

<Frame>
  <img src="/images/DA-29.png" alt="Locale Settings" />
</Frame>

Click on Save and you'll have created a new user in Salesforce.
