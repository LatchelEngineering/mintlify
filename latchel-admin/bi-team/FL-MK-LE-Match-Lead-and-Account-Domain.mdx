<Note>
  Every time a Lead is created, this flow takes his email domain and search it into the existing accounts domains and make sure this match is the account domain and not an email provider domain (gmail, outlook, etc). If there is a match company in our database it is going to add that to the Lead.
</Note>

## Technical aspects

| **New Name** | FL - SL - LE - Match Lead and Account Domain |
| :-- | :-- |
| **Old Name** | Match Lead and Account Domain |
| **Expected person to trigger the flow** | Auto trigerred every time a new lead is created. |
| **Salesforce User IDs included in the flow** | - |
