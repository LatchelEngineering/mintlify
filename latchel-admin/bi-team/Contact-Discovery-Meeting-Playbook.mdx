## Description

This flow and his Subflows are the ones that make it possible for an SDR to qualify a prospect during the Discovery meeting call through Salesloft, this is the main difference between this particular Flow and the other ones created for the same purpose. The other ones need to be triggered within Salesforce. The SDR qualifies the prospect and attempts to schedule a Demo meeting with the AE. This Flow shows a list of advice to the SDRs to help them to achieve the objectives and also, shows a template that helps the Rep assess whether the prospect is a good fit for our services. The SDR has to manually enter essential information into the template<sup>1</sup>  and the Flow converts some multis-select picklist answers to single select to be processed into Salesforce.

This Flow has a Subflow that referenced a “Master Flow” named Qualification Playbook and this is the one that has the template mentioned in the previous paragraph. QuasacascascaacsascascaslifiQcation Playbook
