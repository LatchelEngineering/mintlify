## How to Handle: New Requests

1. When a new feature is requested by a property manager, PS should gather as many as details as possible from the PM. Such as:

- What is the problem we are solving for? We want to focus on the problem vs the potential solutions. PMs are notorious for requesting an update or feature without sharing the “WHY” or “What is happening?” that has led to this request. We need to understand what we are working to solve for.
- Solution Recs are great! And this is valuable to gather from the PM too.
- Specific examples - Work Orders, Email template, SMS templates, etc.

<Note>
  Based on the request, review and follow if applicable:

  - [https://latchel.atlassian.net/wiki/x/AoA-tg](https://latchel.atlassian.net/wiki/x/AoA-tg)
  - [https://latchel.atlassian.net/wiki/x/AYD\_pw](https://latchel.atlassian.net/wiki/x/AYD_pw)
</Note>

2. If request cannot be achieved via an Action Set or PD/CN, PS should send [the Feature Request for PM Experience SF Template](https://latchel1.lightning.force.com/lightning/r/EmailTemplate/00XUS00000B83qP2AR/view?queryScope=userFolders) to the PM.
3. PS submit the request in the #feature-requests Slack channel via the Feature Request workflow.
4. Once submitted, [copy the Slack Url and paste it to the Case Thread](https://latchel.atlassian.net/wiki/spaces/SR/pages/3198353472/SOP+Latchel+Department+Escalation+Process+for+Product+Support+Cases#3.-Escalating-to-a-Latchel-Department).
5. PS move the case to Closed.
6. The template with the CSAT Survey will send **24 hours** after the case is moved into the Closed state.

## How to Handle: Follow-Ups on Requests

1. PS should send [the Follow-Up on Feature Request SF Template](https://latchel1.lightning.force.com/lightning/r/EmailTemplate/00XUS00000CgGhN2AV/view).
2. If the PM pushes back and/or gets frustrated, [PS should escalate the case to the CSM](https://latchel.atlassian.net/wiki/x/VoDpnQ).

---

<Warning>
  If the PM is unresponsive to either email template, PS does **not** need to send the Unresponsive Follow-Up Template. PS can simply close the case as Closed Unresponsive.
