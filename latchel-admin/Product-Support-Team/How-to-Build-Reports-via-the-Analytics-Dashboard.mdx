### **Purpose**

This SOP outlines how to use the **Latchel Analytics Dashboard** to view performance metrics, apply filters, and generate reports for analysis, sharing, or recordkeeping.

### **Accessing Analytics**

Navigate to **Analytics** within Latchel to view dashboards and reporting tools that provide insight into maintenance performance, work order trends, and operational efficiency.

  ![3772907527 1](/images/3772907527-1.png)
</Frame>

### **Building & Exporting Reports**

<Note>
  _The Analytics Dashboard allows viewing and downloading reports for data **created between 90 days and 2 days ago**_
</Note>

#### **Step 1: Apply Filters**

**Use filters to ensure your report reflects the correct data set:**

- **Date Range – Select the timeframe you want to review.**
- **Portfolio / Owner – Narrow data to specific portfolios or owners.**

<Tip>
  **Tip: Always apply filters before exporting reports**
</Tip>

<Frame>
  ![3774119947 1](/images/3774119947-1.png)
</Frame>

### **Step 2: Review Key Metrics**

The Analytics dashboard displays high-level performance metrics, including:

- Work Orders Prevented
- Emergencies De-escalated
- Resident Review Score
- Conversations Handled
- Hours to Schedule
- Days to Complete Work Orders

Each metric tile allows you to:

- Filter data
- View underlying data
- Refresh data
- Export results

### **Step 3: Analyze Work Order Trends**

Use the **Work Orders Created vs. Completed** chart to:

- Compare monthly work order volume
- Identify backlog or completion trends

### **Step 4: Export Reports**

#### **Option A: Export from Key Metrics**

1. Select a key metric tile.
2. Click the ellipsis (…).
3. Choose **Export**.
4. Select the desired file type and save.

<Frame>
  ![3774185481](/images/3774185481.png)
</Frame>

<Note>
  Note: This export includes only data tied to the selected metric
</Note>

#### **Option B: Direct Download**

1. Select **Download / Export** within Analytics.
2. Choose specific reports or download the **entire workbook**.
3. Confirm file format (_PDF is the default_).

<Frame>
  ![3773005836 1](/images/3773005836-1.png)
</Frame>

## **Export Other Reports**

Available report types include:

- Work Orders
- Residents
- Properties
- Invoices

Steps:

1. Click **Reports** → **Export**.
2. Select **Reports** from the dropdown.
3. Choose the report type and apply filters.
4. Click **Export**.

  ![3773595660 1](/images/3773595660-1.png)
</Frame>

#### **Option D: Export via Email**

1. From **Download / Export**.
2. Select **Export via Email**.
3. Enter recipient email address(es).
4. Adjust settings as needed and click **Export**.
