### **Resident Public Dashboard URL**

<Info>
  The Resident Public Dashboard Link provides residents with direct access to their Latchel [dashboard](https://help.latchel.com/s/article/Resident-Dashboard-Overview)quickly and easily.

  This can be particularly helpful for property managers as it allows them to guide residents more effectively by directing them to their customized dashboard. This enables residents to see or update maintenance progress, file new requests, or access their benefits.
</Info>

Here’s how a Property Manager can locate the Resident Public Dashboard Link:

**Step-by-step:**

1. Navigate to People
2. Choose "Residents"
   - Select the resident
3. From the Details Tab, Scroll down to find the box that is labeled "Miscellaneous"
4. Open the box by clicking on the carrot.
5. Click on the "Copy" button

### **Service Provider Public Dashboard URL**

<Info>
  Similar to the Resident Public Dashboard Link, we also offer a Service Provider Dashboard Link. This process mirrors the one used for residents.

  However, it's important to note that this is an admin-only URL, meaning that a property manager does not have access to the public dashboard URL for any of their service providers. Only a Latchel admin can provide this link to either residents or property managers.
</Info>
