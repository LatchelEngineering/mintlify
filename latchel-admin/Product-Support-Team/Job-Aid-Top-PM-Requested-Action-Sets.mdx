<Info>
  This Job Aid is used to share the details of the top PM requested action sets. It will also to serve to share those that can be updated and changed based on the specific PM request.
</Info>

## How to: Request New Action Sets

1. Review the below table.
2. Find the Action Set you would like model your request after.
3. In the #pd\_cn\_actionset Slack Channel, fill out the Action Set Request workflow and submit:

<img
  src="/images/3244982273.png"
  alt="3244982273"
  width="335"
  title="3244982273"
/>

- **Company Name (as it appears in Latchel)**
  - Copy & paste the Company Name from Latchel database.
- **PM Company ID**
  - When on the PM Company page, the ID are the last digits found in the URL link.

<Frame>
  ![3245080577](/images/3245080577.png)
</Frame>

- **Action Set Group URL the request is based off of:**
  - Find the Action Set Group in the table below.
  - Click on the Action Set Group hyperlink to take you to page in Latchel.
  - Copy & paste the URL in this section.
  - [Video demo here.](https://www.loom.com/share/ba2a58e2f4b44d429a44f7823d2b4fcb?sid=11c814e9-65ab-48f1-ac78-1b17e7cb295d)
- **Specific Action Set URL the request is based off of:**
  - Find the Specific Action Set in the table below.
  - Click on the Specific Action Set hyperlink to take you to page in Latchel.
  - Copy & paste the URL in this section.
  - [Video demo here.](https://www.loom.com/share/4955361b506c45e5a1a73ac14850bab6?sid=e2bf4a7f-512a-454d-bb3b-ebee85371f82)
- **New Action Set Request:**
  - Share all the details for the new request, such as:
    - Emergency WOs, Nonemergency WOs, Both
    - Specific script/verbiage
    - Who the notification should go to
    - Email, SMS, Both
    - State changes for the WO
    - WO Problem(s)
